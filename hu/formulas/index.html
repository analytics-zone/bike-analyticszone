Kerékpáros Teljesítmény Képletek

A Bike Analytics Metrikák Matematikai Alapjai

Implementációs Útmutató

Ez az oldal másolható-beilleszthető képleteket és lépésről lépésre számítási módszereket biztosít az összes Bike Analytics metrikához. Használd ezeket egyéni implementációkhoz, ellenőrzéshez vagy a teljesítmény-alapú edzés mélyebb megértéséhez.

⚠️ Implementációs Megjegyzések

  • Minden watt érték wattban (W), idő másodpercben, hacsak nincs másként megadva
  • Az FTP és CP egyéni specifikus küszöbök—nincsenek univerzális értékek
  • Mindig validáld a bemeneteket ésszerű tartományokra (0-2000W tipikus)
  • Kezeld a szélsőséges eseteket (nullával osztás, negatív watt)
  • A wattadatok 1 másodperces rögzítési intervallumot igényelnek a pontossághoz

Alap Teljesítmény Metrikák

1. Training Stress Score (TSS)

Képlet:

TSS = (időtartam_másodperc × NP × IF) / (FTP × 3600) × 100
ahol IF = NP / FTP

Számított Példa:

Szcenárió: 2 órás edzés, NP = 235W, FTP = 250W

  1. IF számítása: IF = 235 / 250 = 0,94
  2. Időtartam másodpercben: 2 óra × 3600 = 7200 másodperc
  3. TSS = (7200 × 235 × 0,94) / (250 × 3600) × 100
  4. TSS = 1 590 720 / 900 000 × 100 = 176,7 TSS

Értelmezés: Nehéz edzés (>150 TSS), 2-3 nap regenerálódás várható

2. Normalizált Teljesítmény (NP)

Algoritmus (30 másodperces gördülő átlag):

1. Számítsd ki a 30 másodperces gördülő átlag wattot az egész edzésre
2. Emeld minden 30mp értéket a 4. hatványra
3. Vedd ezeknek a ^4 értékeknek az átlagát
4. Vedd ennek az átlagnak a 4. gyökét
NP = ⁴√(átlag [30s_átlag^4])

Miért a 4. Hatvány?

A negyedfokú (4. hatvány) kapcsolat tükrözi a változó erőfeszítések nem-lineáris fiziológiai költségét. Egy kitörésekkel és pihenésekkel teli edzés több energiájába kerül, mint az állandó watts ugyanazon az átlagon.

Példa:

  • Állandó edzés: 200W 1 órán át → NP = 200W, Átlag = 200W
  • Változó edzés: Váltakozó 300W/100W → Átlag = 200W, NP = 225W

Ugyanaz az átlagos watt, de a változó edzésnek 12%-kal magasabb az NP-je a kitörések fiziológiai költsége miatt

3. Intensity Factor (IF)

IF = NP / FTP

Értelmezési Útmutató:

  • IF < 0,75: Regenerációs/könnyű edzés
  • IF 0,75-0,85: Állóképességi edzés
  • IF 0,85-0,95: Tempó edzés
  • IF 0,95-1,05: Küszöb edzés
  • IF > 1,05: Versenytempó/VO₂max intervallumok

Teljesítménymenedzsment Metrikák

4. CTL (Krónikus Edzésterhelés)

CTL_ma = CTL_tegnap + (TSS_ma - CTL_tegnap) / τ_CTL
ahol τ_CTL = 42 nap (időállandó)

5. ATL (Akut Edzésterhelés)

ATL_ma = ATL_tegnap + (TSS_ma - ATL_tegnap) / τ_ATL
ahol τ_ATL = 7 nap (időállandó)

6. TSB (Training Stress Balance)

TSB = CTL - ATL

Értelmezés:

  • TSB > +25: Túl kipihent (fittségvesztés kockázata)
  • TSB +5 és +25 között: Friss, kész teljesítményre
  • TSB -10 és +5 között: Produktív edzésben
  • TSB < -30: Túlterhelt, pihenés szükséges

Kritikus Teljesítmény Képletek

7. Teljesítmény-Időtartam Modell

t = W' / (P - CP)

Ahol:

  • t = kimerülésig tartó idő (másodperc)
  • W' = anaerob munkakapacitás (joule)
  • P = teljesítmény (watt)
  • CP = kritikus teljesítmény (watt)

8. W' Balance

W'_bal(t) = W' - ∫(P(u) - CP) du (ha P > CP)
W' helyreáll, ha P < CP
