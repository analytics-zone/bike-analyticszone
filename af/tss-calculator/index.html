Bereken Oefenstres-telling vir jou fietsoefenings deur gebruik te maak van krag, duur, en FTP

Wat is Oefenstres-telling (TSS)?

Oefenstres-telling (TSS) kwantifiseer die oefen-lading van 'n fietsoefen deur intensiteit en duur te kombineer gebaseer op jou krag-uitset. Ontwikkel deur Dr. Andrew Coggan, gebruik TSS jou Funksionele Drempelkrag (FTP) as die verwysings punt. 'n 1-uur oefening by FTP = 100 TSS.

TSS Formule

TSS = (sekondes × NP × IF) / (FTP × 3600) × 100

Waar:

  • NP (Genormaliseerde Krag) = Fisiologiese "koste" van die rit in watt
  • IF (Intensiteitsfaktor) = NP / FTP (intensiteit relatief tot drempel)
  • Duur = Totale rit tyd in sekondes
  • FTP = Jou Funksionele Drempelkrag in watt

Vereenvoudig: TSS = Duur (ure) × IF² × 100

TSS Riglyne volgens Oefening Tipe

Oefening Tipe TSS Reeks Intensiteitsfaktor Beskrywing
Herstel Rit 20-50 TSS IF < 0.65 Maklike trap, 30-60 minute
Maklike Uithouvermoë 50-100 TSS IF 0.65-0.75 Gesprek tempo, 1-2 ure
Matige Uithouvermoë 100-150 TSS IF 0.75-0.85 Bestendige ry, 2-3 ure
Tempo Rit 150-200 TSS IF 0.85-0.95 Sweet spot, tempo werk, 2-3 ure
Drempel Oefening 200-300 TSS IF 0.95-1.05 FTP intervalle, wedren sim, 2-4 ure
VO₂max Intervalle 150-250 TSS IF 1.05-1.15 Harde intervalle, 1-2 ure hoë intensiteit
Wedren/Harde Gebeurtenis 200-400 TSS IF 0.90-1.05 Criteriums, padwedre, 2-5 ure
