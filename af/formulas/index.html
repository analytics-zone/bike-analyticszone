Fietse Krag Formules - Wiskundige Grondslag

Volledige fietse krag formules: TSS berekening, NP algoritme, IF formule, CP model vergelykings, en W'bal opsporing.

Wiskundige Grondslag

Hierdie bladsy bevat alle formules wat Bike Analytics gebruik om fietsprestasie-statistieke te bereken. Alle formules is gebaseer op portuurbeoordeelde navorsing.

Funksionele Drempelkrag (FTP)

20-Minuut Toets

FTP = 0.95 × Gemiddelde Krag (20 minute)

Voorbeeld: 250W vir 20 min → FTP = 250 × 0.95 = 238W

Oefenstres-telling (TSS)

TSS Formule

TSS = (sekondes × NP × IF) / (FTP × 3600) × 100

Waar:

  • NP = Genormaliseerde Krag
  • IF = Intensiteitsfaktor (NP / FTP)
  • FTP = Funksionele Drempelkrag

Vereenvoudig: TSS = (IF)² × duur (ure) × 100

Genormaliseerde Krag (NP)

NP Algoritme

NP = ⁴√(gemiddeld van (30s rollende gemiddeld krag)⁴)

NP verreken die nie-lineêre verhouding tussen krag en fisiologiese koste.

Intensiteitsfaktor (IF)

IF Formule

IF = NP / FTP

Voorbeeld: NP = 210W, FTP = 250W → IF = 210 / 250 = 0.84

Veranderlikheidsindeks (VI)

VI Formule

VI = NP / Gemiddelde Krag

Voorbeeld: NP = 240W, Gem = 200W → VI = 240 / 200 = 1.20

Prestasiebestuur-grafiek (PMC)

CTL (Chroniese Oefen-lading)

CTL_vandag = CTL_gister + (TSS_vandag - CTL_gister) / 42

42-dag eksponensieel geweegde bewegende gemiddeld van daaglikse TSS.

ATL (Akute Oefen-lading)

ATL_vandag = ATL_gister + (TSS_vandag - ATL_gister) / 7

7-dag eksponensieel geweegde bewegende gemiddeld van daaglikse TSS.

TSB (Oefenstres-balans)

TSB = CTL - ATL

Positief TSB = vars, negatief TSB = moeg.

Kritiese Krag (CP) Model

CP Formule

Tyd tot Uitputting = W' / (Krag - CP)

Waar:

  • CP = Kritiese Krag (watt)
  • W' = Anaërobiese kapasiteit (joules)
  • Krag = Huidige krag-uitset (watt)

Krag-tot-Gewig Verhouding

W/kg Formule

W/kg = FTP (watt) / Liggaam Gewig (kg)

Voorbeeld: 250W FTP, 70kg → W/kg = 250 / 70 = 3.57 W/kg

VAM (Velocità Ascensionale Media)

VAM Formule

VAM = (Hoogte Gewen in meters / Tyd in ure)

Voorbeeld: 1000m in 60 min → VAM = 1000 / 1 = 1000 VAM

✅ Deursigtige Berekeninge

Bike Analytics gebruik hierdie formules presies soos gedokumenteer. Geen swart boks algoritmes nie—alles is deursigtig en gebaseer op bewese wetenskap.
