การวัดกำลังการปั่นจักรยาน: อธิบาย NP, IF, VI และ W'bal

ศูนย์กลางที่ใช้งานได้จริงสำหรับการเชื่อมต่อไฟล์การขับขี่เข้ากับ FTP, พลังวิกฤต, โซนการฝึกอบรมการตัดสินใจเรื่องจังหวะ และ TSS

ตอบด่วน

การวัดกำลังการปั่นจักรยาน เปลี่ยนวัตต์ดิบเป็นการตัดสินใจ กำลังไฟฟ้ามาตรฐาน (NP) ประมาณการต้นทุนที่แท้จริงของการขี่แบบแปรผัน ปัจจัยความเข้ม (IF) แสดงให้เห็นว่าการขับขี่นั้นยากเพียงใด FTP, ดัชนีความแปรปรวน (VI) แสดงให้เห็นว่าความพยายามนั้นมั่นคงหรือวุ่นวายเพียงใด และ วาบาล ช่วยตีความว่าคุณใช้ความจุที่เกินเกณฑ์ไปมากน้อยเพียงใด

  • การใช้งานที่ดีที่สุด: ใบสั่งการฝึกอบรม อัตราความเร็วการแข่งขัน และการวิเคราะห์หลังการขับขี่
  • สมอที่มีประโยชน์ที่สุด: ทันสมัย FTP หรือ พลังวิกฤต ค่า
  • ข้อผิดพลาดที่ใหญ่ที่สุด: เปรียบเทียบ NP, IF หรือ W'bal โดยไม่มีบริบทจากภูมิประเทศ ระยะเวลา และประเภทเหตุการณ์
  • ขั้นตอนต่อไปที่ดีที่สุด: เชื่อมต่อเมตริกกับ โซนการฝึกอบรม และ TSS แทนที่จะอ่านแยกกัน

ประเด็นสำคัญ

  • เอ็นพี บอกคุณว่าทำไมการขี่สองครั้งที่มีกำลังเฉลี่ยเท่ากันจึงให้ความรู้สึกแตกต่างไปจากเดิมอย่างสิ้นเชิง
  • IF ความสัมพันธ์กลับเข้มข้นขึ้น FTP ดังนั้นตัวเลขจึงสามารถกลายเป็นการตัดสินใจในการฝึกอบรมได้
  • วี แยกความพยายามบนถนนที่มั่นคงออกจากเกณฑ์ที่มีกระแสไฟกระชากสูงและการแข่งรถ MTB
  • วาบาล จะมีประโยชน์มากที่สุดเมื่อคุณสนใจเกี่ยวกับความพยายามที่เกินเกณฑ์ซ้ำๆ และการฟื้นตัวระหว่างกัน
  • คลัสเตอร์ที่แท้จริง:FTP, พลังวิกฤต, FTP กับ Critical Power, โซนการฝึกอบรม, อากาศพลศาสตร์, และ TSS

นักปั่นจักรยานส่วนใหญ่ไม่ต้องการการวัดเพิ่มเติม พวกเขาต้องการวิธีที่ดีกว่าในการตีความไฟล์ที่มีอยู่ในไฟล์การเดินทาง หน้านี้เป็นศูนย์กลางในทางปฏิบัติของระบบดังกล่าว: ความหมายของการวัดกำลังหลัก ช่วยได้เมื่อใด ล้มเหลวตรงไหน และหน้าใดที่เกี่ยวข้องที่คุณควรใช้ถัดไป

คำถามที่มีประโยชน์นั้นง่าย:

  • การขับขี่ครั้งนี้ยากจริงๆ หรือแค่มีเสียงดัง?
  • ฉันทุ่มเทความพยายามให้กับงานนี้ได้ดีหรือไม่?
  • ตัวชี้วัดใดมีความสำคัญมากกว่าสำหรับการออกกำลังกายนี้: กำลังคงที่ ความแปรปรวน หรือความสามารถในการทำซ้ำที่สูงกว่าเกณฑ์
  • ตัวเลขเหล่านี้ควรเปลี่ยนสิ่งที่ฉันทำต่อไปอย่างไร

สิ่งที่ตัวชี้วัดเหล่านี้วัดได้จริง

กำลังเฉลี่ย บอกคุณถึงสิ่งที่เกิดขึ้นโดยเฉลี่ย ตัวชี้วัดกำลังการปั่นจักรยานหลักจะบอกคุณ ความพยายามถูกกระจายอย่างไร, มันเครียดแค่ไหนเมื่อเทียบกับเกณฑ์, และ ไม่ว่าคุณจะใช้จ่ายความจุเกินขีดจำกัดอย่างชาญฉลาดหรือไม่.

เมตริกมันวัดอะไร.ดีที่สุดสำหรับการใช้ผิดวิธีทั่วไป
กำลังไฟฟ้ามาตรฐาน (NP)ต้นทุนทางสรีรวิทยาของกำลังไฟฟ้าที่แปรผันการตีความการขี่แบบกลุ่มยาก, MTB, คริติคอล, ภูมิประเทศที่มีการกลิ้ง และข้อผิดพลาดในการเว้นจังหวะถือว่ามันเป็นเป้าหมายการเว้นจังหวะด้วยตัวมันเอง โดยไม่มีระยะเวลาของเหตุการณ์หรือบริบทของภูมิประเทศ
ปัจจัยความเข้ม (IF)ความเข้มของการขี่สัมพันธ์กับ FTPใบสั่งยา การทบทวนการออกกำลังกาย และการจำแนกเซสชั่นการใช้ FTP ที่ล้าสมัย ดังนั้นการขับขี่ทุกครั้งจึงดูยากขึ้นหรือง่ายกว่าที่เป็นจริง
ดัชนีความแปรปรวน (VI)ความพยายามนั้นมั่นคงหรือหนักหน่วงเพียงใดการวิเคราะห์จังหวะและการเปรียบเทียบวินัยการเรียกเครื่องเล่น VI สูงทั้งหมดว่า "จังหวะไม่ดี" เมื่อเหตุการณ์นั้นต้องการการเพิ่มขึ้นอย่างรวดเร็ว
วาบาลกำลังการผลิตที่เหลืออยู่โดยประมาณข้างต้น พลังวิกฤตกลยุทธ์การแข่งขัน การโจมตีซ้ำ การปีนทางเทคนิค และการออกแบบช่วงเวลาการเชื่อถือแบบจำลองแบบสุ่มสี่สุ่มห้าเมื่อการประมาณการของ CP หรือ W' นั้นอ่อนแอ

แบบจำลองทางจิตที่เป็นประโยชน์

คิดถึง NP เป็นต้นทุน, IF เป็นบริบทเทียบกับเกณฑ์, VI เป็นรูปจังหวะ, และ W'bal เป็นงบประมาณที่เพิ่มขึ้นอย่างจำกัดของคุณ. พวกเขาร่วมกันอธิบายมากกว่าพลังเฉลี่ยเพียงอย่างเดียว

เมื่อแต่ละเมตริกมีประโยชน์

📋 ใบสั่งยาการฝึกอบรม

ใช้ FTP หรือ พลังวิกฤต ที่จะสร้าง โซนการฝึกอบรมจากนั้นใช้ IF และ NP เพื่อตรวจสอบว่าเซสชันไปถึงจุดที่คุณต้องการหรือไม่

ฮอลเรซ เพซิ่ง

ใช้ VI และ NP เพื่อตัดสินว่าคุณขี่บนถนนที่มั่นคงได้ดีหรือใช้เวลามากเกินไปกับคลื่นเร็วที่ทำให้เส้นชัยจางหายไป

📊 การวิเคราะห์หลังการขี่

NP, IF และ TSS ช่วยเปรียบเทียบเซสชันในช่วงเวลาและภูมิประเทศที่แตกต่างกัน โดยไม่ต้องแสร้งทำเป็นว่าทุกวัตต์โดยเฉลี่ยบอกเล่าเรื่องราวเดียวกัน

⚡ การทำซ้ำ

W'bal สำคัญที่สุดเมื่อเหตุการณ์ขึ้นอยู่กับความพยายามซ้ำแล้วซ้ำอีกเหนือเกณฑ์ เช่น XC MTB การแข่งรถบนถนนที่ดุเดือด หรือการเร่งความเร็วตามเกณฑ์

⚠️ การพึ่งพาที่สำคัญ: ถ้าคุณ FTP หรือ พลังวิกฤต การประมาณการไม่ถูกต้อง IF โซนฝึกซ้อม TSS และการตีความ W'bal ล้วนลอยไปตามนั้น

ที่ที่นักปั่นจักรยานใช้ตัวชี้วัดเหล่านี้ในทางที่ผิด

ข้อผิดพลาดทั่วไป

  • การเปรียบเทียบกำลังเฉลี่ยจากการไต่ระดับคงที่ไปยัง NP จากเกณฑ์ ราวกับว่าสิ่งเหล่านั้นหมายถึงสิ่งเดียวกัน
  • การใช้ IF กับ FTP ที่ไม่ได้ทดสอบซ้ำเป็นเวลาหลายเดือน
  • ไล่ตาม VI ที่ต่ำในรูปแบบการแข่งขันที่หลีกเลี่ยงความแปรปรวนไม่ได้
  • การอ่าน W'bal เหมือนมาตรวัดน้ำมันเชื้อเพลิงที่แท้จริงแทนที่จะเป็นแบบจำลองที่เชื่อมโยงกับสมมติฐาน
  • ใช้ไฟล์การปั่นที่น่าประทับใจเพียงไฟล์เดียวเพื่อกำหนดฟิตเนสใหม่โดยไม่มีข้อมูลแนวโน้ม

กฎการตีความที่ดีกว่า

  • เริ่มต้นด้วยประเภทกิจกรรม: TT, road race, gran fondo, XC, enduro หรือ crit
  • ตรวจสอบว่ารูปทรงจังหวะตรงกับเป้าหมายหรือไม่ก่อนตัดสินตัวเลข
  • ใช้ โซน และ TSS เพื่อตัดสินใจว่าการขี่หมายถึงอะไรสำหรับการฝึกซ้อม
  • จับคู่ W'bal ด้วย พลังวิกฤต และเรียกร้องความพยายามซ้ำแล้วซ้ำเล่า
  • ใช้แนวโน้มข้ามเซสชัน ไม่ใช่เมตริกพาดหัวเดี่ยวๆ

วิธีที่ Power Metrics เชื่อมต่อกับ FTP, CP และ Zones

คลัสเตอร์นี้จะมีประโยชน์เมื่อเพจสนับสนุนซึ่งกันและกันเท่านั้น:

  • FTP ทอดสมอโซนฝึกซ้อมและทำให้ IF สามารถตีความได้
  • พลังวิกฤต ให้แบบจำลองที่ดีกว่าสำหรับงานที่อยู่เหนือเกณฑ์และ W'bal
  • โซนการฝึกอบรม เปลี่ยนเกณฑ์มาตรฐานให้เป็นใบสั่งยารายวัน
  • อากาศพลศาสตร์ อธิบายว่าทำไมนักบิดคนเดียวกันจึงต้องใช้กำลังที่แตกต่างกันเพื่อรักษาความเร็วเท่ากัน
  • TSS แปลงความเข้มข้นบวกระยะเวลาเป็นคะแนนโหลดที่คุณสามารถเปรียบเทียบเมื่อเวลาผ่านไป

เหตุการณ์ที่มั่นคง

ตัวชี้วัดลำดับความสำคัญ: FTP, IF, VI และวินัยในการเว้นจังหวะ

ตัวอย่าง: การทดสอบครั้ง, การปีนไกล, ความพยายามโซโลอย่างต่อเนื่อง

เหตุการณ์ที่เปลี่ยนแปลงได้

ตัวชี้วัดลำดับความสำคัญ: NP, VI, W'bal และการฟื้นตัวระหว่างไฟกระชาก

ตัวอย่าง: เกณฑ์, จักรยานเสือภูเขา, การแข่งขันบนถนนที่ดุเดือด, การปีนผาทางเทคนิค

ตัวอย่างการปีนถนนที่มั่นคง

โปรไฟล์ไรเดอร์: FTP 300W

  • ระยะเวลา: 20 นาที
  • กำลังเฉลี่ย: 282วัตต์
  • NP: 286W
  • IF: 0.95
  • วี: 1.01

การตีความ: นี่เป็นความพยายามที่มั่นคงเกือบจะสมบูรณ์แบบ NP อยู่ใกล้กับกำลังเฉลี่ย, VI ยืนยันจังหวะที่สะอาด และ IF แสดงให้เห็นว่าผู้ขับขี่อยู่ภายใต้เกณฑ์ สำหรับความพยายามประเภทนี้ พลังโดยเฉลี่ยและบริบทเกณฑ์สามารถบอกเล่าเรื่องราวส่วนใหญ่ได้

ตัวอย่าง MTB / เกณฑ์ที่แปรผันได้

โปรไฟล์ไรเดอร์: FTP 300W, CP 305W, W' 18 กิโลจูล

  • ระยะเวลา: 60 นาที
  • กำลังเฉลี่ย: 228วัตต์
  • NP: 276W
  • IF: 0.92
  • วี: 1.21

การตีความ: การขับขี่จะดูปานกลางหากคุณอ่านได้เฉพาะกำลังเฉลี่ย แต่ NP และ VI แสดงการกระชากเหนือเกณฑ์ซ้ำแล้วซ้ำอีก นี่คือที่ที่ W'bal และ พลังวิกฤต เนื่องจากผู้ขับขี่ใช้จ่ายซ้ำๆ และฟื้นฟูความสามารถในการทำงานแบบไม่ใช้ออกซิเจนที่จำกัดบางส่วน

เหตุใดตัวอย่างเหล่านี้จึงมีความสำคัญ

นักบิดคนเดียวกันสามารถจบสองเซสชั่นด้วยกำลังเฉลี่ยที่ใกล้เคียงกันและต้นทุนทางสรีรวิทยาที่แตกต่างกันโดยสิ้นเชิง นั่นคือเหตุผลว่าทำไม NP, IF, VI และ W'bal จึงเป็นตัวชี้วัดที่เป็นมิตรต่อการอ้างอิง โดยอธิบายความแตกต่างระหว่างงานที่ทำกับความเครียดที่ได้รับ

สรุปเมตริกต่อเมตริก

กำลังไฟฟ้ามาตรฐาน (NP)

เอ็นพี ประมาณการค่าใช้จ่ายในการขับขี่หลังจากการถ่วงน้ำหนักเพิ่มขึ้นมากกว่าการขี่แบบคงที่ จำเป็นสำหรับการตีความการปั่นเป็นกลุ่มที่ยาก การวิ่งแบบโรลลิ่ง และไฟล์ MTB ทางเทคนิค ซึ่งกำลังโดยเฉลี่ยจะซ่อนความตึงเครียดที่แท้จริงไว้

ปัจจัยความเข้ม (IF)

IF เป็นวิธีที่ง่ายที่สุดในการวางรถตามเกณฑ์ขั้นต่ำ มันจะมีประโยชน์ก็ต่อเมื่อคุณ FTP น่าเชื่อถือ หาก FTP เก่าแล้ว IF จะกลายเป็นค่าความแม่นยำที่ผิดพลาด

ดัชนีความแปรปรวน (VI)

วี บอกคุณว่าความพยายามนั้นคงที่หรือวุ่นวาย ในการทดลองใช้เวลา ค่า VI ที่ต่ำมักจะเป็นจังหวะที่ดี ในการแข่งขันหรือเกณฑ์การแข่งขัน XC นั้น VI ที่สูงกว่าอาจสะท้อนถึงความต้องการของการแข่งขันได้

วาบาล

วาบาล ช่วยให้ผู้ขับขี่ที่จริงจังสามารถตีความการทำงานซ้ำๆ ข้างต้นได้ พลังวิกฤต. ซึ่งจะมีประโยชน์มากที่สุดเมื่อจับคู่กับบริบทการแข่งขัน ไม่ใช่เป็นหมายเลขแดชบอร์ดสากลสำหรับการขับขี่ทุกครั้ง

ดำเนินการต่อด้วยคลัสเตอร์

FTP

เริ่มต้นที่นี่หากคุณต้องการจุดยึดเกณฑ์ที่ทำให้ IF และโซนการฝึกใช้งานได้

พลังวิกฤต

ให้เจาะลึกกว่านี้หากการขี่ของคุณขึ้นอยู่กับความพยายามซ้ำๆ ที่สูงกว่าเกณฑ์และการตีความ W

โซนการฝึกอบรม

ใช้หน้านี้เพื่อเปลี่ยนจำนวนเกณฑ์ให้เป็นโครงสร้างเซสชันและการแจกแจงรายสัปดาห์

อากาศพลศาสตร์

อ่านข้อความต่อไปนี้หากคุณต้องการเชื่อมต่อกำลังเอาท์พุตเข้ากับความเร็ว การลาก และการกำหนดจังหวะในโลกแห่งความเป็นจริง

เครื่องคิดเลข TSS

หากคุณเข้าใจเกณฑ์และความเข้มข้นของการปั่นแล้ว ขั้นตอนการปฏิบัติถัดไปคือการให้คะแนนภาระการฝึกซ้อมด้วย TSS เพื่อให้การปั่นแต่ละครั้งสามารถแจ้งการตัดสินใจรายสัปดาห์และระดับบล็อกได้

คำถามที่พบบ่อย

เหตุใด NP จึงมักสูงกว่ากำลังเฉลี่ย

เพราะไฟกระชากอย่างหนักทำให้เกิดต้นทุนทางสรีรวิทยาที่ไม่สมส่วน NP จะถ่วงน้ำหนักการกระชากเหล่านั้นหนักกว่า ดังนั้นการขับขี่แบบแปรผันมักจะดูยากขึ้นเมื่อผ่าน NP มากกว่าการใช้กำลังโดยเฉลี่ยเพียงอย่างเดียว

VI ที่ดีสำหรับการปั่นจักรยานคืออะไร?

ไม่มี VI ที่ดีสากล การทดลองใช้เวลาอาจกำหนดเป้าหมายประมาณ 1.00-1.03 ในขณะที่เกณฑ์หรือการแข่งขัน XC สามารถนั่งได้สูงกว่ามาก ตัดสิน VI เทียบกับเหตุการณ์นี้ ไม่ใช่เกณฑ์มาตรฐานระดับโลกเดียว

ฉันจำเป็นต้องมีพลังวิกฤตหรือไม่หากฉันใช้ FTP อยู่แล้ว?

ไม่เสมอไป FTP ก็เพียงพอแล้วสำหรับนักบิดหลายๆ คน พลังคริติคอลจะมีประโยชน์มากขึ้นเมื่อคุณสนใจงานเหนือเกณฑ์ซ้ำๆ, W'bal และแบบจำลองที่แข็งแกร่งยิ่งขึ้นของความพยายามที่มีความเข้มข้นรุนแรง

ฉันควรอัปเดตเมตริกเหล่านี้บ่อยแค่ไหน

ทดสอบซ้ำหรือประเมินจุดยึดเกณฑ์ของคุณใหม่ทุกๆ 6-8 สัปดาห์หรือหลังบล็อกการฝึกที่มีความหมาย เมตริกจะมีประโยชน์พอๆ กับค่าเกณฑ์ที่อยู่เบื้องหลังเท่านั้น
