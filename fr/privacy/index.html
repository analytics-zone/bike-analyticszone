Politique de Confidentialité

Dernière mise à jour : 27 octobre 2025

Votre Vie Privée est Protégée

BikeAnalytics ne collecte, ne stocke ni ne transmet aucune de vos données personnelles. Tout le traitement des données se fait localement sur votre appareil.

Vue d'Ensemble

BikeAnalytics est conçue avec votre vie privée comme priorité absolue. Cette Politique de Confidentialité explique notre engagement à protéger vos informations personnelles et vos données.

En bref : Nous ne collectons pas vos données parce que nous n'en avons pas besoin. Tout ce que vous faites dans l'application reste sur votre appareil.

Informations que Nous NE Collectons PAS

BikeAnalytics ne collecte, ne transmet ni ne stocke :

  • Informations personnelles (nom, email, numéro de téléphone, etc.)
  • Données d'activité cycliste (distance, puissance, vitesse, temps, etc.)
  • Données de santé (fréquence cardiaque, calories, etc.)
  • Données de localisation ou coordonnées GPS
  • Informations sur l'appareil ou identifiants
  • Statistiques d'utilisation ou analyses
  • Rapports de plantage ou diagnostics
  • Toute autre information personnelle ou sensible

Comment Fonctionne l'Application

Tous les calculs et le traitement des données dans BikeAnalytics se produisent entièrement sur votre appareil :

  • Vos données cyclistes sont stockées uniquement dans le stockage local de votre appareil
  • Tous les calculs (puissance, FTP, TSS, zones, etc.) sont effectués localement
  • Aucune donnée n'est jamais envoyée à nos serveurs ou à des services tiers
  • Aucune connexion Internet n'est requise pour que l'application fonctionne

Stockage des Données

Toutes les données que vous saisissez ou générez dans l'application sont stockées exclusivement sur votre appareil en utilisant les mécanismes de stockage local iOS. Ces données restent sous votre contrôle complet :

  • Vos données restent sur votre appareil en permanence
  • Nous ne pouvons pas accéder à vos données
  • Si vous supprimez l'application, toutes les données sont définitivement supprimées de votre appareil
  • Les données ne sont pas synchronisées avec iCloud ou tout service cloud sauf si vous activez explicitement les sauvegardes d'applications iOS

Services Tiers

BikeAnalytics n'utilise aucun service tiers, y compris :

  • Aucun service d'analyse (Google Analytics, Firebase, etc.)
  • Aucun réseau publicitaire
  • Aucun service de rapport de plantage
  • Aucune intégration de réseaux sociaux
  • Aucun processeur de paiement (l'application est gratuite)

Autorisations

L'application peut demander les autorisations iOS suivantes uniquement si vous choisissez d'utiliser des fonctionnalités spécifiques :

  • HealthKit/Application Health : Uniquement si vous choisissez d'importer des données d'entraînement. Toutes les données restent sur votre appareil.
  • Notifications : Uniquement si vous activez les rappels d'entraînement. Aucune donnée n'est envoyée hors de votre appareil.

Toutes les autorisations sont facultatives et vous pouvez les révoquer à tout moment via les Réglages iOS.

Confidentialité des Enfants

Parce que nous ne collectons aucune donnée, BikeAnalytics est sûre pour les utilisateurs de tous âges, y compris les enfants de moins de 13 ans. Cependant, nous recommandons l'encadrement parental pour toutes les activités physiques.

Modifications de cette Politique de Confidentialité

Nous pouvons mettre à jour cette Politique de Confidentialité de temps en temps pour refléter les changements dans l'application ou les exigences légales. Toute modification sera publiée sur cette page avec une date de "Dernière mise à jour" actualisée.

Important : Nous ne changerons jamais notre engagement fondamental à ne pas collecter vos données.

Contactez-nous

Si vous avez des questions sur cette Politique de Confidentialité ou l'application, veuillez nous contacter via notre page de contact.

Résumé

BikeAnalytics est une application axée sur la confidentialité. Nous croyons que vos données vous appartiennent et doivent rester avec vous.

  • ❌ Aucune collecte de données
  • ❌ Aucun suivi ou analyse
  • ❌ Aucun service tiers
  • ✅ Traitement 100% local
  • ✅ Contrôle complet sur vos données
