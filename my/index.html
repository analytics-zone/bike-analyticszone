🚴ပရော်ဖက်ရှင်နယ် စက်ဘီးစီး ခွဲခြမ်းစိတ်ဖြာမှု

ပိုမို ထက်မြက်စွာ လေ့ကျင့်ပါ၊ ပိုမို မြန်ဆန်စွာ စီးနင်းပါ၊ ပိုမို အားကောင်းစွာ တက်လှမ်းပါ

Road စက်ဘီးသမားများနှင့် တောင်တက် စက်ဘီးသမားများအတွက် FTP၊ TSS နှင့် စွမ်းဆောင်ရည် ခြေရာခံခြင်းတို့ ပါဝင်သော စွမ်းအား-အခြေခံ iOS အက်ပ်။ သင်၏ iPhone ပေါ်တွင်သာ လုံခြုံစိတ်ချစွာ locale-based လုပ်ဆောင်ပေးသည်။

ကျွန်ုပ်တို့၏ ဖြေရှင်းနည်းများကို လေ့လာပါ

လုပ်ဆောင်ချက်များ

အဆင့်မြင့် စက်ဘီးစီး စွမ်းဆောင်ရည် Metrics

အဆင့်တိုင်းရှိ စက်ဘီးသမားများအတွက် ရည်ရွယ်ထားသော ပရော်ဖက်ရှင်နယ်-အဆင့် စက်ဘီးစီး ခွဲခြမ်းစိတ်ဖြာမှု

သိပ္ပံနည်းကျ စွမ်းအား Metrics

FTP (Functional Threshold Power) သည် သင်၏ threshold ကို သတ်မှတ်ပေးပြီး၊ ခိုင်မာသော အားကစား သိပ္ပံ သုတေသနများအပေါ် အခြေခံ၍ Training Stress Score (TSS) တွက်ချက်ခြင်းနှင့် CTL/ATL/TSB စွမ်းဆောင်ရည် ခြေရာခံခြင်း တို့ကို လုပ်ဆောင်ပေးသည်။

တစ်ဦးချင်း စိတ်ကြိုက် လေ့ကျင့်မှု ဇုန်များ

သင်၏ FTP နှင့်အညီ ချိန်ညှိထားသော တစ်ဦးချင်း စိတ်ကြိုက် စွမ်းအား-အခြေခံ လေ့ကျင့်မှု ဇုန် ၇ ခု။ ပြန်လည်ကောင်းမွန်ရေး၊ aerobic ဖွံ့ဖြိုးတိုးတက်ရေး၊ threshold လေ့ကျင့်မှု သို့မဟုတ် VO₂max တိုးတက်ရေးတို့အတွက် စီးနင်းမှုတိုင်းကို အကောင်းဆုံး ဖြစ်အောင် လုပ်ဆောင်ပါ။

Road နှင့် MTB Modes

Road စက်ဘီးစီးခြင်းနှင့် တောင်တက် စက်ဘီးစီးခြင်း နှစ်ခုစလုံးအတွက် အထူးပြု ခွဲခြမ်းစိတ်ဖြာမှုများ ပါဝင်ပြီး၊ စီးနင်းမှု ပုံစံတစ်ခုစီနှင့် ကိုက်ညီသော မတူညီသည့် စွမ်းအား ညှိနှိုင်းမှု Algorithms များနှင့် metrics များ ပါဝင်သည်။

ပြည့်စုံသော ကိုယ်ရေးကိုယ်တာ ကာကွယ်မှု

စက်ဘီးစီး ဒေတာ အားလုံးကို သင်၏ iOS စက်ပစ္စည်းပေါ်တွင်သာ local-based လုပ်ဆောင်သည်။ Server များ၊ cloud storage များ သို့မဟုတ် ခြေရာခံခြင်းများ မရှိပါ။ သင်၏ စက်ဘီးစီး ခွဲခြမ်းစိတ်ဖြာမှုများကို သင်ကိုယ်တိုင် ပိုင်ဆိုင်ပြီး ထိန်းချုပ်နိုင်သည်။

မည်သည့်နေရာသို့မဆို Export လုပ်နိုင်ခြင်း

စီးနင်းမှုများနှင့် စက်ဘီးစီး စွမ်းဆောင်ရည် metrics များကို JSON, CSV, HTML သို့မဟုတ် PDF ပုံစံများဖြင့် ထုတ်ယူပါ။ နည်းပြများ၊ spreadsheets များနှင့် လေ့ကျင့်ရေး ပလပ်ဖောင်းများနှင့် လိုက်ဖက်မှု ရှိသည်။

မြန်ဆန်သော စွမ်းဆောင်ရည်

Local-first architecture ဖြင့် ၀.၃၅ စက္ကန့်အတွင်း အက်ပ်ကို ဖွင့်နိုင်သည်။ Sync များ သို့မဟုတ် ဒေါင်းလုဒ်များကို စောင့်ဆိုင်းနေစရာ မလိုဘဲ သင်၏ စက်ဘီးစီး ခွဲခြမ်းစိတ်ဖြာမှုများကို ချက်ချင်း ကြည့်ရှုနိုင်သည်။

ရှုပ်ထွေးမှုများ မရှိဘဲ ပရော်ဖက်ရှင်နယ် စက်ဘီးစီး စွမ်းဆောင်ရည် Metrics များ

🎯

FTP စမ်းသပ်မှု ပရိုတိုကော

သင်၏ functional threshold power ကို ဆုံးဖြတ်ရန် မိနစ်-၂၀ FTP စမ်းသပ်မှု ပရိုတိုကော ပါဝင်သည်။ တိုးတက်မှုကို ခြေရာခံရန်နှင့် စက်ဘီးစီး လေ့ကျင့်မှု ဇုန်များကို အလိုအလျောက် ချိန်ညှိရန် ၆-၈ ပတ်တိုင်း ပြုလုပ်ပါ။

📱

Native iOS စက်ဘီးစီး အက်ပ်

မြန်ဆန်သော စွမ်းဆောင်ရည်နှင့် iOS ပေါင်းစည်းမှုအတွက် SwiftUI ဖြင့် တည်ဆောက်ထားသည်။ စက်ဘီးစီး ခွဲခြမ်းစိတ်ဖြာမှုများ၊ widgets ထောက်ပံ့မှုနှင့် ရင်းနှီးပြီးသား Apple design language များအတွက် Health အက်ပ်နှင့် ချောမွေ့စွာ sync လုပ်နိုင်သည်။။

🔬

သုတေသနအခြေပြု Metrics

စက်ဘီးစီး စွမ်းဆောင်ရည် metrics အားလုံးသည် ပညာရပ်ဆိုင်ရာ သုတေသနများအပေါ် အခြေခံထားသည်။ Andrew Coggan ထံမှ FTP၊ IF ဖော်မြူလာပါသော Training Stress Score၊ အတည်ပြုထားသော CTL/ATL မော်ဒယ်များ။

👥

နည်းပြများအတွက် အဆင်ပြေသော အစီရင်ခံစာများ

နည်းပြများအတွက် အသေးစိတ် စက်ဘီးစီး ခွဲခြမ်းစိတ်ဖြာမှု အစီရင်ခံစာများကို Export လုပ်ပါ။ အီးမေးလ်မှတစ်ဆင့် HTML အနှစ်ချုပ်များ၊ spreadsheet ခွဲခြမ်းစိတ်ဖြာမှုအတွက် CSV သို့မဟုတ် လေ့ကျင့်မှု မှတ်တမ်းများအတွက် PDF တို့ကို မျှဝေပါ။

FAQ

မေးလေ့ရှိသော မေးခွန်းများ

ဤစက်ဘီးစီး ခွဲခြမ်းစိတ်ဖြာမှု အက်ပ်က ကျွန်ုပ်၏ ဒေတာကို မည်သို့ ရရှိသလဲ?

Bike Analytics သည် လိုက်ဖက်သော မည်သည့်စက်ပစ္စည်း သို့မဟုတ် အက်ပ်မှမဆို မှတ်တမ်းတင်ထားသော စက်ဘီးစီး လေ့ကျင့်ခန်းများကို import လုပ်ရန် Apple Health နှင့် sync လုပ်သည်။ ၎င်းတွင် စက်ဘီးကွန်ပျူတာများ၊ smart trainers များနှင့် လက်ဖြင့် ထည့်သွင်းမှုများ ပါဝင်သည်။ အက်ပ်သည် အဆင့်မြင့် စက်ဘီးစီး စွမ်းဆောင်ရည် metrics များကို တွက်ချက်ရန် ဤဒေတာများကို locale-based လုပ်ဆောင်သည်။

FTP ဆိုတာ ဘာလဲ၊ ဘယ်လို စမ်းသပ်မလဲ?

FTP (Functional Threshold Power) သည် သင် တစ်နာရီခန့် ရေရှည်ထိန်းထားနိုင်သော အမြင့်ဆုံး စွမ်းအား ဖြစ်သည်။ အက်ပ်တွင် မိနစ်-၂၀ FTP စမ်းသပ်မှု ပရိုတိုကော ပါဝင်သည်: ကောင်းစွာ သွေးပူပြီးနောက် မိနစ် ၂၀ ကြာ အမြင့်ဆုံး ရေရှည်တည်တံ့သော အားထုတ်မှုဖြင့် စီးနင်းပါ။ သင်၏ FTP သည် ပျမ်းမျှ စွမ်းအား၏ ၉၅% ဖြစ်သည်။ တိုးတက်မှုကို ခြေရာခံရန်နှင့် လေ့ကျင့်မှု ဇုန်များကို update လုပ်ရန် ၆-၈ ပတ်တိုင်း ပြုလုပ်ပါ။

Bike Analytics ကို သုံးဖို့ Power Meter လိုအပ်ပါသလား?

ဟုတ်ကဲ့။ တိကျသော FTP၊ TSS နှင့် လေ့ကျင့်မှု ဇုန် တွက်ချက်မှုများအတွက် Bike Analytics သည် စွမ်းအား ဒေတာ လိုအပ်သည်။ စက်ဘီးကွန်ပျူတာများ သို့မဟုတ် smart trainers များမှတစ်ဆင့် Apple Health သို့ sync လုပ်သော power meter အားလုံးနှင့် လိုက်ဖက်မှု ရှိသည်။

စက်ဘီးစီး TSS ကို ဘယ်လို တွက်ချက်သလဲ?

စက်ဘီးစီးခြင်းအတွက် Training Stress Score (TSS) ကို စွမ်းအား ဒေတာနှင့် သင်၏ FTP သုံး၍ တွက်ချက်သည်။ ဖော်မြူလာသည် လေ့ကျင့်ခန်း ဝန်ပိမှုကို ပမာဏ သတ်မှတ်ရန် ပြင်းထန်မှု (normalized power vs FTP) နှင့် ကြာချိန် နှစ်ခုစလုံးကို ထည့်သွင်းစဉ်းစားသည်။ ၎င်းသည် CTL မှတစ်ဆင့် ကြံ့ခိုင်မှုကို တိကျစွာ ခြေရာခံခြင်း၊ ATL မှတစ်ဆင့် မောပန်းမှုကို စောင့်ကြည့်ခြင်းနှင့် TSB ဖြင့် ပုံစံကို အကဲဖြတ်ခြင်းတို့ကို လုပ်ဆောင်နိုင်စေသည်။

Road နှင့် MTB ခွဲခြမ်းစိတ်ဖြာမှု အကြား ဘာကွာခြားသလဲ?

Bike Analytics သည် road စက်ဘီးစီးခြင်းနှင့် တောင်တက် စက်ဘီးစီးခြင်းတို့အတွက် အထူးပြု ခွဲခြမ်းစိတ်ဖြာမှု modes များ ပေးထားသည်။ Road mode သည် ပုံမှန် စက္ကန့် ၃၀ power smoothing ကို အသုံးပြုပြီး၊ MTB mode သည် တောလမ်းစီးနင်းမှု၏ ထူးခြားသော မြေပြင်အနေအထားနှင့် အားထုတ်မှုများအတွက် မတူညီသော algorithms များကို အသုံးပြုသည်။ mode တစ်ခုစီသည် သက်ဆိုင်ရာ metrics များကို ပေးသည်။

ကျွန်ုပ်၏ စက်ဘီးစီး ဒေတာများ လုံခြုံပါသလား?

ဟုတ်ကဲ့။ Bike Analytics သည် စက်ဘီးစီး ဒေတာ အားလုံးကို သင်၏ iPhone ပေါ်တွင်သာ local-based လုပ်ဆောင်သည်။ ပြင်ပ server များ၊ cloud အကောင့်များ သို့မဟုတ် ဒေတာ လွှဲပြောင်းမှုများ မရှိပါ။ သင်သည် export များကို ထိန်းချုပ်နိုင်သည်: သင်၏ စက်ဘီးစီး စွမ်းဆောင်ရည် metrics များဖြင့် JSON, CSV, HTML သို့မဟုတ် PDF ဖိုင်များကို ဖန်တီးပြီး သင်အလိုရှိသလို မျှဝေနိုင်သည်။

ကျွန်ုပ်၏ စက်ဘီးစီး ဒေတာများကို နည်းပြများနှင့် မျှဝေရန် export လုပ်နိုင်ပါသလား?

သေချာပေါက် လုပ်နိုင်ပါသည်။ စီးနင်းမှုများနှင့် စွမ်းဆောင်ရည် metrics များကို ပုံစံမျိုးစုံဖြင့် ထုတ်ယူနိုင်သည်: developer များအတွက် JSON၊ spreadsheets အတွက် CSV၊ web ကြည့်ရှုရန် HTML သို့မဟုတ် ပရင့်ထုတ်နိုင်သော အစီရင်ခံစာများအတွက် PDF။ အီးမေးလ်၊ messaging သို့မဟုတ် သင်နှစ်သက်ရာ နည်းလမ်းဖြင့် မျှဝေပါ။

လစဉ်နှင့် နှစ်စဉ် အစီအစဉ်များအကြား ဘာကွာခြားသလဲ?

အစီအစဉ် နှစ်ခုလုံးတွင် တူညီသော လုပ်ဆောင်ချက်များ ပါဝင်သည်: စက်ဘီးစီး စွမ်းဆောင်ရည် metrics အားလုံး၊ အကန့်အသတ်မရှိ လေ့ကျင့်မှု ဇုန်များ၊ Road နှင့် MTB modes များ၊ export မျိုးစုံနှင့် အခမဲ့ update များ။ ကွာခြားချက်မှာ စျေးနှုန်းသာ ဖြစ်သည်: နှစ်စဉ် အစီအစဉ်သည် ၁၈% သက်သာသည် (တစ်လလျှင် €3.25 နှင့် €3.99 ယှဉ်လျှင်)။

ကျွန်ုပ်၏ စာရင်းပေးသွင်းမှုကို အချိန်မရွေး ပယ်ဖျက်နိုင်ပါသလား?

ဟုတ်ကဲ့။ စာရင်းပေးသွင်းမှုများကို App Store မှတစ်ဆင့် စီမံခန့်ခွဲသောကြောင့် Settings → [သင်၏အမည်] → Subscriptions မှတစ်ဆင့် အချိန်မရွေး ပယ်ဖျက်နိုင်သည်။ အကယ်၍ ပယ်ဖျက်ပါက လက်ရှိ ပေးချေထားသော ကာလ ကုန်ဆုံးသည်အထိ ဆက်လက် အသုံးပြုနိုင်မည် ဖြစ်သည်။

သင်၏ စက်ဘီးစီး စွမ်းဆောင်ရည်ကို ပြောင်းလဲရန် အဆင်သင့်ဖြစ်ပြီလား?

စွမ်းဆောင်ရည် မြှင့်တင်ရန် သိပ္ပံနည်းကျ စွမ်းအား-အခြေခံ ခွဲခြမ်းစိတ်ဖြာမှုများကို အသုံးပြုနေသော ထောင်ပေါင်းများစွာသော စက်ဘီးသမားများနှင့် ပူးပေါင်းလိုက်ပါ။ ကိုယ်ရေးကိုယ်တာ ဦးစားပေး ဤ iOS စက်ဘီးစီး အက်ပ်ကို ယနေ့တွင် ၇ ရက်တာ အခမဲ့ စမ်းသပ်မှုဖြင့် စတင်လိုက်ပါ။

အကောင့်ဖွင့်ရန် မလိုပါ • အချိန်မရွေး ပယ်ဖျက်နိုင်သည် • သင်၏ ဒေတာများသည် သင့် စက်ပစ္စည်းပေါ်တွင်သာ ရှိနေမည်

စက်ဘီးစီး စွမ်းဆောင်ရည် Metrics အကြောင်း ပိုမို လေ့လာရန်

စက်ဘီးစီး ခွဲခြမ်းစိတ်ဖြာမှုများ၏ နောက်ကွယ်မှ သိပ္ပံပညာကို ပိုမို နက်ရှိုင်းစွာ လေ့လာပါ

VO2max ဆိုတာ ဘာလဲ?

စက်ဘီးသမားများအတွက် VO2max အကြောင်း၊ ၎င်းကို မည်သို့ စမ်းသပ်ရမည်၊ အသက်အရွယ်အလိုက် ပျမ်းမျှ တန်ဖိုးများနှင့် aerobic စွမ်းရည် တိုးတက်စေရန် သက်သေပြထားသော နည်းလမ်းများကို လေ့လာပါ။
