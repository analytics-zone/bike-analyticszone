Cycling Power Metrics ရှင်းပြထားသည်- NP၊ IF၊ VI၊ W'bal

ပိုမိုထက်မြက်သော လေ့ကျင့်မှုနှင့် ပြိုင်ကားများအတွက် အဆင့်မြင့် ပါဝါမက်ထရစ်များကို နားလည်ခြင်း။

ပျမ်းမျှစွမ်းအားကိုကျော်လွန်ခြင်း- အဘယ်ကြောင့် အဆင့်မြင့်မက်ထရစ်များ အရေးကြီးသနည်း။

ပျမ်းမျှအားသည် တုံးသော တူရိယာတစ်ခုဖြစ်သည်။ တူညီသောပျမ်းမျှစွမ်းအားရှိသော စီးနင်းမှုနှစ်ခုသည် ကြိုးစားအားထုတ်မှု မည်မျှပြောင်းလဲနိုင်သည်အပေါ် မူတည်၍ ဇီဝကမ္မဆိုင်ရာကုန်ကျစရိတ်များ ကွာခြားလှသည်။ တစ်နာရီအတွက် တည်ငြိမ်သော 250W သည် 150W နှင့် 400W အကြား ပျမ်းမျှ 250W မှ လုံးဝကွဲပြားသည်ဟု ခံစားရသည်။

ဤနေရာတွင် အဆင့်မြင့် ပါဝါမက်ထရစ်များ ဝင်လာပါသည်။ပုံမှန်ဓာတ်အား (NP)၊ပြင်းထန်မှုအချက် (IF)၊ကွဲပြားမှုအညွှန်းကိန်း (VI), နှင့်W' Balance (W'bal)ထိထိရောက်ရောက် လေ့ကျင့်ရန်၊ ပြိုင်ပွဲများကို မှန်ကန်စွာ အရှိန်မြှင့်ရန်နှင့် သင်၏ ကြိုးစားအားထုတ်မှု၏ စစ်မှန်သော ကုန်ကျစရိတ်ကို နားလည်ရန် လိုအပ်သော သိမ်မွေ့သော နားလည်မှုကို ပေးဆောင်ပါ။

အမြန်နှိုင်းယှဉ်မှု- လမ်းနှင့် MTB

  • လမ်းပေါ်တက်ခြင်း (၁၀ မိနစ်)။246W ပျမ်းမျှ၊ 246W NP၊ VI = 1.00
  • MTB တောင်တက် (၁၀ မိနစ်)။220W ပျမ်းမျှ၊ 265W NP၊ VI = 1.20
  • ရလဒ်-MTB တို့နဲ့ အားထုတ်တယ်။အောက်ပိုင်းပျမ်းမျှစွမ်းအားသည် အမှန်တကယ်ဖြစ်သည်။ပိုခက်တယ်။ဇီဝဗေဒအရ

Normalized Power (NP): ပြောင်းလဲနိုင်သော ကြိုးပမ်းမှုများ၏ စစ်မှန်သော ကုန်ကျစရိတ်

ပုံမှန်ဓာတ်အား (NP)ပါဝါထွက်ရှိမှုနှင့် ပင်ပန်းနွမ်းနယ်မှုကြား လိုင်းမညီသော ဆက်နွယ်မှုကို တွက်ချက်ခြင်းဖြင့် စီးနင်းမှု၏ ဇီဝကမ္မ "ကုန်ကျစရိတ်" ကို ခန့်မှန်းသည်။ သတ်မှတ်ချက်ထက် ပြင်းထန်သော အားထုတ်မှုများသည် တည်ငြိမ်သော စီးနင်းမှုနှင့် နှိုင်းယှဉ်ပါက ပင်ပန်းနွမ်းနယ်မှုကို အချိုးမညီမျှစွာ တိုးစေသည်။

NP နောက်ကွယ်မှ အယ်လဂိုရီသမ်

NP သည် ခက်ခဲသောအားထုတ်မှုများကို အလေးပေးသည့် အလေးချိန်တွက်ချက်မှုကို အသုံးပြုသည်-

  1. စက္ကန့် 30 လှည့်ပတ်ပျမ်းမျှ-တစ်စက္ကန့်မှ စက္ကန့်အထိ ဆူညံသံများကို ပြေလျော့စေပါသည်။
  2. 4th ပါဝါကို မြှင့်တင်ပါ။ပြင်းထန်မှုမြင့်မားသော ကြိုးပမ်းမှုများကို ချဲ့ထွင်ပါ (200W⁴ နှင့် 300W⁴ ကြီးမားသည်)
  3. ပျမ်းမျှ 4th ပါဝါတန်ဖိုးများ-တွက်ဆအားထုတ်မှု၏ ဆိုလိုရင်းကို ရှာပါ။
  4. 4th root ကိုယူပါ။ဝပ်သို့ ပြန်ပြောင်းပါ။

ရိုးရှင်းသောဖော်မြူလာ

NP = ⁴√(ပျမ်းမျှ [30s rolling average power]⁴)

4th power weighting ဆိုသည်မှာ 200W ကို 20 စက္ကန့်စီးခြင်းထက် သင့်ခန္ဓာကိုယ်ကို 10 စက္ကန့်အတွင်း 400W "ကုန်ကျစရိတ်" ပိုများစေသည်။

NP အရေးအကြီးဆုံးအချိန်

🚵 တောင်တက်စက်ဘီးစီးခြင်း။

MTB ပြိုင်ကားကို ပြောင်းလဲနိုင်သော ပါဝါဖြင့် သတ်မှတ်သည်။ ပုံမှန် XC ပြိုင်ပွဲတွင် FTP ထက် 80+ surges တစ်ခုစီတွင် 5-25 စက္ကန့်ကြာသည်။ ပျမ်းမျှစွမ်းအားသည် စစ်မှန်သောအားထုတ်မှုကို ပြင်းပြင်းထန်ထန် လျှော့တွက်သည်။

ဥပမာ XC ပြိုင်ပွဲ

  • ပျမ်းမျှ ပါဝါ- 245W
  • ပုံမှန် ပါဝါ- 285W
  • VI- 1.16 (အလွန်ပြောင်းလဲနိုင်သော)
  • စကားပြန်-ပြိုင်ပွဲသည် 285W မဟုတ်ဘဲ 245W တည်ငြိမ်စွာ စီးရသကဲ့သို့ ခံစားရပါသည်။

🏁 သတ်မှတ်ချက်များနှင့် ပတ်လမ်းပြိုင်ပွဲများ

စံသတ်မှတ်ချက်များသည် ထောင့်များ၊ တိုက်ခိုက်မှုများနှင့် နေရာချထားရန် အားထုတ်မှုများမှ အဆက်မပြတ် အရှိန်အဟုန်ဖြင့် ပါရှိသည်။ Surge-and-recover ပုံစံသည် ပျမ်းမျှပါဝါနှင့် မြင့်မားသော NP ဆက်စပ်မှုကို ဖန်တီးပေးသည်။

နမူနာစံသတ်မှတ်ချက်-

  • ပျမ်းမျှ ပါဝါ- 220W
  • ပုံမှန် ပါဝါ- 265W
  • VI: ၁.၂၀
  • မိနစ် 60 အတွင်း 88 အရှိန်> 300W

🚴 Surges များဖြင့် အုပ်စုဖွဲ့စီးခြင်း။

ရန်လိုသောအုပ်စုသည် တိုက်ခိုက်မှုများနှင့် ပေါင်းကူးရန်ကြိုးပမ်းမှုများဖြင့် စီးနင်းမှုအရှိန်အဟုန် အလယ်အလတ်ဟုခံစားရသည့်တိုင် မြင့်မားသော NP ကိုထုတ်ပေးသည်။

NP မကူညီသောအခါ

အချိန်စမ်းသပ်မှုများ၊ မှန်မှန်တက်ခြင်းနှင့် တစ်ကိုယ်တော် tempo စီးနင်းခြင်းသည် ပျမ်းမျှစွမ်းအား (VI = 1.00-1.03) နှင့် NP နီးပါးကို ထုတ်ပေးသည်။ ဤအခြေအနေမျိုးတွင်၊ ပျမ်းမျှအားသည်လုံလောက်သည်။

🔬 သုတေသနဖောင်ဒေးရှင်း

ဒေါက်တာ အင်ဒရူး ကော့ဂန်သည် ATP လျော့နည်းခြင်း၊ glycogen အသုံးချခြင်း၊ နို့တိုက်ခြင်း စုဆောင်းခြင်းနှင့် နှလုံးသွေးကြောဆိုင်ရာ ဖိစီးမှု ပြင်းထန်မှုနှင့်အတူ အထစ်အငေါ့မရှိ—တစ်ပြေးညီမဟုတ်—ပြင်းထန်မှုဖြင့် တိုးလာကြောင်းပြသသည့် ဇီဝကမ္မသုတေသနကို အခြေခံ၍ ဒေါက်တာ Andrew Coggan သည် ပုံမှန်စွမ်းအင်ကို တီထွင်ခဲ့သည်။ 4th ပါဝါဆက်ဆံရေးသည် ဤလိုင်းမဟုတ်သော ပင်ပန်းနွမ်းနယ်မှုတုံ့ပြန်မှုကို ခန့်မှန်းပါသည်။

အရင်းအမြစ်-Allen, H., & Coggan, A.R. (၂၀၁၉)။ပါဝါမီတာဖြင့် လေ့ကျင့်ခြင်းနှင့် ပြိုင်ခြင်း။(၃)ကြိမ်မြောက်။ VeloPress

ပြင်းထန်မှုအချက် (IF)- နှိုင်းရအားထုတ်မှု အရေအတွက်

ပြင်းထန်မှုအချက် (IF)သင်စီးရတာ ဘယ်လောက်ခက်ခဲကြောင်း ဖော်ပြတယ်။လုပ်ဆောင်နိုင်သော အဆင့်သတ်မှတ်ပါဝါ (FTP). ၎င်းသည် Normalized Power ၏ FTP အချိုးဖြစ်သည်။

ဖော်မြူလာ

IF = ပုံမှန်စွမ်းအင် (NP) / FTP

ဥပမာ-

300W ၏ FTP ရှိသော Rider သည် 255W ၏ NP ဖြင့် ပြီးဆုံးသည်-

IF = 255 / 300 =၀.၈၅

ဒါကို ကိုယ်စားပြုတယ်။တော်ရုံတန်ရုံ-ခက်ကြိုးစားအားထုတ်မှု။

Intensity Factor ကို ဘာသာပြန်ခြင်း။

IF အတိုင်းအတာ အားထုတ်မှုအဆင့် နမူနာလေ့ကျင့်ခန်းများ ရေရှည်တည်တံ့သောကြာချိန်
< 0.65 လွယ်ကူသော / ပြန်လည်ထူထောင်ရေး ပြန်လည်ထူထောင်ရေးလှည့်ခြင်း၊ လွယ်ကူသောအုပ်စုလိုက်စီးခြင်း။ ၃-၆+ နာရီ
၀.၆၅-၀.၇၅ ခံနိုင်ရည်/အလယ်အလတ် အကြာကြီး မှန်မှန်စီး၊ အခြေခံလေ့ကျင့်မှု 2-5 နာရီ
၀.၇၅-၀.၈၅ Tempo/Moderate-Hard Tempo ကြားကာလ၊ အားကစားအရှိန်အဟုန် 1-3 နာရီ
၀.၈၅-၀.၉၅ တံခါးခုံ/ခက် အမွှေးအပြောက်၊ တံခါးခုံကြားကာလ 40-90 မိနစ်
၀.၉၅-၁.၀၅ FTP/အလွန်ခက်ခဲသည်။ FTP စမ်းသပ်မှု၊ 40km TT 30-60 မိနစ်
၁.၀၅-၁.၁၅ VO2max/ အလွန်ခက်ခဲသည်။ VO2max ကြားကာလ၊ သတ်မှတ်ချက် 10-30 မိနစ်
> ၁.၁၅ Anaerobic/Maximal တိုတောင်းသော TT၊ ခြေရာခံလိုက်စားခြင်း။ < 10 မိနစ်

လေ့ကျင့်မှုအတွက် IF ကို အသုံးပြုခြင်း။

Session ဒီဇိုင်း

ပစ်မှတ် IF သည် လေ့ကျင့်ခန်းပြင်းထန်မှုကို ညွှန်းပေးသည်-

  • လွယ်ကူသောနေ့IF < 0.65 သည် ပြန်လည်ရယူခြင်းကို သေချာစေသည်။
  • Tempo စက်ရှင်-IF 0.80-0.85 သည် အေရိုးဗစ်စွမ်းရည်ကို တည်ဆောက်သည်။
  • အဆင့်သတ်မှတ်ချက် အလုပ်-IF 0.90-1.00 သည် FTP ကို တိုးတက်စေသည်။

လူမျိုးရေး ဆန်းစစ်ခြင်း

ပြိုင်ပွဲလွန် IF သည် အရှိန်အဟုန်သည် သင့်လျော်မှုရှိမရှိ ဖော်ပြသည်-

  • IF မြင့်မားလွန်းသည်-ခက်ခက်ခဲခဲ စတင်ခဲ့ပြီး နောက်ဆုံးတွင် မှိန်သွားခဲ့သည်။
  • IF သင့်လျော်သည်-ကြိုးစားအားထုတ်မှုပင်လျှင် ခိုင်ခံ့သော ပြီးဆုံးမှု
  • IF အလွန်နည်းသည်-အလွန်အကျွံ ဆုပ်ကိုင်ထားပြီး အားအင်တွေ ကျန်နေခဲ့တယ်။

TSS တွက်ချက်မှုတွင် IF

IF သည် အဓိကအစိတ်အပိုင်းတစ်ခုဖြစ်သည်။လေ့ကျင့်ရေးစိတ်ဖိစီးမှုရမှတ် (TSS)ပြင်းထန်မှုနှင့် ကြာချိန်တို့ကို ပေါင်းစပ်ခြင်းဖြင့် စုစုပေါင်းလေ့ကျင့်ရေးဝန်အား ပမာဏကို တွက်ချက်ပေးသည်။

TSS ဖော်မြူလာ

TSS = (IF)² × ကြာချိန် (နာရီ) × 100

IF သည် နှစ်ထပ်ကိန်းဖြစ်ပြီး၊ ဆိုလိုသည်မှာ ပြင်းထန်မှုသည် လေ့ကျင့်ရေးဆိုင်ရာ ဖိစီးမှုအပေါ် ထပ်ကိန်းသက်ရောက်မှုရှိသည်။

ကွဲပြားမှုအညွှန်းကိန်း (VI): တည်ငြိမ်မှုမက်ထရစ်

ကွဲပြားမှုအညွှန်းကိန်း (VI)စီးနင်းစဉ်အတွင်း သင်၏ ပါဝါထွက်ရှိမှု မည်မျှပြောင်းလဲနိုင်သည်ကို တိုင်းတာသည်။ ၎င်းသည် ပုံမှန်စွမ်းအင်မှ ပျမ်းမျှစွမ်းအား အချိုးဖြစ်သည်။

ဖော်မြူလာ

VI = Normalized Power (NP)/Average Power

ဥပမာ-

NP ၏ 270W နှင့် ပျမ်းမျှစွမ်းအား 250W ဖြင့် စီးပါ

VI = 270 / 250 =၁.၀၈

အတန်အသင့် ပြောင်းလဲနိုင်သော အားထုတ်မှု (အချို့သော လှိုင်းများနှင့်အတူ အုပ်စုလိုက်စီးခြင်းအတွက် ပုံမှန်)။

VI Benchmarks စည်းကမ်း

🚴 လမ်းတွင် စက်ဘီးစီးခြင်း။

စီးခြင်းအမျိုးအစား ရိုးရိုး VI လက္ခဏာများ
အချိန်အစမ်း ၁.၀၀-၁.၀၂ ပြီးပြည့်စုံသော တည်ငြိမ်မှု၊ အကောင်းဆုံး အရှိန်အဟုန်
တစ်ကိုယ်တော် တက်ခြင်း။ ၁.၀၂-၁.၀၅ တည်ငြိမ်သောအားထုတ်မှု၊ ကွဲပြားမှုအနည်းဆုံး
လမ်းပြိုင်ပွဲ ၁.၀၅-၁.၁၀ တိုက်ခိုက်မှုအချို့၊ နေရာချထားရေး ကြိုးပမ်းမှုများ၊
သတ်မှတ်ချက် ၁.၁၅-၁.၂၅ အဆက်မပြတ် အရှိန်မြှင့်မှုများ၊ အလွန်ပြောင်းလဲနိုင်သည်။

🚵 တောင်တက်စက်ဘီးစီးခြင်း။

စီးခြင်းအမျိုးအစား ရိုးရိုး VI လက္ခဏာများ
XC ပြိုင်ပွဲ 1.10-1.20+ ပြိုင်ပွဲတစ်ခုလျှင် 88+ လှိုင်းများ ၊ အလွန်ပြောင်းလဲနိုင်သည်။
လမ်းကြမ်းစီးခြင်း။ ၁.၀၈-၁.၁၅ နည်းပညာပိုင်း၊ တောင်တက်၊ ဆင်းသက်မှုများ
Enduro 1.15-1.30+ တိုတောင်းသော ဖောက်ခွဲရေး အားထုတ်မှုများအကြား ပြန်လည်ထူထောင်ရေး၊

High VI က ဘာကြောင့် အရေးကြီးတာလဲ။

Anaerobic Stress တိုးလာသည်။

High VI သည် သတ်မှတ်ချက်ထက် မကြာခဏ ကြိုးစားအားထုတ်မှုကို ညွှန်ပြပြီး W' (anaerobic capacity) ကို ထပ်ခါတလဲလဲ လျော့နည်းစေသည်။ ၎င်းသည် တူညီသော ပျမ်းမျှစွမ်းအားဖြင့် ပုံမှန်စီးနင်းခြင်းထက် ပို၍ပင်ပန်းသည်။

Glycogen လျော့နည်းခြင်း။

တူညီသောပျမ်းမျှစွမ်းအားဖြင့် တည်ငြိမ်သောအားထုတ်မှုများထက် ကွဲပြားသောအားထုတ်မှုများသည် glycogen ကိုပိုမိုလောင်ကျွမ်းစေသည်။ မြင့်မားသော VI စီးနင်းမှုများသည် မကြာခဏ လောင်စာဆီပိုလိုအပ်သည်။

အာရုံကြောကြွက်သားများ ပင်ပန်းနွမ်းနယ်ခြင်း။

ထပ်ခါတလဲလဲ အရှိန်မြှင့်ခြင်းနှင့် ပြင်းထန်သော အားထုတ်မှုများသည် နှလုံးသွေးကြောဆိုင်ရာ ဝန်ကို ခန့်မှန်းထားသည်ထက် အာရုံကြောဆိုင်ရာ ပင်ပန်းနွမ်းနယ်မှုကို ဖြစ်စေသည်။

စည်းကမ်းများကို ပိုင်းခြားရန် VI ကို အသုံးပြုခြင်း။

VI သည် တောင်တက်စက်ဘီးစီးခြင်းနှင့် လမ်းစက်ဘီးစီးခြင်းကို ကွဲပြားစေသည့် အဓိက မက်ထရစ်ဖြစ်သည်။

🚴 လမ်းတွင် စက်ဘီးစီးသူ

VI: 1.02-1.05

တည်ငြိမ်သောစွမ်းအား၊ လှိုင်းအနည်းငယ်သာရှိသည်။ FTP ကို အာရုံစိုက်ပြီး တံခါးပေါက်တွင် ရေရှည်တည်တံ့သော ပါဝါကို အာရုံစိုက်ပါ။

🚵 Mountain Biker

VI: 1.10-1.20+

ပေါက်ကွဲအား၊ မကြာခဏ လှိုင်းထခြင်း။ ထပ်တလဲလဲဖြစ်နိုင်မှုနှင့် W' စီမံခန့်ခွဲမှုအပေါ်အာရုံစိုက်ပါ။

⚠️ အရှိန်အဟုန် သက်ရောက်မှုများ

အရှိန်အဟုန် အရေးကြီးသော ပြိုင်ပွဲများ (လမ်းပြေးပြိုင်ပွဲများ၊ TTs၊ မာရသွန် MTB) တွင် VI < 1.05 ကို ရည်ရွယ်ပါသည်။ လှိုင်းစီးမှုတိုင်းသည် သင့်အား ပုံမှန်စီးနင်းခြင်းထက် ပိုကုန်ကျပြီး အစတွင် VI မြင့်မားခြင်းသည် သင်အဆုံးတွင် မှိန်သွားလိမ့်မည်ဟု ဆိုလိုသည်။

ခြွင်းချက်-သတ်မှတ်ချက်များနှင့် နည်းပညာပိုင်းဆိုင်ရာ MTB ပြိုင်ပွဲများသည် သဘာဝအားဖြင့် မြင့်မားသော VI လိုအပ်သည်။ ဤတောင်းဆိုမှုအတွက် အထူးလေ့ကျင့်ပါ။

W' Balance (W'bal) - သင်၏ Anaerobic ဘက်ထရီ

W' Balance (W'bal)စီးနင်းနေစဉ်အတွင်း သင့်လက်ကျန် anaerobic စွမ်းရည်ကို အချိန်နှင့်တပြေးညီ ခြေရာခံပါ။ ၎င်းသည် အထက်ဖော်ပြပါ ကြိုးစားအားထုတ်မှုများအတွက် သင့်တွင် စွမ်းအင်မည်မျှရှိသည်ကို ပြသသည့် ဘက်ထရီ ကိရိယာတစ်ခုနှင့် တူသည်။ဝေဖန်ပိုင်းခြားနိုင်သောစွမ်းအား (CP).

W' နှင့် CP နားလည်ခြင်း။

ဝေဖန်ပိုင်းခြားနိုင်သောစွမ်းအား (CP)အေရိုးဗစ်နှင့် anaerobic ဇီဝြဖစ်ပျက်မှုကြား နယ်နိမိတ်ကို ကာလရှည်ကြာအောင် ထိန်းထားနိုင်သည့် အမြင့်ဆုံးစွမ်းအားဖြစ်သည်။ FTP နှင့် ဆင်တူသော်လည်း သိပ္ပံနည်းကျ ပိုမိုကြံ့ခိုင်သည်။

W' (W-prime)CP ထက် ကီလိုဂျိုး (kJ) ဖြင့် တိုင်းတာသော အလုပ်အတွက် သင်၏ အကန့်အသတ် စွမ်းရည်ဖြစ်သည်။ ပုံမှန်တန်ဖိုးများသည် 15-25 kJ မှဖြစ်သည်။

CP နှင့် W' Analogy

ူ = တည်တံ့လာခဲ့ပါတယ်။(လမ်းကြမ်းပေါ်တွင် အကန့်အသတ်မရှိ စီးရသလို)
W' = လှိုင်းတက်ရန်အတွက် ဘက်ထရီ(Boost Button ကဲ့သို့ အချိန်အကြာကြီးအသုံးပြုနိုင်ပါသည်)

CP အထက်စီးရင် W' ကို အားကုန်သွားပါပြီ။ သင် CP အောက်တွင်စီးပါက W' ပြန်လည်ကောင်းမွန်သည်—သို့သော် ပြန်လည်နာလန်ထမှုသည် ကုန်ဆုံးခြင်းထက် နှေးကွေးသည်။

W'bal ဘယ်လိုအလုပ်လုပ်သလဲ။

CP အထက်တွင် ကျဆင်းခြင်း။

သင် CP အထက်စီးသည့်စက္ကန့်တိုင်း၊ သင်သည် CP အထက်မည်မျှအကွာအဝေးနှင့် အချိုးကျသည့်နှုန်းဖြင့် W ကို "သုံးစွဲသည်"

  • CP သည် 280W → 20 J/s တွင် W' ကို ဖျက်သောအခါ 300W အားစိုက်ထုတ်သည်
  • CP သည် 280W → 70 J/s တွင် W' ကို ဖျက်လိုက်သောအခါ 350W အားထုတ်မှု (ပိုမိုမြန်ဆန်သည်!)

CP အောက်တွင်ပြန်လည်ရယူခြင်း။

သင် CP အောက်တွင်စီးပါက W' သည် အချိန် 300-500 စက္ကန့်ခန့် (ပင်ပန်းနွမ်းနယ်မှုအဆင့်ပေါ်မူတည်၍) အဆက်မပြတ်ပြန်လည်ကောင်းမွန်လာသည်။

ရိုးရှင်းသော W' Balance Equations

ကုန်ခမ်းခြင်း-W'bal သည် တစ်စက္ကန့်လျှင် (ပါဝါ - CP) လျော့နည်းသွားသည်။
ပြန်လည်ရယူခြင်း-W'bal သည် W'max သို့ အတိုးနှုန်းပြန်တက်လာသည်။

အစစ်အမှန်မော်ဒယ်များ (Skiba၊ Bartram) သည် ပိုမိုရှုပ်ထွေးပြီး ပြန်လည်နာလန်ထူနှုန်းအပေါ် ပင်ပန်းနွမ်းနယ်မှု သက်ရောက်မှုများအတွက် စာရင်းကိုင်ထားသည်။

လက်တွေ့ကမ္ဘာ ဥပမာ- MTB ပြိုင်ပွဲ

ဇာတ်လမ်း- နည်းပညာတက်ခြင်းနှင့် XC MTB ပြိုင်ပွဲ

Rider ပရိုဖိုင်-

  • CP: 280W
  • W': 18,000 J (18 kJ)

ပြိုင်ပွဲ အပိုင်း-

  1. စတင်လှိုင်း (20 စက္ကန့် @ 400W):
    • W' depletion- (400-280) × 20 = 2,400 J
    • W'bal: 18,000 → 15,600 J (87% ကျန်)
  2. ပြန်လည်ရယူခြင်း နင်းခြင်း (60 စက္ကန့် @ 220W):
    • တစ်စိတ်တစ်ပိုင်း W' ပြန်လည်ရယူခြင်း- ~1,800 J
    • W'bal: 15,600 → 17,400 J (97% ကျန်)
  3. နည်းပညာပိုင်းတက်ခြင်း (5 မိနစ် @ 320W ပျှမ်းမျှ၊ 6 မှ 380W အထိ)
    • အထက်-CP ကြိုးပမ်းအားထုတ်မှုမှ ကြီးမားသော W' အားကုန်ခမ်းခြင်း + မြင့်တက်ခြင်း။
    • W'bal: 17,400 → 4,200 J (၂၃% ကျန်)
  4. ဆင်းသက်ခြင်း ပြန်လည်ရယူခြင်း (3 မိနစ် @ 150W):
    • ပြန်လည်ရယူခြင်း- ~ 6,000 J
    • W'bal- 4,200 → 10,200 J (57% ကျန်)

သုံးသပ်ချက်-မြင်းစီးသူသည် နောက်ထပ် ပြင်းထန်သော အားထုတ်မှုကို ခံနိုင်ရည်ရှိသော်လည်း W'bal 20% အောက်ကို လျော့သွားခြင်းမှ ရှောင်ကြဉ်သင့်သည် သို့မဟုတ် ၎င်းတို့သည် တိုက်ခိုက်မှုများကို တုံ့ပြန်နိုင်စွမ်းမရှိပေ။

Race Strategy အတွက် W'bal ကို အသုံးပြုခြင်း။

Pacing Strategy

  • တောင်တက်ခြင်းအပေါ် W'bal ကို စောင့်ကြည့်ပါထိပ်က တိုက်ခိုက်မှုတွေကို မတုံ့ပြန်နိုင်လောက်အောင် နက်နဲမနေပါနဲ့။
  • ပြန်လည်ထူထောင်ရေးကာလများကို အများဆုံး ချဲ့ရန်-W' အား အားပြန်သွင်းရန် အဆင်းအတက်နှင့် ပြန့်ပြူးသော အပိုင်းများတွင် ပျော့ပျောင်းသောနင်း
  • နောက်ဆုံးကြိုးပမ်းမှုများကို စီစဉ်ပါ-ပြေးခြင်း သို့မဟုတ် နောက်ဆုံးအဆင့်တက်ခြင်းအတွက် သင် W' မည်မျှကျန်ခဲ့သည်ကို သိပါ။

သင်တန်းလျှောက်လွှာများ

  • ကြားကာလ ဒီဇိုင်းW' ကို ထပ်ခါတလဲလဲ ဖျက်/ပြန်ယူရန် ဖွဲ့စည်းပုံကြားကာလများ
  • ပြိုင်ပွဲ သရုပ်သကန်-လူမျိုးအလိုက် အခြေအနေများတွင် W'bal စီမံခန့်ခွဲခြင်းကို လေ့ကျင့်ပါ။
  • အားနည်းချက်ကို ဖော်ထုတ်ခြင်း-Low W' = anaerobic စွမ်းရည် ပိုမိုလုပ်ဆောင်ရန် လိုအပ်သည်။

💡 MTB-Specific W'bal သင်တန်း

ဆက်တိုက်ကြားကာလများ-2 မိနစ် tempo (90% FTP) + 15-second surge (150% FTP)၊ 6-8x ထပ်လုပ်ပါ

၎င်းသည် XC ပြိုင်ကားတောင်းဆိုမှုများကို တုပသည်- ထပ်ခါတလဲလဲ လှိုင်းများနှင့်အတူ စဉ်ဆက်မပြတ် အရှိန်မြှင့်သည်။ CP နှင့် W' recovery rate နှစ်မျိုးလုံး ရထားများ။

ဆက်စပ်-သင့်တွက်နည်းကို လေ့လာပါ။Critical Power နှင့် W'

🔬 သုတေသနဖောင်ဒေးရှင်း

W'bal modeling ကို Critical Power concept (Monod & Scherrer, 1965) မှ အစပြုခဲ့ပြီး W' reconstitution အတွက် Philip Skiba ၏ 2012 model မှ ပြန်လည် သန့်စင်ခဲ့သည်။ မကြာသေးမီက သုတေသနပြုချက်အရ W'bal သည် ပြောင်းလဲနိုင်သော ပြင်းထန်မှုလေ့ကျင့်ခန်းတွင် အချိန်ကုန်သွားခြင်းအတွက် ကြိုတင်ခန့်မှန်းချက်အဖြစ် အတည်ပြုသည်။

အရင်းအမြစ်များ

  • Skiba, P.F., et al. (၂၀၁၂)။W' ၏ အသုံးစရိတ်နှင့် ပြန်လည်ဖွဲ့စည်းခြင်းပုံစံကို ပုံဖော်ခြင်း။အားကစားနှင့် လေ့ကျင့်ခန်းတွင် ဆေးပညာနှင့် သိပ္ပံ။
  • Jones, A.M., et al. (၂၀၁၉)။ဝေဖန်ပိုင်းခြားနိုင်သောစွမ်းအား- သီအိုရီနှင့် အသုံးချမှုများ။အသုံးချဇီဝကမ္မဗေဒဂျာနယ်၊ 126(6)၊ 1905-1915။

ပါဝါမက်ထရစ် နှိုင်းယှဉ်မှု

မက်ထရစ် ၎င်းသည် အဘယ်အရာ တိုင်းတာသနည်း။ ဖော်မြူလာ အကောင်းဆုံးအသုံးပြုမှု Case
ပျမ်းမျှစွမ်းအား ဆိုလိုတာက ပါဝါအထွက် ဝပ်/စက္ကန့် ပေါင်းလဒ် တည်ငြိမ်သော အားထုတ်မှုများ (TTs၊ တစ်ကိုယ်တော် တောင်တက်ခြင်း)
ပုံမှန်ဓာတ်အား (NP) ဇီဝကမ္မ ကုန်ကျစရိတ် (အလေးချိန်) ⁴√(ပျမ်းမျှ [30s ပျမ်းမျှ]⁴) ပြောင်းလဲနိုင်သော ကြိုးပမ်းမှုများ (ဝေဖန်ချက်များ၊ MTB၊ အုပ်စုလိုက်စီးနင်းမှုများ)
ပြင်းထန်မှုအချက် (IF) ဆက်စပ်အားထုတ်မှုနှင့် FTP NP/FTP Session ပြင်းထန်မှု ဆေးညွှန်း
ကွဲပြားမှုအညွှန်းကိန်း (VI) စွမ်းအားတည်ငြိမ်မှု NP/Average Power အရှိန်အဟုန်ကို ခွဲခြမ်းစိတ်ဖြာခြင်း၊ စည်းကမ်း နှိုင်းယှဉ်ခြင်း။
W' Balance (W'bal) Anaerobic စွမ်းရည် ကျန်ရှိနေပါသည်။ ရှုပ်ထွေးသော (Skiba မော်ဒယ်) အချိန်နှင့်တပြေးညီ ပြိုင်ပွဲဗျူဟာ၊ ကြားကာလဒီဇိုင်း

စည်းကမ်းဖြင့် လက်တွေ့နမူနာများ

အမေးများသောမေးခွန်းများ

NP သည် ပျမ်းမျှစွမ်းအားထက် အဘယ်ကြောင့် အမြဲမြင့်မားသနည်း။

NP သည် အချိုးမညီသော ပင်ပန်းနွမ်းနယ်မှုကို ဖန်တီးပေးသောကြောင့် ပြင်းထန်မှုအား ပိုမိုပြင်းထန်စေသည်။ 4th power calculation သည် ပျမ်းမျှအထက် surges များကို ချဲ့ထွင်ပြီး NP ≥ ပျမ်းမျှ power ကို ဖြစ်စေသည်။ တည်ငြိမ်သောအားထုတ်မှုများအတွက်၊ NP သည် ပျမ်းမျှစွမ်းအား (VI = 1.0) နှင့် ညီမျှသည်။

မတူညီသောလေ့ကျင့်ခန်းများအတွက် "ကောင်းသော" ပြင်းထန်မှုအချက်ကဘာလဲ။

ပြန်လည်ရယူခြင်း-IF < 0.65 |ခံနိုင်ရည်-IF 0.65-0.75 |Tempo-IF 0.75-0.85 |အဆင့်သတ်မှတ်ချက်-IF 0.85-0.95 |FTP စမ်းသပ်မှု-IF 0.95-1.05။ ပိုမိုမြင့်မားသော IF = ပိုမိုခက်ခဲသော စက်ရှင်ဖြစ်သော်လည်း ရေရှည်တည်တံ့မှုအပေါ် မူတည်သည်။

စီးနင်းမှုအားလုံးတွင် VI ကို နိမ့်ကျစေရန် ပစ်မှတ်ထားသင့်ပါသလား။

နံပါတ်။ အချိန်စမ်းသပ်မှုများနှင့် တစ်ကိုယ်တော်တက်ခြင်းများသည် အကောင်းဆုံးအမြန်နှုန်းအတွက် အနိမ့်ဆုံး VI (1.00-1.03) မှ အကျိုးကျေးဇူးရရှိမည်ဖြစ်သည်။ သို့သော် သတ်မှတ်ချက်များ၊ MTB ပြိုင်ပွဲများနှင့် အုပ်စုလိုက်စီးနင်းမှုများသည် သဘာဝအားဖြင့် VI (1.10-1.25) မြင့်မားသည်။ သင်၏ပန်းတိုင်ဖြစ်ရပ်များနှင့်ကိုက်ညီသော VI ပရိုဖိုင်ဖြင့်လေ့ကျင့်ပါ။

W'bal သည် FTP နှင့် မည်သို့ကွာခြားသနည်း။

FTP (သို့မဟုတ် CP) သည် တစ်ခုဖြစ်သည်။နှုန်း- watts ဖြင့်ရေရှည်တည်တံ့သောပါဝါ။ W' သည် aစွမ်းရည်— FTP အထက် ကြိုးပမ်းအားထုတ်မှုများအတွက် ရရှိနိုင်သော စုစုပေါင်းစွမ်းအင်ကို ကီလိုဂျိုးများဖြင့် တိုင်းတာသည်။ စဉ်းစားကြည့်ပါ- FTP = သင် အဆက်မပြတ် မောင်းနှင်နိုင်သည် မည်မျှမြန်သည်၊ W' = ထိုအရှိန်ထက် ပေါက်ကွဲရန်အတွက် သင်၏ နိုက်ထရပ်ကန်အရွယ်အစား။

လေ့ကျင့်မှုအားဖြင့် ကျွန်ုပ်၏ W' ကို တိုးတက်နိုင်ပါသလား။

ဟုတ်ကဲ့။ ပြင်းထန်မှုမြင့်မားသောကြားကာလများ (VO2max၊ anaerobic) သည် 8-12 ပတ်အတွင်း W' ကို 10-20% တိုးစေသည်။ Sprint လေ့ကျင့်ခြင်း၊ တောင်တက်ခြင်း နှင့် တိုတောင်းသော ကြားကာလများ (120-150% FTP တွင်) အထူးသဖြင့် W' development ကို ပစ်မှတ်ထားသည်။

ကျွန်ုပ်သည် အရေးကြီးသောပါဝါ လိုအပ်ပါသလား သို့မဟုတ် FTP လုံလောက်ပါသလား။

အခြေခံလေ့ကျင့်မှုအတွက်၊ FTP သည် ကောင်းမွန်စွာအလုပ်လုပ်ပါသည်။ သို့သော် အကယ်၍ သင်သည် MTB၊ စံသတ်မှတ်ချက်များ၊ သို့မဟုတ် ပြောင်းလဲနိုင်သော ပြင်းထန်မှုဖြစ်ရပ်တစ်ခုခုကို ယှဉ်ပြိုင်ပါက၊ CP နှင့် W' ကို ပုံစံထုတ်ခြင်းဖြင့် သင်တိုက်ခိုက်နိုင်သည့်အချိန်တွင် ဗျူဟာနှင့် နားလည်မှုများအတွက် ကြီးမားသောအားသာချက်များကို ပေးဆောင်သည်။ CP သည် FTP ထက် သိပ္ပံနည်းကျ ပိုမိုကြံ့ခိုင်သည်။

MTB သည် လမ်းပေါ်တွင် စက်ဘီးစီးခြင်းထက် အဘယ်ကြောင့် VI ပိုများသနည်း။

MTB မြေပြင်အနေအထားသည် ပြောင်းလဲနိုင်သော ပါဝါကို လိုအပ်သည်- အတားအဆီးများအပေါ် အရှိန်မြှင့်ခြင်း၊ နည်းပညာပိုင်း ကဏ္ဍများတွင် အရှိန်မြှင့်ခြင်း၊ ဆင်းသက်လာမှုတွင် ပြန်လည်ရယူခြင်း။ လမ်းတွင် စက်ဘီးစီးခြင်း (အထူးသဖြင့် တစ်ကိုယ်တော်) တည်ငြိမ်သော စွမ်းအားကို ရရှိစေပါသည်။ဤအခြေခံခြားနားချက်သည် MTB စီးနင်းသူများသည် စဉ်ဆက်မပြတ် ကန့်သတ်စွမ်းအားသာမကဘဲ ထပ်တလဲလဲဖြစ်နိုင်မှုနှင့် W' စီမံခန့်ခွဲမှုတို့ လိုအပ်ပါသည်။

ဆက်စပ်အရင်းအမြစ်များ

  • Functional Threshold Power (FTP) လမ်းညွှန်
  • Critical Power နှင့် W' ကို ရှင်းပြထားသည်။
  • စွမ်းအားပေါ်အခြေခံ၍ လေ့ကျင့်ရေးဇုန်များ
  • Pedaling Mechanics နှင့် စွမ်းဆောင်ရည်
  • Power-Based Training ဖြင့် စတင်လိုက်ပါ။
