ឥតគិតថ្លៃ TSS ម៉ាស៊ីនគិតលេខ - ការបណ្តុះបណ្តាលពិន្ទុស្ត្រេសសម្រាប់ការជិះកង់

គណនា​ពិន្ទុ​នៃ​ការ​ហ្វឹកហាត់​សម្រាប់​ការ​ហាត់ប្រាណ​ដោយ​ជិះ​កង់​របស់​អ្នក​ដោយ​ប្រើ​ថាមពល រយៈពេល និង FTP

តើពិន្ទុស្ត្រេសបណ្តុះបណ្តាល (TSS) ជាអ្វី?

ពិន្ទុស្ត្រេសហ្វឹកហាត់ (TSS)កំណត់បរិមាណនៃបន្ទុកហ្វឹកហាត់នៃការហាត់ប្រាណជិះកង់ដោយរួមបញ្ចូលគ្នានូវអាំងតង់ស៊ីតេ និងរយៈពេលដោយផ្អែកលើទិន្នផលថាមពលរបស់អ្នក។ បង្កើតឡើងដោយបណ្ឌិត Andrew Coggan TSS ប្រើថាមពលកម្រិតមុខងាររបស់អ្នក (FTP) ជាចំណុចយោង។ ការហាត់ប្រាណ 1 ម៉ោងនៅ FTP = 100 TSS ។

ឥតគិតថ្លៃ TSS ម៉ាស៊ីនគិតលេខ

គណនា​ភាពតានតឹង​ក្នុង​ការ​ហ្វឹកហាត់​សម្រាប់​ការ​ហាត់ប្រាណ​ដោយ​ជិះ​កង់​ដោយ​ប្រើ​ទិន្នន័យ​ថាមពល។

ថាមពលកម្រិតមុខងាររបស់អ្នកគិតជាវ៉ាត់
រយៈពេលជិះសរុបគិតជានាទី (1-720)
NP ពីទិន្នន័យជិះរបស់អ្នក (មិនមែនថាមពលមធ្យមទេ)

ពិន្ទុស្ត្រេសបណ្តុះបណ្តាលរបស់អ្នក។

បំពេញដោយ JavaScript

របៀបដែល TSS ត្រូវបានគណនា

TSS រូបមន្ត

TSS = (វិនាទី × NP × IF) / (FTP × 3600) × 100

កន្លែងណា៖

  • NP (ថាមពលធម្មតា)= សរីរវិទ្យា "ថ្លៃ" នៃការជិះគិតជាវ៉ាត់
  • IF (កត្តាអាំងតង់ស៊ីតេ)= NP / FTP (អាំងតង់ស៊ីតេទាក់ទងនឹងកម្រិត)
  • រយៈពេល= ពេលវេលាជិះសរុបគិតជាវិនាទី
  • FTP= ថាមពលកម្រិតមុខងាររបស់អ្នកគិតជាវ៉ាត់

សាមញ្ញ៖ TSS = រយៈពេល (ម៉ោង) × IF² × 100

ឧទាហរណ៍ដែលបានធ្វើការ

ឧទាហរណ៍ទី 1: ការជិះងាយស្រួល

ទិន្នន័យអ្នកជិះ៖

  • FTP៖250W
  • រយៈពេល៖120 នាទី (7200s)
  • ថាមពលធម្មតា៖150W

ជំហានទី 1: គណនាកត្តាអាំងតង់ស៊ីតេ

IF = NP / FTP
IF = 150W / 250W
IF =0.60

ជំហានទី 2៖ គណនា TSS

TSS = (7200 × 150 × 0.60) / (250 × 3600) × 100
TSS = (648,000) / (900,000) × 100
TSS =72 TSS

ការបកស្រាយ៖ការជិះងាយស្រួលក្នុងកម្រិតអាំងតង់ស៊ីតេ 60% - ល្អឥតខ្ចោះសម្រាប់ការកសាងមូលដ្ឋាន និងការស្តារឡើងវិញ។

ឧទាហរណ៍ទី 2៖ ចន្លោះពេលកម្រិត

ទិន្នន័យអ្នកជិះ៖

  • FTP៖250W
  • រយៈពេល៖90 នាទី (5400s)
  • ថាមពលធម្មតា៖235 វ៉

ជំហានទី 1: គណនាកត្តាអាំងតង់ស៊ីតេ

IF = NP / FTP
IF = 235W / 250W
IF =០.៩៤

ជំហានទី 2៖ គណនា TSS

TSS = (5400 × 235 × 0.94) / (250 × 3600) × 100
TSS = (1,192,860) / (900,000) × 100
TSS =133 TSS

ការបកស្រាយ៖វគ្គកម្រិតរឹងនៅអាំងតង់ស៊ីតេ 94% - ការជំរុញការបណ្តុះបណ្តាលដ៏សំខាន់សម្រាប់ការកែលម្អ FTP ។

ឧទាហរណ៍ទី 3: ជិះក្រុមរឹង

ទិន្នន័យអ្នកជិះ៖

  • FTP៖250W
  • រយៈពេល៖180 នាទី (10800s)
  • ថាមពលធម្មតា៖210W

ជំហានទី 1: គណនាកត្តាអាំងតង់ស៊ីតេ

IF = NP / FTP
IF = 210W / 250W
IF =០.៨៤

ជំហានទី 2៖ គណនា TSS

TSS = (10800 × 210 × 0.84) / (250 × 3600) × 100
TSS = (1,905,120) / (900,000) × 100
TSS =212 TSS

ការបកស្រាយ៖ការជិះដ៏លំបាកក្នុងកម្រិត 84% - ការផ្ទុកការហ្វឹកហាត់ខ្ពស់ដែលតម្រូវឱ្យមានការងើបឡើងវិញ 1-2 ថ្ងៃ។

ឧទាហរណ៍ទី 4៖ VO₂max ចន្លោះពេល

ទិន្នន័យអ្នកជិះ៖

  • FTP៖250W
  • រយៈពេល៖75 នាទី (4500s)
  • ថាមពលធម្មតា៖270 វ៉

ជំហានទី 1: គណនាកត្តាអាំងតង់ស៊ីតេ

IF = NP / FTP
IF = 270W / 250W
IF =1.08

ជំហានទី 2៖ គណនា TSS

TSS = (4500 × 270 × 1.08) / (250 × 3600) × 100
TSS = (1,312,200) / (900,000) × 100
TSS =146 TSS

ការបកស្រាយ៖ពិបាកខ្លាំងណាស់ VO₂max វគ្គលើសពីកម្រិត - ការជំរុញការបណ្តុះបណ្តាលខ្ពស់ទោះបីជារយៈពេលខ្លីជាងក៏ដោយ។

TSS ការណែនាំតាមប្រភេទលំហាត់ប្រាណ

ប្រភេទលំហាត់ប្រាណ TSS ជួរ កត្តាអាំងតង់ស៊ីតេ ការពិពណ៌នា
ជិះស្តារឡើងវិញ 20-50 TSS IF < 0.65 ការបង្វិលងាយស្រួល 30-60 នាទី។
ការស៊ូទ្រាំងាយស្រួល 50-100 TSS IF 0.65-0.75 ល្បឿនសន្ទនា 1-2 ម៉ោង។
ការស៊ូទ្រាំកម្រិតមធ្យម 100-150 TSS IF 0.75-0.85 ជិះថេរ 2-3 ម៉ោង។
ជិះ Tempo 150-200 TSS IF 0.85-0.95 កន្លែងផ្អែម, ការងារបណ្តោះអាសន្ន, 2-3 ម៉ោង។
ការហាត់ប្រាណកម្រិត 200-300 TSS IF 0.95-1.05 FTP ចន្លោះពេល ស៊ីមប្រណាំង 2-4 ម៉ោង។
VO₂max ចន្លោះពេល 150-250 TSS IF 1.05-1.15 ចន្លោះពេលរឹង 1-2 ម៉ោងអាំងតង់ស៊ីតេខ្ពស់។
ការប្រណាំង / ព្រឹត្តិការណ៍លំបាក 200-400 TSS IF 0.90-1.05 លក្ខណៈវិនិច្ឆ័យ ការប្រណាំងផ្លូវ 2-5 ម៉ោង។

ប្រចាំសប្តាហ៍ TSS កំណត់គោលដៅតាមកម្រិតអ្នកជិះកង់

អ្នកជិះកង់ចាប់ផ្តើម

ប្រចាំសប្តាហ៍ TSS៖200-400

ជិះ 3-4 ដងក្នុងមួយសប្តាហ៍ 50-100 TSS នីមួយៗ។ ផ្តោតលើការកសាងមូលដ្ឋាន aerobic និងជំនាញគ្រប់គ្រងកង់។

អ្នកជិះកង់កំសាន្ត

ជិះ 4-5 ដងក្នុងមួយសប្តាហ៍ 80-120 TSS នីមួយៗ។ លាយនៃការស៊ូទ្រាំ និងវគ្គគុណភាពមួយចំនួន។

អ្នកស្ម័គ្រចិត្តប្រកួតប្រជែង

ប្រចាំសប្តាហ៍ TSS៖៦០០-៩០០

ជិះ 5-7 ដងក្នុងមួយសប្តាហ៍ 85-130 TSS នីមួយៗ។ ការហ្វឹកហ្វឺនដែលមានរចនាសម្ព័ន្ធជាមួយនឹងការកំណត់ពេលនិងការប្រណាំង។

10-15+ sessions per week. High volume professional training load. Post-Grand Tour CTL: 150-170.

⚠️ ចំណាំសំខាន់ៗអំពី TSS

  • ទាមទារភាពត្រឹមត្រូវ FTP៖FTP របស់អ្នកត្រូវតែជាបច្ចុប្បន្ន (សាកល្បងក្នុងរយៈពេល 4-6 សប្តាហ៍) សម្រាប់ភាពត្រឹមត្រូវ TSS ។
  • ប្រើថាមពលធម្មតា មិនមែនថាមពលមធ្យមទេ៖NP គណនីសម្រាប់ការប្រែប្រួល និងតម្លៃសរីរវិទ្យា។
  • Indoor vs Outdoor:ក្នុងផ្ទះ TSS អាចមានអារម្មណ៍ពិបាកជាងមុន ដោយសារមិនមានឆ្នេរសមុទ្រ ឬចុះក្រោម។
  • បំរែបំរួលបុគ្គល៖ដូចគ្នា TSS មានអារម្មណ៍ខុសគ្នាសម្រាប់អ្នកជិះផ្សេងគ្នា។ កែតម្រូវដោយផ្អែកលើការងើបឡើងវិញរបស់អ្នក។
  • អត្រានៃការឡើងភ្នំមានសារៈសំខាន់៖បង្កើន TSS ប្រចាំសប្តាហ៍បន្តិចម្តងៗ—3-8 CTL ពិន្ទុក្នុងមួយសប្តាហ៍មាននិរន្តរភាព។

ហេតុអ្វី TSS បញ្ហា៖ CTL, ATL, TSB

ពិន្ទុស្ត្រេសបណ្តុះបណ្តាលគឺជាមូលដ្ឋានគ្រឹះសម្រាប់តារាងគ្រប់គ្រងការអនុវត្ត៖

  • CTL (ផ្ទុកការបណ្តុះបណ្តាលរ៉ាំរ៉ៃ)៖កម្រិតសម្បទារបស់អ្នក - ទម្ងន់អិចស្ប៉ូណង់ស្យែលរយៈពេល 42 ថ្ងៃនៃប្រចាំថ្ងៃ TSS
  • ATL (ផ្ទុកការបណ្តុះបណ្តាលស្រួចស្រាវ)៖ភាពអស់កម្លាំងរបស់អ្នក - ទម្ងន់អិចស្ប៉ូណង់ស្យែលរយៈពេល 7 ថ្ងៃនៃប្រចាំថ្ងៃ TSS
  • TSB (សមតុល្យស្ត្រេសនៃការបណ្តុះបណ្តាល)៖ទម្រង់របស់អ្នក - TSB = CTL - ATL (វិជ្ជមាន = ស្រស់ អវិជ្ជមាន = អស់កម្លាំង)
  • ការមករដូវ៖រៀបចំផែនការដំណាក់កាលបណ្តុះបណ្តាល (មូលដ្ឋាន ស្ថាបនា កំពូល ខ្លី) ដោយប្រើគោលដៅ CTL វឌ្ឍនភាព
  • ពេលវេលាប្រណាំង៖តិចបំផុតដើម្បីសម្រេចបាន TSB ពី +10 ដល់ +25 នៅថ្ងៃប្រណាំងសម្រាប់ដំណើរការខ្ពស់បំផុត

សំណួរដែលសួរញឹកញាប់

ចុះបើខ្ញុំមិនមានម៉ាសុីនភ្លើង?

TSS ទាមទារទិន្នន័យថាមពល (FTP និងថាមពលធម្មតា)។ បើគ្មានឧបករណ៍វាស់ថាមពលទេ អ្នកអាចប្រើជម្រើសដែលផ្អែកលើអត្រាបេះដូងដូចជា HRSS (ពិន្ទុស្ត្រេសអត្រាបេះដូង) ឬប្រើការប៉ាន់ប្រមាណដែលយល់ឃើញ ប៉ុន្តែទាំងនេះមានភាពត្រឹមត្រូវតិចជាង។ ម៉ែត្រថាមពលគឺជាស្តង់ដារមាសសម្រាប់ TSS ការគណនា និងការគ្រប់គ្រងបន្ទុកហ្វឹកហាត់ក្នុងការជិះកង់។

តើ TSS ត្រឹមត្រូវប៉ុណ្ណា?

TSS មានភាពត្រឹមត្រូវខ្ពស់នៅពេលផ្អែកលើ FTP បច្ចុប្បន្ន និងបានគណនាថាមពលធម្មតា។ ការសិក្សាបង្ហាញថា TSS អាចទស្សន៍ទាយបានថា បន្ទុកបណ្តុះបណ្តាល និងតម្រូវការសង្គ្រោះ។ ភាពត្រឹមត្រូវអាស្រ័យលើ៖ ការធ្វើតេស្ត FTP ថ្មីៗនេះ (ក្នុងរយៈពេល 4-6 សប្តាហ៍) ការគណនា NP ត្រឹមត្រូវ និងភាពត្រឹមត្រូវនៃម៉ែត្រថាមពល (ភាគច្រើនគឺ ±1-2%)។

តើខ្ញុំអាចប្រៀបធៀប TSS រវាងផ្លូវ និង MTB បានទេ?

បាទ ប៉ុន្តែជាមួយនឹងការព្រមាន។ ការជិះកង់តាមដងផ្លូវ TSS អាចព្យាករណ៍បានច្រើនជាងមុនជាមួយនឹងទម្រង់ថាមពលថេរ (VI 1.02-1.05) ។ MTB TSS គឺ​មាន​អថេរ​ខ្ពស់ (VI 1.10-1.20+) ដោយ​មាន​ការ​ផ្ទុះ​ញឹកញាប់​លើស​កម្រិត។ ដូចគ្នា TSS ពី MTB អាចមានអារម្មណ៍ពិបាកជាងមុន ដោយសារតម្រូវការបច្ចេកទេស។ ផ្តោតលើការវិភាគនិន្នាការជាជាងការប្រៀបធៀបដាច់ខាតរវាងវិញ្ញាសា។

តើអ្វីជា TSS ល្អសម្រាប់ការហាត់ប្រាណមួយ?

វាអាស្រ័យលើគោលដៅនៃការហ្វឹកហាត់៖ ការជិះឡើងវិញ៖ 20-50 TSS ការស៊ូទ្រាំងាយស្រួល៖ 50-100 TSS ការជិះក្នុងកម្រិតមធ្យម៖ 100-150 TSS ការហាត់ប្រាណខ្លាំង៖ 150-250 TSS ពិបាកណាស់ 02 វគ្គ TSS។ ការជិះហ្វឹកហាត់ភាគច្រើនធ្លាក់ចន្លោះពី 50-150 TSS។ ចន្លោះពេលគុណភាពបង្កើត 100-200 TSS ។ ការប្រណាំង និងព្រឹត្តិការណ៍រយៈពេលវែង៖ 200-400+ TSS ។

តើខ្ញុំគួរធ្វើ TSS ប៉ុន្មានក្នុងមួយសប្តាហ៍?

គោលដៅ TSS ប្រចាំសប្តាហ៍តាមកម្រិត៖ អ្នកចាប់ផ្តើមដំបូង៖ 200-400 TSS/សប្តាហ៍ ការកំសាន្ត៖ 400-600 TSS/សប្តាហ៍ អ្នកស្ម័គ្រចិត្តប្រកួតប្រជែង៖ 600-900 __TECH_1_5/000+- Elite TSS/សប្តាហ៍។ ចាប់ផ្តើមដោយអភិរក្ស និងបង្កើន 3-8 CTL ពិន្ទុក្នុងមួយសប្តាហ៍។ TSS ប្រចាំសប្តាហ៍ប្រកបដោយនិរន្តរភាពរបស់អ្នក អាស្រ័យលើប្រវត្តិនៃការបណ្តុះបណ្តាល ពេលវេលាដែលមាន និងលទ្ធភាពនៃការស្តារឡើងវិញ។

តើខ្ញុំគួរប្រើ TSS សម្រាប់ការហ្វឹកហាត់ក្នុងផ្ទះដែរឬទេ?

ដាច់ខាត។ TSS គឺល្អសម្រាប់ការហ្វឹកហ្វឺនក្នុងផ្ទះ ពីព្រោះថាមពលគឺស្រប និងមិនមានអថេរបរិស្ថាន។ ក្នុងផ្ទះ TSS រួមចំណែកដោយផ្ទាល់ដល់ CTL/ATL/TSB របស់អ្នក។ ទោះជាយ៉ាងណាក៏ដោយ ការជិះក្នុងផ្ទះអាចមានអារម្មណ៍ពិបាកជាងការជិះនៅខាងក្រៅសម្រាប់ TSS ដូចគ្នា ដោយសារមិនមានឆ្នេរសមុទ្រ ការធ្លាក់ ឬផ្នែកដែលជំនួយដោយខ្យល់។

តើអ្វីជាភាពខុសគ្នារវាង TSS និងគីឡូជូល?

Kilojoules វាស់ការងារសរុបដែលបានធ្វើ (ការចំណាយថាមពល)—ដូចគ្នាសម្រាប់មនុស្សគ្រប់គ្នាដែលមានថាមពលដូចគ្នា។ TSS វាស់ភាពតានតឹងការបណ្តុះបណ្តាលទាក់ទងនឹង FTP របស់អ្នក។ ឧទាហរណ៍៖ 200W សម្រាប់ 1 ម៉ោង = 720 kJ សម្រាប់អ្នករាល់គ្នា។ ប៉ុន្តែប្រសិនបើ FTP របស់អ្នកគឺ 200W, TSS=100. ប្រសិនបើ FTP របស់អ្នកគឺ 300W, TSS= 44.
