бһ–бһ·бһ“бҹ’бһ‘бһ»бһҹбҹ’бһҸбҹ’бһҡбҹҒбһҹбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢ (TSS)бһҖбҹҶбһҺбһҸбҹӢбһ”бһҡбһ·бһҳбһ¶бһҺбһ“бҹғбһ”бһ“бҹ’бһ‘бһ»бһҖбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбһ“бҹғбһҖбһ¶бһҡбһ бһ¶бһҸбҹӢбһ”бҹ’бһҡбһ¶бһҺбһҮбһ·бҹҮбһҖбһ„бҹӢбһҠбҹ„бһҷбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһӮбҹ’бһ“бһ¶бһ“бһјбһңбһўбһ¶бҹҶбһ„бһҸбһ„бҹӢбһҹбҹҠбһёбһҸбҹҒ бһ“бһ·бһ„бһҡбһҷбҹҲбһ–бҹҒбһӣбһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ‘бһ·бһ“бҹ’бһ“бһ•бһӣбһҗбһ¶бһҳбһ–бһӣбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһҠбҹ„бһҷбһ”бһҺбҹ’бһҢбһ·бһҸ Andrew Coggan TSS бһ”бҹ’бһҡбһҫбһҗбһ¶бһҳбһ–бһӣбһҖбһҳбҹ’бһҡбһ·бһҸбһҳбһ»бһҒбһ„бһ¶бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ (FTP) бһҮбһ¶бһ…бҹҶбһҺбһ»бһ…бһҷбҹ„бһ„бҹ” бһҖбһ¶бһҡбһ бһ¶бһҸбҹӢбһ”бҹ’бһҡбһ¶бһҺ 1 бһҳбҹүбҹ„бһ„бһ“бҹ… FTP = 100 TSS бҹ”

бһӮбһҺбһ“бһ¶вҖӢбһ—бһ¶бһ–бһҸбһ¶бһ“бһҸбһ№бһ„вҖӢбһҖбҹ’бһ“бһ»бһ„вҖӢбһҖбһ¶бһҡвҖӢбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢвҖӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢвҖӢбһҖбһ¶бһҡвҖӢбһ бһ¶бһҸбҹӢбһ”бҹ’бһҡбһ¶бһҺвҖӢбһҠбҹ„бһҷвҖӢбһҮбһ·бҹҮвҖӢбһҖбһ„бҹӢвҖӢбһҠбҹ„бһҷвҖӢбһ”бҹ’бһҡбһҫвҖӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷвҖӢбһҗбһ¶бһҳбһ–бһӣбҹ”

TSS = (бһңбһ·бһ“бһ¶бһ‘бһё Г— NP Г— IF) / (FTP Г— 3600) Г— 100

бһҖбһ“бҹ’бһӣбҹӮбһ„бһҺбһ¶бҹ–

  • NP (бһҗбһ¶бһҳбһ–бһӣбһ’бһҳбҹ’бһҳбһҸбһ¶)= бһҹбһҡбһёбһҡбһңбһ·бһ‘бҹ’бһҷбһ¶ "бһҗбҹ’бһӣбҹғ" бһ“бҹғбһҖбһ¶бһҡбһҮбһ·бҹҮбһӮбһ·бһҸбһҮбһ¶бһңбҹүбһ¶бһҸбҹӢ
  • IF (бһҖбһҸбҹ’бһҸбһ¶бһўбһ¶бҹҶбһ„бһҸбһ„бҹӢбһҹбҹҠбһёбһҸбҹҒ)= NP / FTP (бһўбһ¶бҹҶбһ„бһҸбһ„бҹӢбһҹбҹҠбһёбһҸбҹҒбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһ“бһ№бһ„бһҖбһҳбҹ’бһҡбһ·бһҸ)
  • бһҡбһҷбҹҲбһ–бҹҒбһӣ= бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҮбһ·бҹҮбһҹбһҡбһ»бһ”бһӮбһ·бһҸбһҮбһ¶бһңбһ·бһ“бһ¶бһ‘бһё
  • FTP= бһҗбһ¶бһҳбһ–бһӣбһҖбһҳбҹ’бһҡбһ·бһҸбһҳбһ»бһҒбһ„бһ¶бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһӮбһ·бһҸбһҮбһ¶бһңбҹүбһ¶бһҸбҹӢ

бһҹбһ¶бһҳбһүбҹ’бһүбҹ– TSS = бһҡбһҷбҹҲбһ–бҹҒбһӣ (бһҳбҹүбҹ„бһ„) Г— IFВІ Г— 100

  • бһўбһҸбҹ’бһҡбһ¶бһ“бҹғбһҖбһ¶бһҡбһЎбһҫбһ„бһ—бҹ’бһ“бҹҶбһҳбһ¶бһ“бһҹбһ¶бһҡбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ–бһ”бһ„бҹ’бһҖбһҫбһ“ TSS бһ”бҹ’бһҡбһ…бһ¶бҹҶбһҹбһ”бҹ’бһҸбһ¶бһ бҹҚбһ”бһ“бҹ’бһҸбһ·бһ…бһҳбҹ’бһҸбһ„бҹ—вҖ”3-8 CTL бһ–бһ·бһ“бҹ’бһ‘бһ»бһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһҹбһ”бҹ’бһҸбһ¶бһ бҹҚбһҳбһ¶бһ“бһ“бһ·бһҡбһ“бҹ’бһҸбһҡбһ—бһ¶бһ–бҹ”

бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһўбҹҶбһ–бһёбһҗбһ¶бһҳбһ–бһӣбһ’бһҳбҹ’бһҳбһҸбһ¶ (NP)

бһҗбһ¶бһҳбһ–бһӣбһ’бһҳбҹ’бһҳбһҸбһ¶бһӮбһәбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҮбһ¶бһ„бһҗбһ¶бһҳбһ–бһӣбһҳбһ’бҹ’бһҷбһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһӮбһҺбһ“бһ¶ TSS бһ–бҹ’бһҡбҹ„бҹҮбһңбһ¶бһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһ…бһјбһӣбһҸбһҳбҹ’бһӣбҹғбһҒбһ¶бһ„бһҹбһҡбһёбһҡбһңбһ·бһ‘бҹ’бһҷбһ¶бһ“бҹғбһ—бһ¶бһ–бһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣбһҗбһ¶бһҳбһ–бһӣбҹ–

бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһ”бһ¶бһ“бһҮбһ¶ NP бһ”бһүбҹ’бһ бһ¶: бһҖбһ¶бһҡбһҮбһ·бҹҮбһ–бһёбһҡбһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ”

бһҮбһ·бҹҮ B: бһҮбһ·бҹҮбһҖбҹ’бһҡбһ»бһҳбһҡбһ№бһ„
  • Duration: 60 minutes
  • бһҗбһ¶бһҳбһ–бһӣбһҮбһ¶бһҳбһ’бҹ’бһҷбһҳ: 200W
  • бһҗбһ¶бһҳбһ–бһӣбһ’бһҳбҹ’бһҳбһҸбһ¶: 240W
  • бһҹбһ“бҹ’бһ‘бһҹбҹ’бһҹбһ“бҹҚбһ—бһ¶бһ–бһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣбҹ– 1.20
  • TSSбҹ– бҹ©бҹў

бһўбһҗбҹҒбһҡбһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„бһҖбһ¶бһҡбһҖбһҫбһ“бһЎбһҫбһ„ - бһҸбһҳбҹ’бһӣбҹғбһҒбһ¶бһ„бһҹбһҡбһёбһҡбһңбһ·бһ‘бҹ’бһҷбһ¶бһҒбҹ’бһ–бһҹбҹӢбһҮбһ¶бһ„бһ…бҹ’бһҡбһҫбһ“бҹ”

бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бҹ–бһҖбһ¶бһҡбһҮбһ·бҹҮбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһҳбһ¶бһ“бһҳбһ’бҹ’бһҷбһҳ 200W бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҮбһ·бҹҮ B бһӮбһәбһ–бһ·бһ”бһ¶бһҖбһҮбһ¶бһ„ 42% (92 бһ‘бһӣбҹӢбһ“бһ№бһ„ 65 TSS) бһҠбҹ„бһҷбһҹбһ¶бһҡбһ—бһ¶бһ–бһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣбһ“бҹғбһҗбһ¶бһҳбһ–бһӣбҹ” NP бһ…бһ¶бһ”бҹӢбһҷбһҖбһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһ“бҹҒбҹҮбҹ”

рҹ’Ўбһҡбһ”бҹҖбһ”бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҗбһ¶бһҳбһ–бһӣбһ’бһҳбҹ’бһҳбһҸбһ¶бҹ”

бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡ бһ“бһ·бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҮбһ·бҹҮбһҖбһ„бҹӢбһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһӮбһҺбһ“бһ¶ NP бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бҹ–

  • Garmin/Wahoo/Hammerheadбҹ–бһ”бһ„бҹ’бһ бһ¶бһү NP бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбһ„бҹ’бһҒбҹҒбһ”бһҖбһ¶бһҡбһҮбһ·бҹҮ
  • TrainingPeaks/Strava/Intervals.icuбҹ–бһӮбһҺбһ“бһ¶ NP бһ–бһёбһҖбһ¶бһҡбһҮбһ·бҹҮбһҠбҹӮбһӣбһ”бһ¶бһ“бһ•бҹ’бһ‘бһ»бһҖбһЎбһҫбһ„
  • Golden Cheetah/WKO5бҹ–бһ§бһ”бһҖбһҡбһҺбҹҚбһңбһ·бһ—бһ¶бһӮ NP бһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢ

бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһҗбһ¶бһҳбһ–бһӣбһҮбһ¶бһҳбһ’бҹ’бһҷбһҳбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮ бһҖбһ¶бһҡбһ”бҹүбһ¶бһ“бҹӢбһҹбҹ’бһҳбһ¶бһ“бҹ– NP вүҲ бһҗбһ¶бһҳбһ–бһӣбһҮбһ¶бһҳбһ’бҹ’бһҷбһҳ Г— 1.03-1.05 бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҮбһ·бҹҮбһҗбҹҒбһҡ Г— 1.10-1.15 бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҮбһ·бҹҮбһўбһҗбҹҒбһҡбҹ”

бһ бҹҒбһҸбһ»бһўбҹ’бһңбһё TSS бһ”бһүбҹ’бһ бһ¶бҹ– CTL, ATL, TSB

бһ–бһ·бһ“бҹ’бһ‘бһ»бһҹбҹ’бһҸбҹ’бһҡбҹҒбһҹбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһӮбһәбһҮбһ¶бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһӮбҹ’бһҡбһ№бҹҮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҸбһ¶бһҡбһ¶бһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбҹ–

  • CTL (бһ•бҹ’бһ‘бһ»бһҖбһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҡбҹүбһ¶бҹҶбһҡбҹүбҹғ)бҹ–бһҖбһҳбҹ’бһҡбһ·бһҸбһҹбһҳбҹ’бһ”бһ‘бһ¶бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ - бһ‘бһҳбҹ’бһ„бһ“бҹӢбһўбһ·бһ…бһҹбҹ’бһ”бҹүбһјбһҺбһ„бҹӢбһҹбҹ’бһҷбҹӮбһӣбһҡбһҷбҹҲбһ–бҹҒбһӣ 42 бһҗбҹ’бһ„бҹғбһ“бҹғбһ”бҹ’бһҡбһ…бһ¶бҹҶбһҗбҹ’бһ„бҹғ TSS
  • ATL (бһ•бҹ’бһ‘бһ»бһҖбһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҹбҹ’бһҡбһҪбһ…бһҹбҹ’бһҡбһ¶бһң)бҹ–бһ—бһ¶бһ–бһўбһҹбҹӢбһҖбһҳбҹ’бһӣбһ¶бҹҶбһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ - бһ‘бһҳбҹ’бһ„бһ“бҹӢбһўбһ·бһ…бһҹбҹ’бһ”бҹүбһјбһҺбһ„бҹӢбһҹбҹ’бһҷбҹӮбһӣбһҡбһҷбҹҲбһ–бҹҒбһӣ 7 бһҗбҹ’бһ„бҹғбһ“бҹғбһ”бҹ’бһҡбһ…бһ¶бҹҶбһҗбҹ’бһ„бҹғ TSS
  • TSB (бһҹбһҳбһҸбһ»бһӣбҹ’бһҷбһҹбҹ’бһҸбҹ’бһҡбҹҒбһҹбһ“бҹғбһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣ)бҹ–бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ - TSB = CTL - ATL (бһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“ = бһҹбҹ’бһҡбһҹбҹӢ бһўбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“ = бһўбһҹбҹӢбһҖбһҳбҹ’бһӣбһ¶бҹҶбһ„)
  • бһҖбһ¶бһҡбһҳбһҖбһҡбһҠбһјбһңбҹ–бһҡбҹҖбһ”бһ…бҹҶбһ•бҹӮбһ“бһҖбһ¶бһҡбһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣ (бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ бһҹбҹ’бһҗбһ¶бһ”бһ“бһ¶ бһҖбҹҶбһ–бһјбһӣ бһҒбҹ’бһӣбһё) бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһӮбҹ„бһӣбһҠбҹ… CTL бһңбһҢбҹ’бһҚбһ“бһ—бһ¶бһ–
  • бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ”бҹ’бһҡбһҺбһ¶бҹҶбһ„бҹ–бһҸбһ·бһ…бһ”бҹҶбһ•бһ»бһҸбһҠбһҫбһҳбҹ’бһ”бһёбһҹбһҳбҹ’бһҡбҹҒбһ…бһ”бһ¶бһ“ TSB бһ–бһё +10 бһҠбһӣбҹӢ +25 бһ“бҹ…бһҗбҹ’бһ„бҹғбһ”бҹ’бһҡбһҺбһ¶бҹҶбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҒбҹ’бһ–бһҹбҹӢбһ”бҹҶбһ•бһ»бһҸ

бһӮбһ“бҹ’бһӣбһ№бҹҮбһӮбһ¶бҹҶбһ‘бҹ’бһҡбҹ– бһҸбһ¶бһҳбһҠбһ¶бһ“бһҸбһ¶бһҡбһ¶бһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

бһ…бһјбһӣбһ”бҹ’бһҡбһ…бһ¶бҹҶбһҗбҹ’бһ„бҹғ TSS бһҖбҹ’бһ“бһ»бһ„ TrainingPeaks, Intervals.icu бһ¬бһҹбҹҖбһңбһ—бҹ…бһ”бһүбҹ’бһҮбһёбҹ” бһҸбһ¶бһҳбһҠбһ¶бһ“ CTL бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ (бһҮбһ¶бһҳбһ’бҹ’бһҷбһҳ 42 бһҗбҹ’бһ„бҹғ) бһ“бһ·бһ„ ATL (бһҳбһ’бҹ’бһҷбһҳ 7 бһҗбҹ’бһ„бҹғ)бҹ” бһӮбҹ„бһӣбһҠбҹ…бһҹбҹ’бһҗбһ·бһҡбһ—бһ¶бһ– CTL бһҖбҹҶбһҺбһҫбһ“ 3-8 бһ–бһ·бһ“бҹ’бһ‘бһ»бһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһҹбһ”бҹ’бһҸбһ¶бһ бҹҚбһҖбҹҶбһЎбһ»бһ„бһ–бҹҒбһӣбһҹбһ¶бһ„бһҹбһ„бҹӢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бҹ” бһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷ TSS 7-14 бһҗбҹ’бһ„бҹғбһҳбһ»бһ“бһҖбһ¶бһҡбһ”бҹ’бһҡбһҺбһ¶бҹҶбһ„ бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷ TSB бһҖбһҫбһ“бһЎбһҫбһ„бһҠбһӣбҹӢбһҸбһҳбҹ’бһӣбҹғбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҒбҹ’бһ–бһҹбҹӢбһ”бҹҶбһ•бһ»бһҸбҹ”

бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳбһўбҹҶбһ–бһё CTL, ATL бһ“бһ·бһ„ TSB вҶ’

бһҹбҹҶбһҺбһҪбһҡбһҠбҹӮбһӣбһҹбһҪбһҡбһүбһ№бһҖбһүбһ¶бһ”бҹӢ

бһ…бһ»бҹҮбһ”бһҫбһҒбҹ’бһүбһ»бҹҶбһҳбһ·бһ“бһҳбһ¶бһ“бһҳбҹүбһ¶бһҹбһ»бһёбһ“бһ—бҹ’бһӣбһҫбһ„?

TSS бһ‘бһ¶бһҳбһ‘бһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҗбһ¶бһҳбһ–бһӣ (FTP бһ“бһ·бһ„бһҗбһ¶бһҳбһ–бһӣбһ’бһҳбҹ’бһҳбһҸбһ¶)бҹ” бһ”бһҫбһӮбҹ’бһҳбһ¶бһ“бһ§бһ”бһҖбһҡбһҺбҹҚбһңбһ¶бһҹбҹӢбһҗбһ¶бһҳбһ–бһӣбһ‘бҹҒ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҮбһҳбҹ’бһҡбһҫбһҹбһҠбҹӮбһӣбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһўбһҸбҹ’бһҡбһ¶бһ”бҹҒбҹҮбһҠбһјбһ„бһҠбһјбһ…бһҮбһ¶ HRSS (бһ–бһ·бһ“бҹ’бһ‘бһ»бһҹбҹ’бһҸбҹ’бһҡбҹҒбһҹбһўбһҸбҹ’бһҡбһ¶бһ”бҹҒбҹҮбһҠбһјбһ„) бһ¬бһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһ”бҹүбһ¶бһ“бҹӢбһ”бҹ’бһҡбһҳбһ¶бһҺбһҠбҹӮбһӣбһҷбһӣбҹӢбһғбһҫбһү бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҳбһ¶бһ“бһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҸбһ·бһ…бһҮбһ¶бһ„бҹ” бһҳбҹүбҹӮбһҸбҹ’бһҡбһҗбһ¶бһҳбһ–бһӣбһӮбһәбһҮбһ¶бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҳбһ¶бһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ TSS бһҖбһ¶бһҡбһӮбһҺбһ“бһ¶ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһ“бҹ’бһ‘бһ»бһҖбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҮбһ·бҹҮбһҖбһ„бҹӢбҹ”

бһҸбһҫ TSS бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһ”бҹүбһ»бһҺбҹ’бһҺбһ¶?

TSS бһҳбһ¶бһ“бһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҒбҹ’бһ–бһҹбҹӢбһ“бҹ…бһ–бҹҒбһӣбһ•бҹ’бһўбҹӮбһҖбһӣбһҫ FTP бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“ бһ“бһ·бһ„бһ”бһ¶бһ“бһӮбһҺбһ“бһ¶бһҗбһ¶бһҳбһ–бһӣбһ’бһҳбҹ’бһҳбһҸбһ¶бҹ” бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһ”бһ„бҹ’бһ бһ¶бһүбһҗбһ¶ TSS бһўбһ¶бһ…бһ‘бһҹбҹ’бһҹбһ“бҹҚбһ‘бһ¶бһҷбһ”бһ¶бһ“бһҗбһ¶ бһ”бһ“бҹ’бһ‘бһ»бһҖбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣ бһ“бһ·бһ„бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҹбһ„бҹ’бһӮбҹ’бһҡбҹ„бҹҮбҹ” бһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһӣбһҫбҹ– бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸ FTP бһҗбҹ’бһҳбһёбҹ—бһ“бҹҒбҹҮ (бһҖбҹ’бһ“бһ»бһ„бһҡбһҷбҹҲбһ–бҹҒбһӣ 4-6 бһҹбһ”бҹ’бһҸбһ¶бһ бҹҚ) бһҖбһ¶бһҡбһӮбһҺбһ“бһ¶ NP бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһ“бһ·бһ„бһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһ“бҹғбһҳбҹүбҹӮбһҸбҹ’бһҡбһҗбһ¶бһҳбһ–бһӣ (бһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһӮбһә Вұ1-2%)бҹ”

бһҸбһҫбһҒбҹ’бһүбһ»бҹҶбһўбһ¶бһ…бһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ” TSS бһҡбһңбһ¶бһ„бһ•бҹ’бһӣбһјбһң бһ“бһ·бһ„ MTB бһ”бһ¶бһ“бһ‘бҹҒ?

бһ”бһ¶бһ‘ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„бһҖбһ¶бһҡбһ–бҹ’бһҡбһҳбһ¶бһ“бҹ” бһҖбһ¶бһҡбһҮбһ·бҹҮбһҖбһ„бҹӢбһҸбһ¶бһҳбһҠбһ„бһ•бҹ’бһӣбһјбһң TSS бһўбһ¶бһ…бһ–бҹ’бһҷбһ¶бһҖбһҡбһҺбҹҚбһ”бһ¶бһ“бһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһ„бһҳбһ»бһ“бһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҗбһ¶бһҳбһ–бһӣбһҗбҹҒбһҡ (VI 1.02-1.05) бҹ” MTB TSS бһӮбһәвҖӢбһҳбһ¶бһ“вҖӢбһўбһҗбҹҒбһҡвҖӢбһҒбҹ’бһ–бһҹбҹӢ (VI 1.10-1.20+) бһҠбҹ„бһҷвҖӢбһҳбһ¶бһ“вҖӢбһҖбһ¶бһҡвҖӢбһ•бҹ’бһ‘бһ»бҹҮвҖӢбһүбһ№бһҖбһүбһ¶бһ”бҹӢвҖӢбһӣбһҫбһҹвҖӢбһҖбһҳбҹ’бһҡбһ·бһҸбҹ” бһҠбһјбһ…бһӮбҹ’бһ“бһ¶ TSS бһ–бһё MTB бһўбһ¶бһ…бһҳбһ¶бһ“бһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһ–бһ·бһ”бһ¶бһҖбһҮбһ¶бһ„бһҳбһ»бһ“ бһҠбҹ„бһҷбһҹбһ¶бһҡбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбҹ” бһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбһ“бһ·бһ“бҹ’бһ“бһ¶бһҖбһ¶бһҡбһҮбһ¶бһҮбһ¶бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ”бһҠбһ¶бһ…бҹӢбһҒбһ¶бһҸбһҡбһңбһ¶бһ„бһңбһ·бһүбҹ’бһүбһ¶бһҹбһ¶бҹ”

бһҸбһҫбһўбҹ’бһңбһёбһҮбһ¶ TSS бһӣбҹ’бһўбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ бһ¶бһҸбҹӢбһ”бҹ’бһҡбһ¶бһҺбһҳбһҪбһҷ?

бһңбһ¶бһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһӣбһҫбһӮбҹ„бһӣбһҠбҹ…бһ“бҹғбһҖбһ¶бһҡбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбҹ– бһҖбһ¶бһҡбһҮбһ·бҹҮбһЎбһҫбһ„бһңбһ·бһүбҹ– 20-50 TSS бһҖбһ¶бһҡбһҹбҹҠбһјбһ‘бҹ’бһҡбһ¶бҹҶбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбҹ– 50-100 TSS бһҖбһ¶бһҡбһҮбһ·бҹҮбһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҡбһ·бһҸбһҳбһ’бҹ’бһҷбһҳбҹ– 100-150 TSS бһҖбһ¶бһҡбһ бһ¶бһҸбҹӢбһ”бҹ’бһҡбһ¶бһҺбһҒбҹ’бһӣбһ¶бҹҶбһ„бҹ– 150-250 TSS бһ–бһ·бһ”бһ¶бһҖбһҺбһ¶бһҹбҹӢ 02 бһңбһӮбҹ’бһӮ TSSбҹ” бһҖбһ¶бһҡбһҮбһ·бҹҮбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһ’бҹ’бһӣбһ¶бһҖбҹӢбһ…бһ“бҹ’бһӣбҹ„бҹҮбһ–бһё 50-150 TSSбҹ” бһ…бһ“бҹ’бһӣбҹ„бҹҮбһ–бҹҒбһӣбһӮбһ»бһҺбһ—бһ¶бһ–бһ”бһ„бҹ’бһҖбһҫбһҸ 100-200 TSS бҹ” бһҖбһ¶бһҡбһ”бҹ’бһҡбһҺбһ¶бҹҶбһ„ бһ“бһ·бһ„бһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһҡбһҷбҹҲбһ–бҹҒбһӣбһңбҹӮбһ„бҹ– 200-400+ TSS бҹ”

бһҸбһҫбһҒбҹ’бһүбһ»бҹҶбһӮбһҪбһҡбһ’бҹ’бһңбһҫ TSS бһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“бһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһҹбһ”бҹ’бһҸбһ¶бһ бҹҚ?
