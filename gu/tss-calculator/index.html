મફત TSS કેલ્ક્યુલેટર - સાયકલિંગ માટે તાલીમ સ્ટ્રેસ સ્કોર

પાવર, અવધિ અને FTP નો ઉપયોગ કરીને તમારા સાયકલિંગ વર્કઆઉટ્સ માટે તાલીમ તણાવ સ્કોરની ગણતરી કરો

ટ્રેનિંગ સ્ટ્રેસ સ્કોર (TSS) શું છે?

તાલીમ તણાવ સ્કોર (TSS)તમારા પાવર આઉટપુટના આધારે તીવ્રતા અને અવધિને સંયોજિત કરીને સાયકલિંગ વર્કઆઉટના તાલીમ લોડનું પ્રમાણ નક્કી કરે છે. ડૉ. એન્ડ્રુ કોગન દ્વારા વિકસિત, TSS તમારા કાર્યાત્મક થ્રેશોલ્ડ પાવર (FTP) નો સંદર્ભ બિંદુ તરીકે ઉપયોગ કરે છે. FTP = 100 TSS પર 1-કલાકનું વર્કઆઉટ.

મફત TSS કેલ્ક્યુલેટર

પાવર ડેટાનો ઉપયોગ કરીને કોઈપણ સાયકલિંગ વર્કઆઉટ માટે તાલીમ તણાવની ગણતરી કરો.

વોટ્સમાં તમારી કાર્યાત્મક થ્રેશોલ્ડ પાવર
મિનિટમાં રાઈડનો કુલ સમય (1-720)
તમારા રાઈડ ડેટામાંથી NP (સરેરાશ પાવર નહીં)

તમારો તાલીમ તણાવ સ્કોર

TSS
--
તીવ્રતા પરિબળ (IF)
--

વર્કઆઉટમાં મુશ્કેલી

JavaScript દ્વારા ભરેલ

TSS ની ગણતરી કેવી રીતે કરવામાં આવે છે

TSS ફોર્મ્યુલા

TSS = (સેકન્ડ × NP × IF) / (FTP × 3600) × 100

ક્યાં:

  • NP (સામાન્ય શક્તિ)= વોટ્સમાં સવારીની શારીરિક "કિંમત".
  • IF (તીવ્રતા પરિબળ)= NP / FTP (થ્રેશોલ્ડને સંબંધિત તીવ્રતા)
  • અવધિ= સેકન્ડમાં કુલ રાઈડ સમય
  • FTP= વોટ્સમાં તમારી કાર્યાત્મક થ્રેશોલ્ડ પાવર

સરળીકૃત: TSS = અવધિ (કલાક) × IF² × 100

કાર્ય કરેલ ઉદાહરણો

ઉદાહરણ 1: સરળ સહનશક્તિ રાઈડ

રાઇડર ડેટા:

  • FTP:250W
  • અવધિ:120 મિનિટ (7200)
  • સામાન્ય શક્તિ:150W

પગલું 1: તીવ્રતા પરિબળની ગણતરી કરો

IF = NP / FTP
IF = 150W / 250W
IF =0.60

પગલું 2: TSS ની ગણતરી કરો

TSS = (7200 × 150 × 0.60) / (250 × 3600) × 100
TSS = (648,000) / (900,000) × 100
TSS =72 TSS

અર્થઘટન:60% તીવ્રતા પર સરળ સહનશક્તિ સવારી - એરોબિક બેઝ બિલ્ડિંગ અને પુનઃપ્રાપ્તિ માટે યોગ્ય.

ઉદાહરણ 2: થ્રેશોલ્ડ અંતરાલ

રાઇડર ડેટા:

  • FTP:250W
  • અવધિ:90 મિનિટ (5400)
  • સામાન્ય શક્તિ:235W

પગલું 1: તીવ્રતા પરિબળની ગણતરી કરો

IF = NP / FTP
IF = 235W / 250W
IF =0.94

પગલું 2: TSS ની ગણતરી કરો

TSS = (5400 × 235 × 0.94) / (250 × 3600) × 100
TSS = (1,192,860) / (900,000) × 100
TSS =133 TSS

અર્થઘટન:94% તીવ્રતા પર સખત થ્રેશોલ્ડ સત્ર - FTP સુધારણા માટે નોંધપાત્ર તાલીમ ઉત્તેજના.

ઉદાહરણ 3: હાર્ડ ગ્રુપ રાઈડ

રાઇડર ડેટા:

  • FTP:250W
  • અવધિ:180 મિનિટ (10800)
  • સામાન્ય શક્તિ:210W

પગલું 1: તીવ્રતા પરિબળની ગણતરી કરો

IF = NP / FTP
IF = 210W / 250W
IF =0.84

પગલું 2: TSS ની ગણતરી કરો

TSS = (10800 × 210 × 0.84) / (250 × 3600) × 100
TSS = (1,905,120) / (900,000) × 100
TSS =212 TSS

અર્થઘટન:84% તીવ્રતા પર લાંબી સખત સવારી - ઉચ્ચ તાલીમ લોડ માટે 1-2 દિવસની પુનઃપ્રાપ્તિની જરૂર છે.

ઉદાહરણ 4: VO₂max અંતરાલ

રાઇડર ડેટા:

  • FTP:250W
  • અવધિ:75 મિનિટ (4500)
  • સામાન્ય શક્તિ:270W

પગલું 1: તીવ્રતા પરિબળની ગણતરી કરો

IF = NP / FTP
IF = 270W / 250W
IF =1.08

પગલું 2: TSS ની ગણતરી કરો

TSS = (4500 × 270 × 1.08) / (250 × 3600) × 100
TSS = (1,312,200) / (900,000) × 100
TSS =146 TSS

અર્થઘટન:થ્રેશોલ્ડ ઉપર ખૂબ જ સખત VO₂max સત્ર- ટૂંકી અવધિ હોવા છતાં ઉચ્ચ તાલીમ ઉત્તેજના.

સાયકલ સવારના સ્તર દ્વારા સાપ્તાહિક TSS લક્ષ્યો

શિખાઉ માણસ સાયકલ સવારો

સાપ્તાહિક TSS: 200-400

દર અઠવાડિયે 3-4 સવારી, 50-100 TSS દરેક. એરોબિક બેઝ અને બાઇક હેન્ડલિંગ કૌશલ્ય બનાવવા પર ધ્યાન કેન્દ્રિત કરો.

મનોરંજન સાયકલ સવારો

સાપ્તાહિક TSS: 400-600

દર અઠવાડિયે 4-5 રાઈડ, 80-120 TSS દરેક. સહનશક્તિ અને કેટલાક ગુણવત્તા સત્રોનું મિશ્રણ.

સ્પર્ધાત્મક એમેચ્યોર્સ

સાપ્તાહિક TSS: 600-900

દર અઠવાડિયે 5-7 સવારી, 85-130 TSS દરેક. પીરિયડાઇઝેશન અને રેસિંગ સાથે સંરચિત તાલીમ.

એલિટ / પ્રોફેશનલ

સાપ્તાહિક TSS: 900-1500+

દર અઠવાડિયે 10-15+ સત્રો. ઉચ્ચ વોલ્યુમ વ્યાવસાયિક તાલીમ લોડ. પોસ્ટ-ગ્રાન્ડ ટૂર CTL: 150-170.

⚠️ TSS વિશે મહત્વપૂર્ણ નોંધો

  • ચોક્કસ FTP ની જરૂર છે:ચોક્કસ TSS માટે તમારું FTP વર્તમાન (4-6 અઠવાડિયામાં પરીક્ષણ) હોવું આવશ્યક છે.
  • સામાન્ય શક્તિનો ઉપયોગ કરો, સરેરાશ શક્તિ નહીં:NP પરિવર્તનશીલતા અને શારીરિક ખર્ચ માટે જવાબદાર છે.
  • ઇન્ડોર વિ આઉટડોર:ઇન્ડોર TSS કોસ્ટિંગ અથવા ડિસેન્સ ન હોવાને કારણે વધુ કઠણ અનુભવી શકે છે.
  • વ્યક્તિગત ભિન્નતા:સમાન TSS જુદા જુદા રાઇડર્સ માટે અલગ લાગે છે. તમારી પુનઃપ્રાપ્તિના આધારે એડજસ્ટ કરો.
  • રેમ્પ રેટ મહત્વપૂર્ણ છે:સાપ્તાહિક TSS ને ધીમે ધીમે વધારો—દર અઠવાડિયે 3-8 CTL પોઈન્ટ ટકાઉ છે.

નોર્મલાઇઝ્ડ પાવર (એનપી) ને સમજવું

TSS ની ગણતરી કરવા માટે સામાન્ય શક્તિ એ સરેરાશ પાવર કરતાં વધુ સચોટ છે કારણ કે તે પાવર વેરિએબિલિટીના શારીરિક ખર્ચ માટે જવાબદાર છે:

શા માટે NP બાબતો: બે રાઇડ્સની સરખામણી

રાઇડ A: સ્ટેડી ટેમ્પો

  • અવધિ: 60 મિનિટ
  • સરેરાશ પાવર: 200W
  • સામાન્ય શક્તિ: 202W
  • પરિવર્તનશીલતા સૂચકાંક: 1.01
  • TSS: 65

સ્થિર, સતત પ્રયત્નો - ઓછી શારીરિક કિંમત

રાઈડ B: હાર્ડ ગ્રુપ રાઈડ
  • અવધિ: 60 મિનિટ
  • સરેરાશ પાવર: 200W
  • સામાન્ય શક્તિ: 240W
  • પરિવર્તનશીલતા સૂચકાંક: 1.20
  • TSS: 92

વધારા સાથે વેરિયેબલ - ઘણી વધારે શારીરિક કિંમત

મુખ્ય આંતરદૃષ્ટિ:બંને રાઇડ્સમાં 200W એવરેજ છે, પરંતુ પાવર વેરિએબિલિટીને કારણે રાઇડ B 42% સખત (92 vs 65 TSS) છે. NP આ તફાવતને પકડે છે.

💡 સામાન્ય શક્તિ કેવી રીતે મેળવવી

મોટાભાગના સાયકલિંગ કોમ્પ્યુટર અને એપ્સ એનપીની આપમેળે ગણતરી કરે છે:

  • ગાર્મિન/વાહૂ/હેમરહેડ:રાઇડ સારાંશમાં NP બતાવે છે
  • TrainingPeaks/Strava/Intervals.icu:અપલોડ કરેલી રાઇડ્સમાંથી NPની ગણતરી કરે છે
  • ગોલ્ડન ચિતા/WKO5:અદ્યતન એનપી વિશ્લેષણ સાધનો

જો તમારી પાસે માત્ર સરેરાશ પાવર હોય, તો અંદાજ કાઢો: સ્થિર રાઈડ માટે NP ≈ સરેરાશ પાવર × 1.03-1.05, વેરિયેબલ રાઈડ માટે × 1.10-1.15.

શા માટે TSS બાબતો: CTL, ATL, TSB

તાલીમ સ્ટ્રેસ સ્કોર એ પર્ફોર્મન્સ મેનેજમેન્ટ ચાર્ટનો પાયો છે:

  • CTL (ક્રોનિક ટ્રેનિંગ લોડ):તમારું ફિટનેસ સ્તર - દૈનિક TSS ની 42-દિવસની ઘાતાંકીય રીતે ભારિત સરેરાશ
  • ATL (એક્યુટ ટ્રેનિંગ લોડ):તમારો થાક - દૈનિક TSS ની 7-દિવસની ઘાતક રીતે ભારિત સરેરાશ
  • TSB (તાલીમ સ્ટ્રેસ બેલેન્સ):તમારું ફોર્મ - TSB = CTL - ATL (પોઝિટિવ = તાજું, નકારાત્મક = થાકેલું)
  • સમયગાળો:લક્ષ્ય CTL પ્રગતિનો ઉપયોગ કરીને તાલીમ તબક્કાઓ (બેઝ, બિલ્ડ, પીક, ટેપર) ની યોજના બનાવો
  • રેસનો સમય:ટોચના પ્રદર્શન માટે રેસના દિવસે +10 થી +25 ની TSB હાંસલ કરવા માટે ટેપર

પ્રો ટીપ: તમારા પરફોર્મન્સ મેનેજમેન્ટ ચાર્ટને ટ્રૅક કરો

TrainingPeaks, Intervals.icu અથવા સ્પ્રેડશીટમાં દરરોજ TSS લોગ કરો. તમારા CTL (42-દિવસની સરેરાશ) અને ATL (7-દિવસની સરેરાશ) નું નિરીક્ષણ કરો. બેઝ બિલ્ડીંગ દરમિયાન દર અઠવાડિયે 3-8 પોઈન્ટ્સની સ્થિર CTL વૃદ્ધિનું લક્ષ્ય રાખો. રેસના 7-14 દિવસ પહેલા TSB ને પીક પરફોર્મન્સ માટે સકારાત્મક મૂલ્યો સુધી પહોંચવા દેવા માટે TSS ને ઘટાડો.

વારંવાર પૂછાતા પ્રશ્નો

જો મારી પાસે પાવર મીટર ન હોય તો શું?

TSS ને પાવર ડેટાની જરૂર છે (FTP અને નોર્મલાઈઝ્ડ પાવર). પાવર મીટર વિના, તમે HRSS (હાર્ટ રેટ સ્ટ્રેસ સ્કોર) જેવા હાર્ટ રેટ-આધારિત વિકલ્પોનો ઉપયોગ કરી શકો છો અથવા કથિત પરિશ્રમ અંદાજનો ઉપયોગ કરી શકો છો, પરંતુ આ ઓછા સચોટ છે. સાયકલિંગમાં TSS ગણતરી અને તાલીમ લોડ મેનેજમેન્ટ માટે પાવર મીટર એ ગોલ્ડ સ્ટાન્ડર્ડ છે.

TSS કેટલું સચોટ છે?

વર્તમાન FTP અને યોગ્ય રીતે ગણતરી કરેલ સામાન્ય શક્તિ પર આધારિત હોય ત્યારે TSS અત્યંત સચોટ છે. અભ્યાસો દર્શાવે છે કે TSS તાલીમ લોડ અને પુનઃપ્રાપ્તિ જરૂરિયાતોની વિશ્વસનીય આગાહી કરે છે. ચોકસાઈ આના પર નિર્ભર કરે છે: તાજેતરની FTP પરીક્ષણ (4-6 અઠવાડિયાની અંદર), સાચી NP ગણતરી અને પાવર મીટરની ચોકસાઈ (મોટા ભાગના ±1-2% છે).

શું હું રોડ અને MTB વચ્ચે TSS ની સરખામણી કરી શકું?

હા, પરંતુ ચેતવણીઓ સાથે. રોડ સાયકલિંગ TSS સ્થિર પાવર પ્રોફાઇલ્સ (VI 1.02-1.05) સાથે વધુ અનુમાનિત છે. MTB TSS થ્રેશોલ્ડ ઉપર વારંવાર વિસ્ફોટ સાથે અત્યંત ચલ (VI 1.10-1.20+) છે. MTB તરફથી સમાન TSS ટેકનિકલ માંગણીઓને કારણે મુશ્કેલ લાગે છે. વિદ્યાશાખાઓ વચ્ચે નિરપેક્ષ સરખામણી કરવાને બદલે વલણ વિશ્લેષણ પર ધ્યાન કેન્દ્રિત કરો.

એક વર્કઆઉટ માટે સારું TSS શું છે?

તે તાલીમના લક્ષ્યો પર આધાર રાખે છે: પુનઃપ્રાપ્તિ રાઇડ્સ: 20-50 TSS, સરળ સહનશક્તિ: 50-100 TSS, મધ્યમ રાઇડ્સ: 100-150 TSS, સખત વર્કઆઉટ્સ: 150-250 TSS, ખૂબ જ સખત સત્રો: 000 PROCT + 00TECT+ રેસ. મોટાભાગની તાલીમ રાઇડ્સ 50-150 TSS ની વચ્ચે આવે છે. ગુણવત્તા અંતરાલ 100-200 TSS જનરેટ કરે છે. રેસિંગ અને લાંબી ઇવેન્ટ્સ: 200-400+ TSS.

મારે દર અઠવાડિયે કેટલું TSS કરવું જોઈએ?

સ્તર દ્વારા સાપ્તાહિક TSS લક્ષ્યો: પ્રારંભિક: 200-400 TSS/અઠવાડિયું, મનોરંજન: 400-600 TSS/અઠવાડિયું, સ્પર્ધાત્મક કલાપ્રેમી: 600-900 TSS/અઠવાડિયું, એલિટ/પ્રોફેશનલ: 900-51/અઠવાડિયા. રૂઢિચુસ્ત રીતે પ્રારંભ કરો અને દર અઠવાડિયે 3-8 CTL પોઈન્ટ્સ વધારો. તમારું ટકાઉ સાપ્તાહિક TSS તાલીમ ઇતિહાસ, ઉપલબ્ધ સમય અને પુનઃપ્રાપ્તિ ક્ષમતા પર આધારિત છે.

શું મારે ઇન્ડોર તાલીમ માટે TSS નો ઉપયોગ કરવો જોઈએ?

ચોક્કસ. TSS ઇન્ડોર તાલીમ માટે આદર્શ છે કારણ કે પાવર સુસંગત છે અને તેમાં કોઈ પર્યાવરણીય ચલ નથી. ઇન્ડોર TSS તમારા CTL/ATL/TSBમાં સીધો ફાળો આપે છે. જો કે, કોસ્ટિંગ, ડિસેન્સ અથવા પવન-સહાયિત વિભાગો ન હોવાને કારણે સમાન TSS માટે આઉટડોર કરતાં ઇન્ડોર રાઇડ્સ વધુ મુશ્કેલ લાગે છે.

TSS અને કિલોજુલ્સ વચ્ચે શું તફાવત છે?

કિલોજુલ્સ કુલ કરેલા કામને માપે છે (ઊર્જા ખર્ચ)—સમાન શક્તિ પર દરેક માટે સમાન. TSS તમારા FTP ને સંબંધિત તાલીમ તણાવને માપે છે. ઉદાહરણ: 1 કલાક માટે 200W = દરેક માટે 720 kJ. પરંતુ જો તમારું FTP 200W છે, TSS = 100. જો તમારું FTP 300W છે, તો TSS = 44. TSS વ્યક્તિગત છે; kJ સંપૂર્ણ છે.

શું મારે મારું FTP જાણવાની જરૂર છે?

હા, TSS ની ગણતરી માટે FTP આવશ્યક છે. તમારા FTP ને જાણ્યા વિના, તમે તીવ્રતા પરિબળ અથવા TSS ની ગણતરી કરી શકતા નથી. 20-મિનિટ અથવા 8-મિનિટના પરીક્ષણનો ઉપયોગ કરીને તમારા FTPનું પરીક્ષણ કરો અથવા રેમ્પ ટેસ્ટ પ્રોટોકોલનો ઉપયોગ કરો. TSS ગણતરીઓને સચોટ રાખવા માટે દર 4-6 અઠવાડિયે ફરીથી પરીક્ષણ કરો કારણ કે તમારી ફિટનેસ સુધરે છે. વધુ જાણો:FTP પરીક્ષણ માર્ગદર્શિકા
