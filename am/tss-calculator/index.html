ነጻ TSS ካልኩሌተር - የብስክሌት ስፖርት የጭንቀት ነጥብ

የኃይል፣ የቆይታ ጊዜ እና FTP በመጠቀም ለብስክሌት ልምምዶችዎ የስልጠና ውጥረትን ያስሉ

የስልጠና ውጥረት ነጥብ (TSS) ምንድን ነው?

የስልጠና ውጥረት ነጥብ (TSS)በኃይል ውፅዓትዎ ላይ በመመስረት ጥንካሬን እና የቆይታ ጊዜን በማጣመር የብስክሌት ስፖርታዊ እንቅስቃሴን የስልጠና ጭነት ይለካል። በዶ/ር አንድሪው ኮግገን የተገነባ፣ TSS የእርስዎን የተግባር ገደብ ኃይል (FTP) እንደ ማመሳከሪያ ነጥብ ይጠቀማል። የ1 ሰአት ስፖርታዊ እንቅስቃሴ በPROTET10X = 100 TSS።

ነጻ TSS ማስያ

የኃይል መረጃን በመጠቀም ለማንኛውም የብስክሌት ስፖርታዊ እንቅስቃሴ የሥልጠና ጭንቀትን አስላ።

የእርስዎ ተግባራዊ ገደብ ኃይል በዋት
ጠቅላላ የጉዞ ጊዜ በደቂቃ (1-720)
NP ከእርስዎ የጉዞ ውሂብ (አማካይ ኃይል አይደለም)

የእርስዎ የስልጠና ውጥረት ነጥብ

TSS
--
የኃይለኛነት ሁኔታ (IF)
--

የአካል ብቃት እንቅስቃሴ ችግር

በጃቫስክሪፕት ተሞልቷል።

TSS እንዴት እንደሚሰላ

TSS ፎርሙላ

TSS = (ሰከንዶች × NP × IF) / (FTP × 3600) × 100

የት፡

  • NP (የተለመደ ኃይል)= ፊዚዮሎጂያዊ "ዋጋ" በዋት ውስጥ የሚጋልቡ
  • IF (የጥንካሬ ሁኔታ)= NP / FTP (ከመጠን በላይ ጥንካሬ)
  • ቆይታ= አጠቃላይ የመጓጓዣ ጊዜ በሰከንዶች ውስጥ
  • FTP= የእርስዎ ተግባራዊ ገደብ ኃይል በዋት

ቀለል ያለ፡ TSS = የሚፈጀው ጊዜ (ሰዓታት) × IF² × 100

የተሰሩ ምሳሌዎች

ምሳሌ 1፡ ቀላል የጽናት ጉዞ

የአሽከርካሪ መረጃ፡

  • FTP:250 ዋ
  • የሚፈጀው ጊዜ፡-120 ደቂቃዎች (7200ዎች)
  • መደበኛ ኃይል;150 ዋ

ደረጃ 1፡ የኃይለኛነት ሁኔታን አስላ

IF = NP / FTP
IF = 150 ዋ / 250 ዋ
IF =0.60

ደረጃ 2፡ TSS አስላ

TSS = (7200 × 150 × 0.60) / (250 × 3600) × 100
TSS = (648,000) / (900,000) × 100
TSS =72 TSS

ትርጓሜ፡-ቀላል የጽናት ግልቢያ በ 60% ጥንካሬ - ለኤሮቢክ መሠረት ግንባታ እና ለማገገም ፍጹም።

ምሳሌ 2፡ የመግቢያ ክፍተቶች

የአሽከርካሪ መረጃ፡

  • FTP:250 ዋ
  • የሚፈጀው ጊዜ፡-90 ደቂቃዎች (5400ዎች)
  • መደበኛ ኃይል;235 ዋ

ደረጃ 1፡ የኃይለኛነት ሁኔታን አስላ

IF = NP / FTP
IF = 235 ዋ / 250 ዋ
IF =0.94

ደረጃ 2፡ TSS አስላ

TSS = (5400 × 235 × 0.94) / (250 × 3600) × 100
TSS = (1,192,860) / (900,000) × 100
TSS =133 TSS

ትርጓሜ፡-የጠንካራ ደረጃ ክፍለ ጊዜ በ 94% ጥንካሬ - ለ FTP መሻሻል ጠቃሚ የሥልጠና ማነቃቂያ።

ምሳሌ 3፡ ሃርድ ግሩፕ ግልቢያ

የአሽከርካሪ መረጃ፡

  • FTP:250 ዋ
  • የሚፈጀው ጊዜ፡-180 ደቂቃዎች (10800ዎች)
  • መደበኛ ኃይል;210 ዋ

ደረጃ 1፡ የኃይለኛነት ሁኔታን አስላ

IF = NP / FTP
IF = 210 ዋ / 250 ዋ
IF =0.84

ደረጃ 2፡ TSS አስላ

TSS = (10800 × 210 × 0.84) / (250 × 3600) × 100
TSS = (1,905,120) / (900,000) × 100
TSS =212 TSS

ትርጓሜ፡-ረጅም ከባድ ግልቢያ በ 84% ጥንካሬ - ከፍተኛ የሥልጠና ጭነት ከ1-2 ቀናት ማገገምን ይፈልጋል።

ምሳሌ 4፡ VO₂max ክፍተቶች

የአሽከርካሪ መረጃ፡

  • FTP:250 ዋ
  • የሚፈጀው ጊዜ፡-75 ደቂቃዎች (4500 ሰ)
  • መደበኛ ኃይል;270 ዋ

ደረጃ 1፡ የኃይለኛነት ሁኔታን አስላ

IF = NP / FTP
IF = 270W / 250 ዋ
IF =1.08

ደረጃ 2፡ TSS አስላ

TSS = (4500 × 270 × 1.08) / (250 × 3600) × 100
TSS = (1,312,200) / (900,000) × 100
TSS =146 TSS

ትርጓሜ፡-በጣም ከባድ የ VO₂max ክፍለ ጊዜ ከአጭር ጊዜ በላይ ቢሆንም ከፍተኛ የሥልጠና ማነቃቂያ።

TSS መመሪያዎች በአካል ብቃት እንቅስቃሴ ዓይነት

የአካል ብቃት እንቅስቃሴ ዓይነትTSS ክልልየኃይለኛነት ሁኔታመግለጫ

ሳምንታዊ TSS ዒላማዎች በብስክሌት ነጂ ደረጃ

ጀማሪ ብስክሌተኞች

ሳምንታዊ TSS፡200-400

በሳምንት 3-4 ጉዞዎች፣ እያንዳንዳቸው 50-100 TSS። የኤሮቢክ መሠረት እና የብስክሌት አያያዝ ችሎታዎችን በመገንባት ላይ ያተኩሩ።

የመዝናኛ ብስክሌተኞች

ሳምንታዊ TSS፡400-600

በሳምንት 4-5 ጉዞዎች፣ እያንዳንዳቸው 80-120 TSS። የጽናት ድብልቅ እና አንዳንድ ጥራት ያላቸው ክፍለ ጊዜዎች።

ተወዳዳሪ አማተር

ሳምንታዊ TSS፡600-900

በሳምንት 5-7 ጉዞዎች፣ እያንዳንዳቸው 85-130 TSS። በወቅታዊነት እና በእሽቅድምድም የተዋቀረ ስልጠና።

ልሂቃን / ባለሙያ

ሳምንታዊ TSS፡900-1500+

በሳምንት ከ10-15+ ክፍለ ጊዜዎች። ከፍተኛ መጠን ያለው ሙያዊ ስልጠና ጭነት. የድህረ-ግራንድ ጉብኝት CTL: 150-170.

⚠️ ስለ TSS ጠቃሚ ማስታወሻዎች

  • ትክክለኛ FTP ያስፈልገዋል፡የእርስዎ FTP ወቅታዊ መሆን አለበት (በ4-6 ሳምንታት ውስጥ መሞከር) ለ TSS።
  • አማካይ ኃይል ሳይሆን መደበኛ ኃይል ይጠቀሙ፡-NP ለተለዋዋጭነት እና የፊዚዮሎጂ ወጪን ይይዛል።
  • የቤት ውስጥ ከውጪየቤት ውስጥ TSS ምንም የባህር ዳርቻ ወይም የትውልድ ቦታ ባለመኖሩ ምክንያት ከባድ ስሜት ሊሰማው ይችላል።
  • የግለሰብ ልዩነት;ለተለያዩ አሽከርካሪዎች ተመሳሳይ TSS የተለየ ስሜት ይሰማዋል። በማገገምዎ ላይ በመመስረት ያስተካክሉ።
  • የራምፕ ፍጥነት ጉዳዮች፡-ሳምንታዊ TSS ቀስ በቀስ ይጨምሩ—በሳምንት 3-8 CTL ነጥቦች ዘላቂ ናቸው።

መደበኛ ኃይል (NP) መረዳት

ከጉልበት ጋር ተለዋዋጭ - በጣም ከፍተኛ የፊዚዮሎጂ ወጪ

ቁልፍ ግንዛቤ፡-ሁለቱም ግልቢያዎች 200W አማካኝ አላቸው፣ነገር ግን Ride B 42% ከባድ ነው (92 vs 65 TSS) በሃይል ልዩነት። NP ይህንን ልዩነት ይይዛል.

💡 መደበኛ ሃይል እንዴት ማግኘት እንደሚቻል

አብዛኛዎቹ የብስክሌት ኮምፒተሮች እና መተግበሪያዎች NPን በራስ ሰር ያሰላሉ፡-

  • ጋርሚን/ዋሁ/መዶሻ ራስ፡ግልቢያ ማጠቃለያ ላይ NP ያሳያል
  • የስልጠና ጫፎች/Strava/Intervals.icu፡NP ከተሰቀሉ ግልቢያዎች ያሰላል
  • ወርቃማው አቦሸማኔ/WKO5፡የላቀ የ NP ትንተና መሳሪያዎች

አማካኝ ሃይል ብቻ ካሎት፣ ግምት፡ NP ≈ አማካኝ ሃይል × 1.03-1.05 ለቋሚ ጉዞዎች፣ ለተለዋዋጭ ጉዞዎች × 1.10-1.15።

ለምን TSS ጉዳዮች፡- CTL፣ ATL፣ TSB

የሥልጠና ውጥረት ውጤት የአፈጻጸም አስተዳደር ገበታ መሠረት ነው፡-

  • CTL (ሥር የሰደደ የሥልጠና ጭነት)የእርስዎ የአካል ብቃት ደረጃ - 42-ቀን በአስደናቂ ክብደት ያለው የዕለታዊ TSS አማካይ
  • ATL (አጣዳፊ የሥልጠና ጭነት)ድካምዎ - የ 7-ቀን በከፍተኛ ሁኔታ የተመዘነ አማካይ የዕለታዊ TSS
  • TSB (የስልጠና ውጥረት ሚዛን)የእርስዎ ቅጽ - TSB = CTL - ATL (አዎንታዊ = ትኩስ ፣ አሉታዊ = ድካም)
  • ወቅታዊነት፡የታለመ CTL ግስጋሴዎችን በመጠቀም የስልጠና ደረጃዎችን ያቅዱ (መሰረታዊ ፣ ግንባታ ፣ ጫፍ ፣ ቴፐር)
  • የውድድር ጊዜ፡ለከፍተኛ አፈጻጸም በሩጫ ቀን TSB ከ +10 እስከ +25 ለመድረስ መታ ያድርጉ

Pro ጠቃሚ ምክር፡ የእርስዎን የአፈጻጸም አስተዳደር ገበታ ይከታተሉ

በየቀኑ TSS በ TrainingPeaks፣ Intervals.icu ወይም የተመን ሉህ ውስጥ ይመዝገቡ. የእርስዎን CTL (የ42-ቀን አማካኝ) እና ATL (የ7-ቀን አማካኝ) ይቆጣጠሩ። በመሠረት ግንባታ ወቅት በየሳምንቱ ከ3-8 ነጥብ ያለው የ CTL እድገትን ያቅዱ። TSB ለከፍተኛ አፈጻጸም ወደ አወንታዊ እሴቶች እንዲያድግ TSS ከሩጫ 7-14 ቀናት በፊት ይቀንሱ።

ስለ CTL፣ ATL እና TSB → የበለጠ ይወቁ

በተደጋጋሚ የሚጠየቁ ጥያቄዎች

የኃይል ቆጣሪ ከሌለኝስ?

TSS የኃይል መረጃን ይፈልጋል (FTP እና መደበኛ ኃይል)። የኃይል መለኪያ ከሌለ እንደ HRSS (የልብ ምት ጭንቀት ነጥብ) ያሉ የልብ ምት ላይ የተመሰረቱ አማራጮችን መጠቀም ወይም የተገመቱ የግምገማ ግምቶችን መጠቀም ይችላሉ ነገርግን እነዚህ ትክክለኛ አይደሉም። የኃይል ሜትሮች የ TSS ስሌት እና የብስክሌት ጭነት አስተዳደር ስልጠና የወርቅ ደረጃዎች ናቸው።

TSS ምን ያህል ትክክል ነው?

TSS አሁን ባለው FTP እና በትክክል በተሰላ መደበኛ ሃይል ላይ በመመስረት በጣም ትክክለኛ ነው። ጥናቶች TSS የስልጠና ጭነት እና የመልሶ ማግኛ ፍላጎቶችን በአስተማማኝ ሁኔታ ይተነብያል። ትክክለኝነት የሚወሰነው በ፡ የቅርብ ጊዜ FTP ሙከራ (በ4-6 ሳምንታት ውስጥ)፣ ትክክለኛው የኤንፒ ስሌት እና የሃይል መለኪያ ትክክለኛነት (አብዛኞቹ ±1-2%) ናቸው።

TSS በመንገድ እና በኤምቲቢ መካከል ማወዳደር እችላለሁ?

አዎ ፣ ግን ከማስጠንቀቂያዎች ጋር። የመንገድ ቢስክሌት TSS በተረጋጋ የኃይል መገለጫዎች (VI 1.02-1.05) የበለጠ ሊተነበይ ይችላል። MTB TSS በጣም ተለዋዋጭ ነው (VI 1.10-1.20+) ከመነሻው በላይ ብዙ ጊዜ የሚፈነዳ ነው። ከኤምቲቢ ተመሳሳይ TSS በቴክኒካዊ ፍላጎቶች ምክንያት ከባድ ስሜት ሊሰማው ይችላል። በዲሲፕሊን መካከል ፍጹም ንፅፅር ሳይሆን በአዝማሚያ ትንተና ላይ ያተኩሩ።

ለአንድ የአካል ብቃት እንቅስቃሴ ጥሩ TSS ምንድነው?

በስልጠና ግቦች ላይ የተመሰረተ ነው: የመልሶ ማግኛ ጉዞዎች: 20-50 TSS, ቀላል ጽናት: 50-100 TSS, መጠነኛ ጉዞዎች: 100-150 TSS, ከባድ የአካል ብቃት እንቅስቃሴዎች: 150-250 TSS, በጣም ከባድ ክፍለ ጊዜዎች / ውድድሮች: 0CTRO + 4000. አብዛኛዎቹ የስልጠና ግልቢያዎች በ50-150 TSS መካከል ይወድቃሉ። የጥራት ክፍተቶች 100-200 TSS ያመነጫሉ. እሽቅድምድም እና ረጅም ክስተቶች፡ 200-400+ TSS።

በሳምንት ምን ያህል TSS ማድረግ አለብኝ?

ሳምንታዊ TSS ኢላማዎች በደረጃ፡ ጀማሪዎች፡ 200-400 TSS/ሳምንት፣ መዝናኛ፡ 400-600 TSS/ሳምንት፣ ተወዳዳሪ አማተር፡ 600-900 TSS/ሳምንት፣ Elite/ፕሮፌሽናል፡ 900-1k. በጥንቃቄ ይጀምሩ እና በሳምንት በ3-8 CTL ነጥቦች ይጨምሩ። የእርስዎ ቀጣይነት ያለው ሳምንታዊ TSS በስልጠና ታሪክ፣ ባለው ጊዜ እና በማገገም ችሎታ ላይ የተመሰረተ ነው።

ለቤት ውስጥ ስልጠና TSS መጠቀም አለብኝ?

በፍጹም። TSS ለቤት ውስጥ ስልጠና ተስማሚ ነው, ምክንያቱም ኃይል የማይለዋወጥ እና ምንም የአካባቢ ተለዋዋጮች የሉም. የቤት ውስጥ TSS በቀጥታ ለእርስዎ CTL/ATL/TSB አስተዋፅዖ ያደርጋል። ነገር ግን፣ የቤት ውስጥ ግልቢያ ከቤት ውጭ ለተመሳሳይ TSS ምንም አይነት የባህር ዳርቻ፣ የቁልቁል መውረድ ወይም በነፋስ የታገዘ ክፍል በሌለበት ሁኔታ ከባድ ሊሰማቸው ይችላል።

በ TSS እና በኪሎጁል መካከል ያለው ልዩነት ምንድን ነው?

ኪሎጁል የተከናወነውን አጠቃላይ ሥራ (የኃይል ወጪዎችን) ይለካሉ - ለሁሉም በተመሳሳይ ኃይል ተመሳሳይ ነው። TSS ከእርስዎ ጥበቃ10X አንጻር የስልጠና ውጥረትን ይለካል። ምሳሌ: 200W ለ 1 ሰዓት = 720 ኪ. ነገር ግን የእርስዎ FTP 200W ከሆነ TSS = 100. የእርስዎ FTP 300W ከሆነ TSS = 44. TSS ግላዊ ነው; ኪጄ ፍጹም ነው።

የእኔን FTP ማወቅ አለብኝ?

አዎ FTP ለ TSS ስሌት አስፈላጊ ነው። የእርስዎን FTP ሳያውቁ የ Intensity Factor ወይም TSS ማስላት አይችሉም። የእርስዎን FTP የ20 ደቂቃ ወይም የ8 ደቂቃ ፈተና በመጠቀም ይሞክሩት ወይም የራምፕ ሙከራ ፕሮቶኮሎችን ይጠቀሙ። የአካል ብቃትዎ እየተሻሻለ ሲመጣ TSS ስሌቶችን በትክክል ለማቆየት በየ4-6 ሳምንቱ እንደገና ይሞክሩ። የበለጠ ተማር፡FTP የሙከራ መመሪያ
