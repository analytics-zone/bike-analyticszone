Kebijakan Privasi untuk Bike Analytics

Terakhir Diperbarui: 10 Januari 2025 | Tanggal Efektif: 10 Januari 2025

Pengantar

Bike Analytics ("kami", atau "aplikasi") berkomitmen untuk melindungi privasi Anda. Kebijakan Privasi ini menjelaskan bagaimana aplikasi seluler kami (iOS dan Android) mengakses, menggunakan, dan melindungi data kesehatan dari perangkat Anda.

Prinsip Privasi Utama: Bike Analytics beroperasi pada arsitektur tanpa server, hanya lokal. Semua data kesehatan yang diakses dari Apple HealthKit (iOS) atau Health Connect (Android) tetap secara eksklusif berada di perangkat fisik Anda dan tidak pernah dikirimkan ke server eksternal, layanan cloud, atau pihak ketiga.

1. Akses Data Kesehatan

Bike Analytics terintegrasi dengan platform kesehatan asli perangkat Anda untuk memberikan analisis latihan bersepeda yang mendetail:

1.1 iOS - Integrasi Apple HealthKit

Pada perangkat iOS, Bike Analytics terintegrasi dengan Apple HealthKit untuk mengakses data latihan bersepeda. Kami meminta akses baca-saja ke:

  • Sesi Latihan: Sesi latihan bersepeda dengan waktu dan durasi
  • Jarak: Total jarak bersepeda
  • Detak Jantung: Data detak jantung selama latihan
  • Energi Aktif: Kalori yang terbakar selama sesi bersepeda
  • Daya Bersepeda: Data daya (watt) untuk analisis performa
  • Kecepatan Bersepeda: Data kecepatan untuk analisis kecepatan

Kepatuhan Apple HealthKit: Bike Analytics mematuhi semua pedoman Apple HealthKit. Data kesehatan Anda diproses sepenuhnya di perangkat iOS Anda dan tidak pernah meninggalkannya. Kami tidak pernah membagikan data HealthKit dengan pihak ketiga, platform periklanan, atau broker data.

1.2 Android - Integrasi Health Connect

Izin Android: Izin ini diminta saat peluncuran pertama aplikasi. Anda dapat mencabut izin ini kapan saja melalui Pengaturan Android ‚Üí Aplikasi ‚Üí Health Connect ‚Üí Bike Analytics.

1.3 Bagaimana Kami Menggunakan Data Kesehatan

Semua data kesehatan digunakan secara eksklusif untuk tujuan-tujuan berikut:

  • Tampilan Latihan: Menampilkan sesi bersepeda Anda dengan metrik terperinci (jarak, waktu, kecepatan, detak jantung, daya)
  • Analitik Performa: Menghitung zona daya, FTP (Functional Threshold Power), dan TSS (Training Stress Score)
  • Pelacakan Kemajuan: Menampilkan tren performa, rekor pribadi, dan ringkasan latihan
  • Ekspor Data: Memungkinkan Anda mengekspor data latihan Anda ke format CSV untuk penggunaan pribadi

1.4 Penyimpanan Data

üîí JAMINAN PRIVASI KRITIS:

Semua data kesehatan tetap secara eksklusif berada di perangkat fisik Anda.

  • iOS: Data disimpan menggunakan iOS Core Data dan UserDefaults (hanya di perangkat)
  • Android: Data disimpan menggunakan Android Room Database (SQLite di perangkat)
  • TIDAK ADA data yang diunggah ke server eksternal
  • TIDAK ADA data yang dikirimkan melalui internet
  • TIDAK ADA sinkronisasi cloud atau cadangan data kesehatan
  • TIDAK ADA akses pihak ketiga ke data kesehatan Anda

Satu-satunya saat data meninggalkan perangkat Anda adalah ketika ANDA secara eksplisit memilih untuk mengekspor latihan Anda ke format CSV dan membagikan file itu sendiri.

2. Izin yang Diperlukan

2.1 Izin iOS

  • Akses HealthKit: Akses baca ke Latihan, Jarak, Detak Jantung, Energi Aktif, Kecepatan Bersepeda, dan Daya Bersepeda
  • Perpustakaan Foto (Opsional): Hanya jika Anda memilih untuk menyimpan ringkasan latihan sebagai gambar

Anda dapat mengelola izin HealthKit kapan saja di Pengaturan iOS ‚Üí Privasi & Keamanan ‚Üí Kesehatan ‚Üí Bike Analytics.

2.2 Izin Android

  • android.permission.health.READ_EXERCISE
  • android.permission.health.READ_DISTANCE
  • android.permission.health.READ_HEART_RATE
  • android.permission.health.READ_SPEED
  • android.permission.health.READ_POWER
  • android.permission.health.READ_TOTAL_CALORIES_BURNED
  • Akses Internet (INTERNET): Digunakan hanya untuk menampilkan konten statis dalam aplikasi dan mengakses manajemen langganan.

3. Data yang TIDAK Kami Kumpulkan

Bike Analytics TIDAK mengumpulkan, menyimpan, atau mengirimkan:

  • ‚ùå Informasi identifikasi pribadi (nama, email, nomor telepon)
  • ‚ùå Pengenal perangkat (IDFA di iOS, ID iklan di Android)
  • ‚ùå Data lokasi atau koordinat GPS
  • ‚ùå Analitik penggunaan atau pelacakan perilaku aplikasi
  • ‚ùå Laporan kerusakan atau data diagnostik ke server eksternal
  • ‚ùå Data apa pun melalui SDK pihak ketiga atau layanan analitik

4. Pembelian Dalam Aplikasi dan Langganan

Bike Analytics menawarkan langganan dalam aplikasi opsional yang dikelola melalui sistem pembayaran asli perangkat Anda (App Store atau Google Play). Kami tidak menerima detail pembayaran, hanya status langganan.

5. Penyimpanan dan Penghapusan Data

5.1 Retensi Data

Data kesehatan disimpan di perangkat Anda tanpa batas waktu hingga dihapus secara manual. Anda mengontrol retensi data Anda.

5.2 Penghapusan Data

Hapus latihan individual dalam aplikasi, atau hapus semua data aplikasi melalui pengaturan sistem atau uninstal. Cabut izin kesehatan kapan saja.

6. Keamanan Data

Kami menggunakan enkripsi tingkat perangkat (iOS Keychain, Android Sandbox). Tidak ada transmisi data yang terjadi, menghilangkan risiko jaringan.

7. Berbagi Data dan Pihak Ketiga

Bike Analytics TIDAK membagikan data kesehatan Anda dengan pihak ketiga mana pun. Kami tidak menjual data atau menggunakan SDK periklanan.

8. Privasi Anak-Anak

Bike Analytics tidak secara sadar mengumpulkan data dari anak-anak di bawah 13 tahun. Orang tua harus mengawasi penggunaan aplikasi.

9. Perubahan pada Kebijakan Privasi Ini

Kami dapat memperbarui kebijakan ini. Tinjau secara berkala. Penggunaan berkelanjutan menyiratkan penerimaan.
