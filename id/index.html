FAQ

Pertanyaan yang Sering Diajukan

Bagaimana aplikasi analitik sepeda ini mendapatkan data saya?

Bike Analytics menyinkronkan dengan Apple Health untuk mengimpor latihan sepeda yang direkam oleh perangkat atau aplikasi apa pun yang kompatibel. Ini termasuk komputer sepeda, smart trainer, dan entri manual. Aplikasi memproses data ini secara lokal untuk menghitung metrik performa sepeda tingkat lanjut.

Apa itu FTP dan bagaimana cara mengujinya?

FTP (Functional Threshold Power) adalah daya maksimum yang dapat Anda pertahankan selama kira-kira satu jam. Aplikasi menyertakan protokol tes FTP 20 menit: pemanasan secara menyeluruh, lalu berkendara dengan upaya berkelanjutan maksimum selama 20 menit. FTP Anda adalah 95% dari daya rata-rata. Ulangi setiap 6-8 minggu untuk melacak kemajuan dan memperbarui zona latihan.

Apakah saya memerlukan power meter untuk menggunakan Bike Analytics?

Ya. Bike Analytics memerlukan data daya dari power meter untuk perhitungan FTP, TSS, dan zona latihan yang akurat. Kompatibel dengan semua power meter yang menyinkronkan ke Apple Health melalui komputer sepeda atau smart trainer.

Bagaimana TSS bersepeda dihitung?

Training Stress Score (TSS) untuk bersepeda dihitung menggunakan data daya dan FTP Anda. Rumusnya mempertimbangkan intensitas (daya normalisasi vs FTP) dan durasi untuk mengukur stres latihan. Ini memungkinkan pelacakan kebugaran yang akurat melalui CTL, pemantauan kelelahan melalui ATL, dan penilaian bentuk dengan TSB.

Apa perbedaan antara analitik jalan raya dan MTB?

Bike Analytics menawarkan mode analisis khusus untuk sepeda jalan raya dan sepeda gunung. Mode jalan raya menggunakan penghalusan daya standar 30 detik, sementara mode MTB menerapkan algoritma berbeda untuk memperhitungkan medan dan upaya yang sangat bervariasi yang khas dalam berkendara off-road. Setiap mode menyediakan metrik spesifik disiplin.

Apakah data sepeda saya pribadi?

Ya. Bike Analytics memproses semua data sepeda secara lokal di iPhone Anda. Tidak ada server eksternal, tidak ada akun cloud, tidak ada transfer data. Anda mengontrol ekspor: buat file JSON, CSV, HTML, atau PDF dengan metrik performa sepeda Anda dan bagikan sesuka Anda.

Bisakah saya mengekspor data sepeda saya untuk dibagikan dengan pelatih?

Tentu saja. Ekspor perjalanan dan metrik performa dalam berbagai format: JSON untuk pengembang, CSV untuk spreadsheet, HTML untuk tampilan web, atau PDF untuk laporan yang dapat dicetak. Bagikan melalui email, pesan, atau metode berbagi file apa pun yang Anda suka.

Apa perbedaan antara paket bulanan dan tahunan?

Kedua paket menawarkan fitur yang identik: semua metrik performa sepeda, zona latihan tanpa batas, mode jalan raya & MTB, beberapa ekspor, dan pembaruan gratis. Satu-satunya perbedaan adalah harga: tahunan menghemat 18% (setara dengan ‚Ç¨3,25/bulan vs ‚Ç¨3,99/bulan).

Bisakah saya membatalkan langganan saya kapan saja?

Ya. Langganan dikelola melalui App Store, jadi Anda dapat membatalkan kapan saja dari Pengaturan ‚Üí [Nama Anda] ‚Üí Langganan. Jika Anda membatalkan, Anda akan tetap memiliki akses hingga akhir periode penagihan Anda saat ini.
