সাইক্লিং পাওয়ার মেট্রিক্স ব্যাখ্যা: NP, IF, VI, W'bal

স্মার্ট প্রশিক্ষণ এবং প্রতিযোগিতার জন্য উন্নত পাওয়ার মেট্রিক্স বোঝা

গড় পাওয়ারের বাইরে: উন্নত মেট্রিক্স কেন গুরুত্বপূর্ণ

গড় পাওয়ার একটি ভোঁতা উপকরণ। একই গড় পাওয়ারের দুটি রাইডের শারীরবৃত্তীয় খরচ সম্পূর্ণ ভিন্ন হতে পারে, নির্ভর করে প্রচেষ্টা কতটা পরিবর্তনশীল ছিল তার উপর। এক ঘন্টার জন্য স্থির ২৫০W সম্পূর্ণ ভিন্ন অনুভূতি দেয় ১৫০W এবং ৪০০W এর মধ্যে ওঠানামা করে গড়ে ২৫০W করার থেকে।

এখানেই উন্নত পাওয়ার মেট্রিক্স আসে। নরমালাইজড পাওয়ার (NP), ইনটেনসিটি ফ্যাক্টর (IF), ভ্যারিয়েবিলিটি ইনডেক্স (VI), এবং W' ব্যালেন্স (W'bal) সূক্ষ্ম বোঝাপড়া প্রদান করে যা আপনার কার্যকরভাবে প্রশিক্ষণ নিতে, প্রতিযোগিতায় সঠিকভাবে গতি নিয়ন্ত্রণ করতে এবং আপনার প্রচেষ্টার প্রকৃত খরচ বুঝতে প্রয়োজন।

দ্রুত তুলনা: রোড বনাম MTB

  • রোড ক্লাইম্বিং (১০ মিনিট): ২৪৬W গড়, ২৪৬W NP, VI = ১.০০
  • MTB ক্লাইম্বিং (১০ মিনিট): ২২০W গড়, ২৬৫W NP, VI = ১.২০
  • ফলাফল:কম গড় পাওয়ারের MTB প্রচেষ্টা আসলে শারীরবৃত্তীয়ভাবে বেশি কঠিন

নরমালাইজড পাওয়ার (NP): পরিবর্তনশীল প্রচেষ্টার প্রকৃত খরচ

নরমালাইজড পাওয়ার (NP) পাওয়ার আউটপুট এবং ক্লান্তির মধ্যে অ-রৈখিক সম্পর্ক বিবেচনা করে একটি রাইডের শারীরবৃত্তীয় "খরচ" অনুমান করে। থ্রেশহোল্ডের উপরে উচ্চ-তীব্রতার প্রচেষ্টা স্থির-অবস্থা রাইডিংয়ের তুলনায় অসমানভাবে ক্লান্তি বাড়ায়।

NP এর পিছনের অ্যালগরিদম

NP একটি ওজনযুক্ত গণনা ব্যবহার করে যা কঠিন প্রচেষ্টাকে জোর দেয়:

  1. ৩০-সেকেন্ডের রোলিং গড়: সেকেন্ড-থেকে-সেকেন্ড শব্দ মসৃণ করুন
  2. ৪র্থ শক্তিতে বাড়ান: উচ্চ-তীব্রতার প্রচেষ্টা বিবর্ধিত করুন (২০০W⁴ বনাম ৩০০W⁴ বিশাল)
  3. ৪র্থ শক্তি মানের গড় নিন: ওজনযুক্ত প্রচেষ্টার গড় খুঁজুন
  4. ৪র্থ মূল নিন: ওয়াটে ফিরে রূপান্তর করুন

সরলীকৃত সূত্র

NP = ⁴√([৩০ সেকেন্ডের রোলিং গড় পাওয়ার]⁴ এর গড়)

৪র্থ শক্তি ওজন মানে ৪০০W তে ১০-সেকেন্ডের সার্জ আপনার শরীরে ২০ সেকেন্ডের জন্য ২০০W রাইড করার চেয়ে অনেক বেশি "খরচ" করে।

NP কখন সবচেয়ে গুরুত্বপূর্ণ

🚵 মাউন্টেন বাইকিং

MTB রেসিং পরিবর্তনশীল পাওয়ার দ্বারা সংজ্ঞায়িত। একটি সাধারণ XC রেসে প্রতিটি ৫-২৫ সেকেন্ড স্থায়ী FTP এর উপরে ৮০+ সার্জ অন্তর্ভুক্ত। গড় পাওয়ার প্রকৃত প্রচেষ্টাকে গুরুতরভাবে অবমূল্যায়ন করে।

উদাহরণ XC রেস:

  • গড় পাওয়ার: ২৪৫W
  • নরমালাইজড পাওয়ার: ২৮৫W
  • VI: ১.১৬ (অত্যন্ত পরিবর্তনশীল)
  • ব্যাখ্যা: রেসটি ২৮৫W স্থির রাইড করার মতো অনুভূত হয়েছিল—২৪৫W নয়

🏁 ক্রাইটেরিয়াম এবং সার্কিট রেস

ক্রাইটেরিয়ামে কোণ থেকে বেরিয়ে আসার সময় ক্রমাগত ত্বরণ, আক্রমণ এবং পজিশনিং প্রচেষ্টা থাকে। সার্জ-এবং-পুনরুদ্ধার প্যাটার্ন গড় পাওয়ারের তুলনায় উচ্চ NP তৈরি করে।

উদাহরণ ক্রাইটেরিয়াম:

  • গড় পাওয়ার: ২২০W
  • নরমালাইজড পাওয়ার: ২৬৫W
  • VI: ১.২০
  • ৬০ মিনিটে >৩০০W তে ৮৮টি ত্বরণ

🚴 সার্জ সহ গ্রুপ রাইড

আক্রমণ এবং ব্রিজিং প্রচেষ্টা সহ আগ্রাসী গ্রুপ রাইডগুলি উচ্চ NP উৎপন্ন করে এমনকি যখন ক্রুজিং গতি মাঝারি মনে হয়।

NP কখন সাহায্য করে না

টাইম ট্রায়াল, স্থির আরোহণ এবং একক টেম্পো রাইড NP গড় পাওয়ারের প্রায় সমান উৎপন্ন করে (VI = ১.০০-১.০৩)। এই ক্ষেত্রে, গড় পাওয়ার যথেষ্ট।

🔬 গবেষণার ভিত্তি

ডঃ অ্যান্ড্রু কগান শারীরবৃত্তীয় গবেষণার উপর ভিত্তি করে নরমালাইজড পাওয়ার তৈরি করেছিলেন যা দেখায় যে ATP হ্রাস, গ্লাইকোজেন ব্যবহার, ল্যাকটেট জমা এবং কার্ডিওভাস্কুলার চাপ তীব্রতার সাথে ঘাতীয়ভাবে বৃদ্ধি পায়—রৈখিকভাবে নয়। ৪র্থ শক্তি সম্পর্ক এই অ-রৈখিক ক্লান্তি প্রতিক্রিয়ার আনুমানিক মান দেয়।

ইনটেনসিটি ফ্যাক্টর (IF): আপেক্ষিক প্রচেষ্টা পরিমাপ

ইনটেনসিটি ফ্যাক্টর (IF) আপনার ফাংশনাল থ্রেশহোল্ড পাওয়ার (FTP) এর তুলনায় একটি রাইড কতটা কঠিন ছিল তা প্রকাশ করে। এটি নরমালাইজড পাওয়ার এবং FTP এর অনুপাত।

সূত্র

IF = নরমালাইজড পাওয়ার (NP) / FTP

উদাহরণ:

৩০০W FTP সহ রাইডার ২৫৫W এর NP সহ রাইড সম্পূর্ণ করে:

IF = ২৫৫ / ৩০০ = ০.৮৫

এটি একটি মাঝারি-কঠিন প্রচেষ্টা প্রতিনিধিত্ব করে।

ইনটেনসিটি ফ্যাক্টর ব্যাখ্যা

IF পরিসীমাপ্রচেষ্টা স্তরউদাহরণ ওয়ার্কআউটটেকসই সময়কাল
< ০.৬৫সহজ/পুনরুদ্ধারপুনরুদ্ধার স্পিন, সহজ গ্রুপ রাইড৩-৬+ ঘন্টা
০.৬৫-০.৭৫সহনশীলতা/মাঝারিদীর্ঘ স্থির রাইড, বেস ট্রেনিং২-৫ ঘন্টা
০.৭৫-০.৮৫টেম্পো/মাঝারি-কঠিনটেম্পো ইন্টারভাল, স্পোর্টিভ পেস১-৩ ঘন্টা
০.৮৫-০.৯৫থ্রেশহোল্ড/কঠিনসুইট স্পট, থ্রেশহোল্ড ইন্টারভাল৪০-৯০ মিনিট
০.৯৫-১.০৫FTP/খুব কঠিনFTP টেস্ট, ৪০km TT৩০-৬০ মিনিট
১.০৫-১.১৫VO2max/অত্যন্ত কঠিনVO2max ইন্টারভাল, ক্রাইটেরিয়াম১০-৩০ মিনিট
> ১.১৫অ্যানারোবিক/সর্বোচ্চসংক্ষিপ্ত TT, ট্র্যাক পারস্যুট< ১০ মিনিট

প্রশিক্ষণের জন্য IF ব্যবহার

সেশন ডিজাইন

লক্ষ্য IF ওয়ার্কআউটের তীব্রতা নির্ধারণ করতে সাহায্য করে:

  • সহজ দিন: IF < ০.৬৫ পুনরুদ্ধার নিশ্চিত করে
  • টেম্পো সেশন: IF ০.৮০-০.৮৫ এরোবিক ক্ষমতা তৈরি করে
  • থ্রেশহোল্ড কাজ: IF ০.৯০-১.০০ FTP উন্নত করে

রেস বিশ্লেষণ

রেস-পরবর্তী IF প্রকাশ করে যে পেসিং উপযুক্ত ছিল কিনা:

  • IF খুব বেশি: খুব কঠিন শুরু, শেষে ম্লান
  • IF উপযুক্ত: সমান প্রচেষ্টা, শক্তিশালী ফিনিশ
  • IF খুব কম: খুব বেশি আটকে রাখা, শক্তি বাকি ছিল

TSS গণনায় IF

IF হল ট্রেনিং স্ট্রেস স্কোর (TSS) এর একটি মূল উপাদান, যা তীব্রতা এবং সময়কাল একত্রিত করে মোট প্রশিক্ষণ লোড পরিমাপ করে।

TSS সূত্র

TSS = (IF)² × সময়কাল (ঘন্টা) × ১০০

IF বর্গ করা হয়, মানে তীব্রতার প্রশিক্ষণ চাপের উপর ঘাতীয় প্রভাব রয়েছে।

ভ্যারিয়েবিলিটি ইনডেক্স (VI): স্থিরতা মেট্রিক

ভ্যারিয়েবিলিটি ইনডেক্স (VI) একটি রাইডের সময় আপনার পাওয়ার আউটপুট কতটা পরিবর্তনশীল ছিল তা পরিমাপ করে। এটি নরমালাইজড পাওয়ার এবং গড় পাওয়ারের অনুপাত।

সূত্র

VI = নরমালাইজড পাওয়ার (NP) / গড় পাওয়ার

উদাহরণ:

২৭০W এর NP এবং ২৫০W এর গড় পাওয়ার সহ রাইড:

VI = ২৭০ / ২৫০ = ১.০৮

মাঝারিভাবে পরিবর্তনশীল প্রচেষ্টা (কিছু সার্জ সহ গ্রুপ রাইডের জন্য সাধারণ)।

ট্রেইল রাইড১.০৮-১.১৫প্রযুক্তিগত অংশ, আরোহণ, অবতরণ
এন্ডুরো১.১৫-১.৩০+সংক্ষিপ্ত বিস্ফোরক প্রচেষ্টা, মধ্যবর্তী পুনরুদ্ধার

উচ্চ VI কেন গুরুত্বপূর্ণ

বর্ধিত অ্যানারোবিক চাপ

উচ্চ VI থ্রেশহোল্ডের উপরে ঘন ঘন প্রচেষ্টা নির্দেশ করে, বারবার W' (অ্যানারোবিক ক্ষমতা) হ্রাস করে। এটি একই গড় পাওয়ারে স্থির রাইডিংয়ের চেয়ে অনেক বেশি ক্লান্তিকর।

গ্লাইকোজেন হ্রাস

পরিবর্তনশীল প্রচেষ্টা একই গড় পাওয়ারে স্থির প্রচেষ্টার চেয়ে বেশি গ্লাইকোজেন পোড়ায়। উচ্চ VI রাইডে আরও ঘন ঘন জ্বালানি প্রয়োজন।

নিউরোমাসকুলার ক্লান্তি

বারবার ত্বরণ এবং কঠিন প্রচেষ্টা কার্ডিওভাস্কুলার লোড যা পূর্বাভাস দেবে তার বাইরে নিউরোমাসকুলার ক্লান্তি তৈরি করে।

শাখাগুলিকে আলাদা করতে VI ব্যবহার

VI হল প্রাথমিক মেট্রিক যা রোড সাইক্লিং এবং মাউন্টেন বাইকিংকে আলাদা করে:

🚴 রোড সাইক্লিস্ট

VI: ১.০২-১.০৫

স্থির পাওয়ার, ন্যূনতম সার্জ। থ্রেশহোল্ডে FTP এবং টেকসই পাওয়ারে ফোকাস।

🚵 মাউন্টেন বাইকার

VI: ১.১০-১.২০+

বার্স্ট পাওয়ার, ঘন ঘন সার্জ। পুনরাবৃত্তিযোগ্যতা এবং W' ব্যবস্থাপনায় ফোকাস।

⚠️ পেসিং প্রভাব

যে রেসগুলিতে পেসিং গুরুত্বপূর্ণ (রোড রেস, TT, ম্যারাথন MTB), VI < ১.০৫ লক্ষ্য করুন। প্রতিটি সার্জ আপনাকে স্থির রাইডিংয়ের চেয়ে বেশি খরচ করে, এবং শুরুতে উচ্চ VI মানে আপনি শেষে ম্লান হবেন।

ব্যতিক্রম: ক্রাইটেরিয়াম এবং প্রযুক্তিগত MTB রেসগুলি প্রকৃতিগতভাবে উচ্চ VI প্রয়োজন। এই চাহিদার জন্য বিশেষভাবে প্রশিক্ষণ নিন।

W' ব্যালেন্স (W'bal): আপনার অ্যানারোবিক ব্যাটারি

W' ব্যালেন্স (W'bal) একটি রাইডের সময় রিয়েল-টাইমে আপনার অবশিষ্ট অ্যানারোবিক ক্ষমতা ট্র্যাক করে। এটি একটি ব্যাটারি গেজের মতো যা দেখায় ক্রিটিকাল পাওয়ার (CP) এর উপরে প্রচেষ্টার জন্য আপনার কতটা শক্তি উপলব্ধ রয়েছে।

W' এবং CP বোঝা

ক্রিটিকাল পাওয়ার (CP) হল সর্বোচ্চ পাওয়ার যা আপনি দীর্ঘ সময়ের জন্য বজায় রাখতে পারেন—এরোবিক এবং অ্যানারোবিক বিপাকের মধ্যে সীমানা। FTP এর অনুরূপ কিন্তু আরও বৈজ্ঞানিকভাবে শক্তিশালী।

W' (W-প্রাইম) হল CP এর উপরে কাজের জন্য আপনার সীমিত ক্ষমতা, কিলোজুল (kJ) এ পরিমাপ করা হয়। সাধারণ মান ১৫-২৫ kJ থেকে পরিসীমা।

CP এবং W' সাদৃশ্য

CP = টেকসই গতি (অনির্দিষ্টকালের জন্য একটি সমতল রাস্তায় রাইড করার মতো)
W' = সার্জের জন্য ব্যাটারি (একটি বুস্ট বোতামের মতো যা আপনি শুধুমাত্র এত দীর্ঘ ব্যবহার করতে পারেন)

যখন আপনি CP এর উপরে রাইড করেন, আপনি W' হ্রাস করেন। যখন আপনি CP এর নীচে রাইড করেন, W' পুনরুদ্ধার হয়—তবে পুনরুদ্ধার হ্রাসের চেয়ে ধীর।

W'bal কীভাবে কাজ করে

CP এর উপরে হ্রাস

প্রতিটি সেকেন্ড আপনি CP এর উপরে রাইড করেন, আপনি W' "খরচ" করেন CP এর উপরে আপনি কতটা দূরে আছেন তার সমানুপাতিক হারে:

  • ৩০০W প্রচেষ্টা যখন CP ২৮০W → ২০ J/s এ W' হ্রাস
  • ৩৫০W প্রচেষ্টা যখন CP ২৮০W → ৭০ J/s এ W' হ্রাস (অনেক দ্রুত!)

CP এর নীচে পুনরুদ্ধার

যখন আপনি CP এর নীচে রাইড করেন, W' ঘাতীয়ভাবে পুনরুদ্ধার হয়, প্রায় ৩০০-৫০০ সেকেন্ডের একটি সময় ধ্রুবক সহ (ক্লান্তি স্তরের উপর নির্ভর করে)।

সরলীকৃত W' ব্যালেন্স সমীকরণ

হ্রাস: W'bal প্রতি সেকেন্ডে (পাওয়ার - CP) দ্বারা হ্রাস পায়
পুনরুদ্ধার: W'bal ঘাতীয়ভাবে W'max এর দিকে পুনরুদ্ধার হয়

প্রকৃত মডেল (স্কিবা, বার্ট্রাম) আরও জটিল, পুনরুদ্ধার হারের উপর ক্লান্তি প্রভাব বিবেচনা করে।

বাস্তব-বিশ্বের উদাহরণ: MTB রেস

পরিস্থিতি: প্রযুক্তিগত আরোহণ সহ XC MTB রেস

রাইডার প্রোফাইল:

  • CP: ২৮০W
  • W': ১৮,০০০ J (১৮ kJ)

রেস সেগমেন্ট:

  1. স্টার্ট সার্জ (২০ সেকেন্ড @ ৪০০W):
    • W' হ্রাস: (৪০০-২৮০) × ২০ = ২,৪০০ J
    • W'bal: ১৮,০০০ → ১৫,৬০০ J (৮৭% অবশিষ্ট)
  2. পুনরুদ্ধার পেডালিং (৬০ সেকেন্ড @ ২২০W):
    • আংশিক W' পুনরুদ্ধার: ~১,৮০০ J
    • W'bal: ১৫,৬০০ → ১৭,৪০০ J (৯৭% অবশিষ্ট)
  3. প্রযুক্তিগত আরোহণ (৫ মিনিট @ ৩২০W গড়, ৬টি সার্জ ৩৮০W এ):
    • টেকসই CP-উপরে প্রচেষ্টা + সার্জ থেকে প্রধান W' হ্রাস
    • W'bal: ১৭,৪০০ → ৪,২০০ J (২৩% অবশিষ্ট)
  4. অবতরণ পুনরুদ্ধার (৩ মিনিট @ ১৫০W):
    • W' পুনরুদ্ধার: ~৬,০০০ J
    • W'bal: ৪,২০০ → ১০,২০০ J (৫৭% অবশিষ্ট)

বিশ্লেষণ: রাইডার আরেকটি কঠিন প্রচেষ্টা সামলাতে পারে তবে W'bal ২০% এর নীচে হ্রাস করা এড়ানো উচিত নতুবা তারা আক্রমণে সাড়া দিতে অক্ষম হবে।

রেস কৌশলের জন্য W'bal ব্যবহার

পেসিং কৌশল

  • আরোহণে W'bal পর্যবেক্ষণ করুন: এত গভীরে যাবেন না যে আপনি শীর্ষে আক্রমণে সাড়া দিতে পারবেন না
  • পুনরুদ্ধার সময়কাল সর্বাধিক করুন: W' রিচার্জ করতে অবতরণ এবং সমতল অংশে নরম-পেডাল করুন
  • চূড়ান্ত প্রচেষ্টা পরিকল্পনা করুন: স্প্রিন্ট বা চূড়ান্ত আরোহণের জন্য আপনার কতটা W' বাকি আছে তা জানুন

প্রশিক্ষণ প্রয়োগ

  • ইন্টারভাল ডিজাইন: W' বারবার হ্রাস/পুনরুদ্ধার করতে ইন্টারভাল গঠন করুন (পুনরাবৃত্তিযোগ্যতা তৈরি করে)
  • রেস সিমুলেশন: রেস-নির্দিষ্ট পরিস্থিতিতে W'bal পরিচালনা অনুশীলন করুন
  • দুর্বলতা সনাক্তকরণ: কম W' = আরও অ্যানারোবিক ক্ষমতা কাজ প্রয়োজন

💡 MTB-নির্দিষ্ট W'bal প্রশিক্ষণ

বার্স্ট ইন্টারভাল: ২-মিনিট টেম্পো (৯০% FTP) + ১৫-সেকেন্ড সার্জ (১৫০% FTP), ৬-৮x পুনরাবৃত্তি

এটি XC রেসিং চাহিদা অনুকরণ করে: বারবার সার্জ সহ টেকসই টেম্পো। CP এবং W' পুনরুদ্ধার হার উভয়ই প্রশিক্ষণ দেয়।

সম্পর্কিত: আপনার ক্রিটিকাল পাওয়ার এবং W' কীভাবে গণনা করবেন তা শিখুন

🔬 গবেষণার ভিত্তি

W'bal মডেলিং ক্রিটিকাল পাওয়ার ধারণা (Monod & Scherrer, 1965) থেকে উৎপন্ন এবং W' পুনর্গঠনের জন্য ফিলিপ স্কিবার ২০১২ মডেল দ্বারা পরিমার্জিত হয়েছিল। সাম্প্রতিক গবেষণা W'bal কে পরিবর্তনশীল-তীব্রতা ব্যায়ামে সময়-থেকে-ক্লান্তির ভবিষ্যদ্বাণীকারী হিসাবে বৈধতা দেয়।

  • Jones, A.M., et al. (2019). Critical Power: Theory and Applications. Journal of Applied Physiology, 126(6), 1905-1915.

পাওয়ার মেট্রিক্স তুলনা

মেট্রিকএটি কী পরিমাপ করেসূত্রসেরা ব্যবহারের ক্ষেত্র
গড় পাওয়ারগড় পাওয়ার আউটপুটওয়াটের যোগফল / সেকেন্ডস্থির প্রচেষ্টা (TT, একক আরোহণ)
নরমালাইজড পাওয়ার (NP)শারীরবৃত্তীয় খরচ (ওজনযুক্ত)⁴√([৩০s গড়]⁴ এর গড়)পরিবর্তনশীল প্রচেষ্টা (crits, MTB, গ্রুপ রাইড)
ইনটেনসিটি ফ্যাক্টর (IF)FTP এর তুলনায় আপেক্ষিক প্রচেষ্টাNP / FTPসেশন তীব্রতা নির্ধারণ
ভ্যারিয়েবিলিটি ইনডেক্স (VI)পাওয়ার স্থিরতাNP / গড় পাওয়ারপেসিং বিশ্লেষণ, শাখা তুলনা

🚴 রোড টাইম ট্রায়াল (৪০km)

  • গড় পাওয়ার: ৩২০W
  • NP: ৩২৫W
  • IF: ০.৯৮ (FTP = ৩৩২W)
  • VI: ১.০২
  • W'bal: ন্যূনতম হ্রাস (CP এর কাছাকাছি থাকা)

বিশ্লেষণ: পুরোপুরি পেসড TT। VI ১.০ এর কাছাকাছি = সর্বোত্তম স্থিরতা। IF ০.৯৮ = সর্বোচ্চ টেকসই প্রচেষ্টা।

🏁 ক্রাইটেরিয়াম (৬০ মিনিট)

  • গড় পাওয়ার: ২২৫W
  • NP: ২৭৫W
  • IF: ০.৮৩ (FTP = ৩৩২W)
  • VI: ১.২২
  • W'bal: বারবার হ্রাস/পুনরুদ্ধার (৮৮টি সার্জ >৩০০W)

বিশ্লেষণ: উচ্চ VI সার্জ-এবং-পুনরুদ্ধার প্যাটার্ন প্রকাশ করে। NP গড়ের চেয়ে ৫০W বেশি প্রকৃত খরচ দেখায়। W'bal ব্যবস্থাপনা গুরুত্বপূর্ণ ছিল।

  • VI: ১.১৬
  • W'bal: অত্যন্ত পরিবর্তনশীল, একাধিক গভীর হ্রাস

বিশ্লেষণ: গড় পাওয়ার ৪০W দ্বারা প্রচেষ্টাকে অবমূল্যায়ন করে। উচ্চ VI MTB এর সাধারণ। প্রযুক্তিগত আরোহণে W'bal হ্রাস অবতরণে স্মার্ট পুনরুদ্ধার প্রয়োজন।

🏔️ আরোহণ সহ গ্রান ফন্ডো (৫ ঘন্টা)

  • গড় পাওয়ার: ১৯৫W
  • NP: ২১৫W
  • IF: ০.৬৫ (FTP = ৩৩২W)
  • VI: ১.১০
  • W'bal: আরোহণে হ্রাস, অবতরণে পুনরুদ্ধার

বিশ্লেষণ: মাঝারি IF ৫ ঘন্টার জন্য টেকসই। আরোহণে সার্জ থেকে VI ১.১০। এই সময়কালের জন্য সঠিক জ্বালানি গুরুত্বপূর্ণ।

প্রায়শই জিজ্ঞাসিত প্রশ্ন

NP সবসময় গড় পাওয়ারের চেয়ে বেশি কেন?

NP উচ্চ-তীব্রতার প্রচেষ্টাকে আরও ভারীভাবে ওজন দেয় কারণ তারা অসমানুপাতিক ক্লান্তি তৈরি করে। ৪র্থ শক্তি গণনা গড়ের উপরে সার্জকে বিবর্ধিত করে, NP ≥ গড় পাওয়ার করে। পুরোপুরি স্থির প্রচেষ্টার জন্য, NP গড় পাওয়ারের সমান (VI = ১.০)।

বিভিন্ন ওয়ার্কআউটের জন্য একটি "ভাল" ইনটেনসিটি ফ্যাক্টর কী?

পুনরুদ্ধার: IF < ০.৬৫ | সহনশীলতা: IF ০.৬৫-০.৭৫ | টেম্পো: IF ০.৭৫-০.৮৫ | থ্রেশহোল্ড: IF ০.৮৫-০.৯৫ | FTP টেস্ট: IF ০.৯৫-১.০৫। উচ্চ IF = কঠিন সেশন, কিন্তু স্থায়িত্ব সময়কালের উপর নির্ভর করে।

সব রাইডে কি আমার কম VI লক্ষ্য করা উচিত?

না। সর্বোত্তম পেসিংয়ের জন্য টাইম ট্রায়াল এবং একক আরোহণ কম VI (১.০০-১.০৩) থেকে উপকৃত হয়। কিন্তু ক্রাইটেরিয়াম, MTB রেস এবং গ্রুপ রাইডগুলি স্বাভাবিকভাবেই উচ্চ VI (১.১০-১.২৫) থাকে। আপনার লক্ষ্য ইভেন্টের সাথে মেলে এমন VI প্রোফাইল সহ প্রশিক্ষণ নিন।

W'bal FTP থেকে কীভাবে আলাদা?

FTP (বা CP) একটি হার—ওয়াটে টেকসই পাওয়ার। W' একটি ক্ষমতা—কিলোজুলে পরিমাপ করা FTP এর উপরে প্রচেষ্টার জন্য উপলব্ধ মোট শক্তি। চিন্তা করুন: FTP = আপনি ক্রমাগত কত দ্রুত চালাতে পারেন, W' = সেই গতির উপরে বিস্ফোরণের জন্য আপনার নাইট্রাস ট্যাঙ্কের আকার।

প্রশিক্ষণের মাধ্যমে আমি কি আমার W' উন্নত করতে পারি?

হ্যাঁ। উচ্চ-তীব্রতার ইন্টারভাল (VO2max, অ্যানারোবিক) ৮-১২ সপ্তাহে W' ১০-২০% বৃদ্ধি করে। স্প্রিন্ট প্রশিক্ষণ, পাহাড় পুনরাবৃত্তি এবং সংক্ষিপ্ত ইন্টারভাল (৩০s-৩min এ ১২০-১৫০% FTP) বিশেষভাবে W' উন্নয়ন লক্ষ্য করে।

আমার কি ক্রিটিকাল পাওয়ার প্রয়োজন, নাকি FTP যথেষ্ট?

মৌলিক প্রশিক্ষণের জন্য, FTP ভাল কাজ করে। কিন্তু যদি আপনি MTB, ক্রাইটেরিয়াম বা কোনো পরিবর্তনশীল-তীব্রতা ইভেন্ট রেস করেন, CP এবং W' মডেলিং পেসিং কৌশল এবং আপনি কখন আক্রমণ করতে পারেন তা বোঝার জন্য বিশাল সুবিধা প্রদান করে। CP FTP এর চেয়ে বৈজ্ঞানিকভাবে আরও শক্তিশালী।

MTB কেন রোড সাইক্লিংয়ের চেয়ে বেশি VI আছে?

MTB ভূখণ্ড পরিবর্তনশীল পাওয়ার দাবি করে: বাধার উপর ত্বরণ, প্রযুক্তিগত অংশে সার্জ, অবতরণে পুনরুদ্ধার। রোড সাইক্লিং (বিশেষত একক) স্থির পাওয়ার অনুমতি দেয়। এই মৌলিক পার্থক্য মানে MTB রাইডারদের ভিন্ন প্রশিক্ষণ প্রয়োজন—পুনরাবৃত্তিযোগ্যতা এবং W' ব্যবস্থাপনা, শুধুমাত্র টেকসই থ্রেশহোল্ড পাওয়ার নয়।
