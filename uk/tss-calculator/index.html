Безкоштовний калькулятор TSS - показник стресу під час тренування для їзди на велосипеді

Розрахуйте показник стресу під час велоспорту за допомогою потужності, тривалості та FTP

Що таке Training Stress Score (TSS)?

Оцінка тренувального стресу (TSS)визначає кількісне навантаження під час велотренажування шляхом поєднання інтенсивності та тривалості на основі вашої потужності. Розроблений доктором Ендрю Когганом, TSS використовує вашу функціональну порогову потужність (FTP) як точку відліку. 1-годинне тренування на FTP = 100 TSS.

Формула TSS

TSS = (секунди × NP × IF) / (FTP × 3600) × 100

Де:

  • NP (нормалізована потужність)= Фізіологічна «вартість» поїздки у ватах
  • IF (коефіцієнт інтенсивності)= NP / FTP (інтенсивність відносно порогу)
  • Тривалість= Загальний час їзди в секундах
  • ЗАХИСТИТИ10X= Ваша функціональна порогова потужність у ватах

Спрощено: TSS = Тривалість (годин) × IF² × 100

Спрацьовані приклади

Приклад 1: Легка їзда на витривалість

Дані водія:

  • FTP:250 Вт
  • Тривалість:120 хвилин (7200 с)
  • Нормована потужність:150 Вт

Крок 1: Обчисліть коефіцієнт інтенсивності

IF = NP / FTP
IF = 150 Вт / 250 Вт
ЗАХИСТИТИ15X =0,60

Крок 2: обчисліть TSS

TSS = (7200 × 150 × 0,60) / (250 × 3600) × 100
TSS = (648 000) / (900 000) × 100
ЗАХИСТИТИ1X =72 ЗАХИСТИТИ1X

Інтерпретація:Легка їзда на витривалість з інтенсивністю 60%—ідеально підходить для нарощування бази аеробіки та відновлення.

Приклад 2: Порогові інтервали

Дані водія:

  • FTP:250 Вт
  • Тривалість:90 хвилин (5400 с)
  • Нормована потужність:235 Вт

Крок 1: Обчисліть коефіцієнт інтенсивності

IF = NP / FTP
IF = 235 Вт / 250 Вт
ЗАХИСТИТИ15X =0,94

Крок 2: обчисліть TSS

TSS = (5400 × 235 × 0,94) / (250 × 3600) × 100
TSS = (1 192 860) / (900 000) × 100
ЗАХИСТИТИ1X =133 ЗАХИСТИТИ1X

Інтерпретація:Жорстке порогове заняття з інтенсивністю 94% — значний тренувальний стимул для покращення FTP.

Приклад 3: жорстка групова поїздка

Дані водія:

  • FTP:250 Вт
  • Тривалість:180 хвилин (10800 с)
  • Нормована потужність:210 Вт

Крок 1: Обчисліть коефіцієнт інтенсивності

IF = NP / FTP
IF = 210 Вт / 250 Вт
ЗАХИСТИТИ15X =0,84

Крок 2: обчисліть TSS

TSS = (10800 × 210 × 0,84) / (250 × 3600) × 100
TSS = (1 905 120) / (900 000) × 100
ЗАХИСТИТИ1X =212 ЗАХИСТИТИ1X

Інтерпретація:Довга інтенсивна їзда з інтенсивністю 84% — високе навантаження на тренуванні вимагає 1-2 днів відновлення.

Приклад 4: Інтервали VO₂max

Дані водія:

  • FTP:250 Вт
  • Тривалість:75 хвилин (4500 с)
  • Нормована потужність:270 Вт

Крок 1: Обчисліть коефіцієнт інтенсивності

IF = NP / FTP
IF = 270 Вт / 250 Вт
ЗАХИСТИТИ15X =1.08

Крок 2: обчисліть TSS

TSS = (4500 × 270 × 1,08) / (250 × 3600) × 100
TSS = (1 312 200) / (900 000) × 100
ЗАХИСТИТИ1X =146 ЗАХИСТИТИ1X

Інтерпретація:Дуже важкий сеанс VO₂max вище порогу — високий тренувальний стимул, незважаючи на меншу тривалість.

Щотижня TSS:200-400

3-4 поїздки на тиждень, 50-100 TSS кожна. Зосередьтеся на створенні аеробної бази та навичках керування велосипедом.

Велосипедисти для відпочинку

Щотижня TSS:400-600

4-5 поїздок на тиждень, 80-120 TSS кожна. Поєднання витривалості та якісних тренувань.

Змагання аматорів

Щотижня TSS:600-900

5-7 поїздок на тиждень, 85-130 TSS кожна. Структуроване навчання з періодизацією та перегонами.

Елітний / Професійний

Щотижня TSS:900-1500+

10-15+ занять на тиждень. Високий обсяг професійного тренувального навантаження. CTL після великого туру: 150-170.

⚠️ Важливі примітки щодо TSS

  • Потрібен точний FTP:Для точного TSS ваш FTP має бути актуальним (перевірено протягом 4-6 тижнів).
  • Використовуйте нормалізовану потужність, а не середню потужність:NP враховує мінливість і фізіологічну вартість.
  • У приміщенні проти зовнішнього:TSS у приміщенні може здатися важчим через відсутність руху накатом або спусків.
  • Індивідуальна варіація:Один і той самий TSS відчувається по-різному для різних водіїв. Налаштуйте залежно від свого одужання.
  • Швидкість підвищення має значення:Поступово збільшуйте щотижневий TSS — 3-8 балів CTL на тиждень є стабільним.

Розуміння нормалізованої потужності (NP)

Нормована потужність є більш точною, ніж середня потужність для розрахунку TSS, оскільки вона враховує фізіологічну вартість мінливості потужності:

Чому NP має значення: порівняння двох поїздок

Поїздка А: стабільний темп

  • Тривалість: 60 хвилин
  • Середня потужність: 200 Вт
  • Нормована потужність: 202 Вт
  • Індекс мінливості: 1,01
  • TSS: 65

Постійне, постійне зусилля — низька фізіологічна вартість

Ride B: Жорстка групова поїздка
  • Тривалість: 60 хвилин
  • Середня потужність: 200 Вт
  • Нормована потужність: 240 Вт
  • Індекс мінливості: 1,20
  • TSS: 92

Змінна зі сплесками — набагато вищі фізіологічні витрати

Ключова інформація:Обидва атракціони мають середню потужність 200 Вт, але Ride B на 42% важчий (92 проти 65 TSS) через змінність потужності. NP фіксує цю різницю.

💡 Як отримати нормалізовану потужність

Більшість велосипедних комп’ютерів і додатків розраховують NP автоматично:

  • Garmin/Wahoo/Hammerhead:Показує NP у звіті про поїздку
  • TrainingPeaks/Strava/Intervals.icu:Розраховує NP із завантажених поїздок
  • Золотий гепард/WKO5:Розширені інструменти аналізу NP

Якщо у вас лише середня потужність, оцініть: NP ≈ Середня потужність × 1,03-1,05 для постійних поїздок, × 1,10-1,15 для змінних поїздок.

Чому TSS важливий: CTL, ATL, TSB

Оцінка тренувального стресу є основою для діаграми управління ефективністю:

  • CTL (хронічне навантаження під час тренувань):Ваш рівень фізичної підготовки - 42-денне експоненціально зважене середнє щоденне значення TSS
  • ATL (Гостре тренувальне навантаження):Ваша втома - 7-денне експоненціально зважене середнє щоденне значення TSS
  • TSB (Баланс тренувального стресу):Ваша форма - TSB = CTL - ATL (позитив = свіжий, негатив = втомлений)
  • Періодизація:Плануйте фази навчання (базова, нарощування, пік, звуження), використовуючи цільові прогресії CTL
  • Хронометраж перегонів:Звужуйте, щоб досягти TSB від +10 до +25 у день змагань для максимальної продуктивності

Порада професіонала: відстежуйте свою діаграму управління ефективністю

Щодня реєструйте TSS у TrainingPeaks, Intervals.icu або в електронній таблиці. Слідкуйте за CTL (середнє значення за 42 дні) і ATL (середнє значення за 7 днів). Націльте стабільне зростання CTL на 3-8 балів на тиждень під час створення бази. Зменшіть TSS за 7-14 днів до перегонів, щоб TSB піднявся до позитивних значень для максимальної продуктивності.

Часті запитання

Що робити, якщо у мене немає лічильника?

Для TSS потрібні дані про живлення (FTP і нормалізована потужність). Без вимірювача потужності ви можете використовувати альтернативи, засновані на частоті серцевих скорочень, як-от HRSS (показник стресу частоти серцевих скорочень) або використовувати оцінки сприйнятого навантаження, але вони менш точні. Вимірювачі потужності є золотим стандартом для розрахунку TSS і контролю тренувального навантаження у велоспорті.

Наскільки точний TSS?

TSS є високоточним на основі поточного FTP і правильно розрахованої нормалізованої потужності. Дослідження показують, що TSS надійно прогнозує тренувальне навантаження та потреби у відновленні. Точність залежить від: останнього тесту FTP (протягом 4-6 тижнів), правильного розрахунку NP і точності вимірювача потужності (більшість становить ±1-2%).

Чи можу я порівняти TSS між дорожнім і гірським велосипедом?

Так, але із застереженнями. Шосейний велосипед TSS є більш передбачуваним зі стабільними профілями потужності (VI 1.02-1.05). MTB TSS дуже варіабельний (VI 1.10-1.20+) із частими спалахами вище порогу. Той же TSS від MTB може здатися складнішим через технічні вимоги. Зосередьтеся на аналізі тенденцій, а не на абсолютних порівняннях між дисциплінами.

Що таке TSS для одного тренування?

Це залежить від цілей тренувань: відновлювальні заїзди: 20-50 TSS, легка витривалість: 50-100 TSS, помірні заїзди: 100-150 TSS, важкі тренування: 150-250 TSS, дуже важкі заняття/перегони: 200-400+ TSS. Більшість тренувальних поїздок коштує 50-150 TSS. Інтервали якості генерують 100-200 TSS. Перегони та довгі події: 200-400+ TSS.

Скільки TSS я повинен робити на тиждень?

Щотижневі цілі TSS за рівнем: для початківців: 200-400 TSS/тиждень, для рекреаційних: 400-600 TSS/тиждень, для аматорських змагань: 600-900 TSS/тиждень, для еліти/професіоналів: 900-1500+ TSS/тиждень. Почніть консервативно та збільшуйте на 3-8 балів CTL на тиждень. Ваш стабільний щотижневий TSS залежить від історії тренувань, доступного часу та здатності до відновлення.

Чи варто використовувати TSS для тренувань у приміщенні?

Абсолютно. TSS ідеально підходить для тренувань у приміщенні, оскільки потужність постійна і немає змінних навколишнього середовища. Внутрішній TSS безпосередньо сприяє вашому CTL/ATL/TSB. Однак їздити в приміщенні може здатися важчим, ніж на відкритому повітрі, для того ж TSS через те, що немає руху накатом, спусків або секцій з вітром.

Яка різниця між TSS і кілоджоулем?

Кілоджоулі вимірюють загальну виконану роботу (витрати енергії) — однакову для всіх при однаковій потужності. TSS вимірює тренувальний стрес відносно ВАШОГО FTP. Приклад: 200 Вт на 1 годину = 720 кДж на кожного. Але якщо ваш FTP має потужність 200 Вт, TSS = 100. Якщо ваш FTP має потужність 300 Вт, TSS = 44. TSS є індивідуальним; кДж є абсолютним.

Чи потрібно мені знати свій FTP?

Так, FTP необхідний для розрахунку TSS. Не знаючи свого FTP, ви не можете розрахувати коефіцієнт інтенсивності або TSS. Перевірте свій FTP за допомогою 20-хвилинного або 8-хвилинного тесту або скористайтеся протоколами Ramp Test. Повторюйте тест кожні 4-6 тижнів, щоб підтримувати точність розрахунків TSS у міру покращення вашої фізичної форми. Докладніше:Посібник з тестування FTP
