骑行功率指标：NP、IF、VI 和 W'bal 解释

将骑行文件连接到的实用枢纽 FTP, 临界功率, 训练区、节奏决定，以及 TSS

快速解答

骑行功率指标 将原始瓦数转化为决策。 归一化功效 (NP) 估计可变骑行的真实成本， 强度因子 (IF) 显示骑行相对于 FTP, 变异指数（VI） 显示工作的稳定或混乱程度，以及 沃巴尔 帮助解释您花费了多少超出阈值的容量。

  • 最佳用途： 训练处方、比赛配速和骑行后分析
  • 最有用的锚： 一个最新的 FTP 或者 临界功率 价值
  • 最大的错误： 在没有地形、持续时间和事件类型背景的情况下比较 NP、IF 或 W'bal
  • 最好的下一步： 将指标连接到 训练区 和 TSS 而不是孤立地阅读它们

要点

  • NP 告诉你为什么两次平均功率相同的骑行感觉会完全不同
  • IF 将骑行强度联系回 FTP 所以一个数字可以成为一个训练决策
  • 六、 将稳定的公路比赛与高强度的绕圈赛和山地车比赛区分开来
  • 沃巴尔 当您关心重复的高于阈值的努力以及它们之间的恢复时，这是最有用的
  • 真实集群：FTP, 临界功率, 训练区, 空气动力学， 和 TSS

大多数骑自行车的人不需要更多的指标。他们需要一种更好的方法来解释骑行文件中已有的内容。此页面是该系统的实用中心：核心功率指标的含义、它们何时起作用、它们在哪里失败以及您接下来应该使用哪些相关页面。

有用的问题很简单：

  • 这次骑行真的很辛苦，还是只是很吵？
  • 我为这次活动做好了准备吗？
  • 对于本次锻炼，哪个指标更重要：稳定的功率、可变性还是高于阈值的可重复性？
  • 这些数字应该如何改变我接下来要做的事情？

这些指标实际衡量什么

平均功率 告诉你平均发生了什么。核心骑行功率指标告诉您 工作量是如何分配的, 相对于阈值的压力有多大， 和 您是否明智地使用了有限的高于阈值的容量.

公制它衡量什么最适合常见误用
归一化功效 (NP)可变功率输出的生理成本解释艰苦的团体骑行、山地车、暴击、起伏的地形和步速错误将其本身视为起搏目标，无需事件持续时间或地形背景
强度因子 (IF)骑行强度相对于 FTP处方、锻炼回顾和训练分类使用过时的 FTP，因此每次骑行看起来比实际更难或更容易
变异指数（VI）努力的稳定性或强度如何节奏分析和学科比较当赛事本身要求激增时，将所有高 VI 游乐设施称为“糟糕的节奏”
沃巴尔以上预计剩余工作能力 临界功率比赛策略、重复进攻、技术攀爬、区间设计当 CP 或 W' 估计较弱时盲目信任模型

有用的心理模型

想想 NP作为成本, IF 作为上下文与阈值, VI 作为起搏形状， 和 W'bal 作为您有限的激增预算。它们共同解释的内容远远超过单独的平均功率。

当每个指标有用时

📋训练处方

使用 FTP 或者 临界功率 建造 训练区，然后使用 IF 和 NP 检查会话是否到达您想要的位置。

🏁 比赛配速

使用 VI 和 NP 来判断您是否在稳定的公路骑行中表现出色，或者在早期的冲刺上花费了太多而导致终点褪色。

📊 骑行后分析

NP、IF 和 TSS 帮助比较不同持续时间和地形的会话，而无需假装每个平均瓦特讲述相同的故事。

⚡ 重复性

当赛事依赖于超过阈值的重复努力时，例如 XC MTB、激烈的公路赛车或绕圈加速，W'bal 最为重要。

⚠️ 关键依赖： 如果你的 FTP 或者 临界功率 估计是错误的，IF、训练区、TSS 和 W'bal 解释都随之漂移。

骑自行车者滥用这些指标的地方

常见错误

  • 比较从标准中稳定爬升到 NP 的平均功率，就好像它们意味着同一件事一样
  • 将 IF 与几个月未重新测试的 FTP 一起使用
  • 在可变性不可避免的比赛形式中追求低 VI
  • 像字面上的电量计一样读取 W'bal，而不是与假设相关的模型
  • 使用一份令人印象深刻的骑行文件重新定义健身，无需趋势数据

更好的解释规则

  • 从赛事类型开始：TT、公路赛、gran fonto、XC、enduro 或 crit
  • 在判断数字之前先检查起搏形状是否与目标相符
  • 使用 区域 和 TSS 决定骑行对训练意味着什么
  • 将 W'bal 与 临界功率 和反复努力的要求
  • 使用跨会话的趋势，而不是一个孤立的标题指标

功率指标如何连接到 FTP、CP 和区域

仅当页面相互支持时，该集群才变得有用：

  • FTP 锚定训练区域并使 IF 可解释
  • 临界功率 为高于阈值的工作和 W'bal 提供了更好的模型
  • 训练区 将阈值基准转化为每日处方
  • 空气动力学 解释了为什么同一骑手可能需要不同的功率来保持相同的速度
  • TSS 将强度加持续时间转换为负载分数，您可以随时间进行比较

稳定的事件

优先级指标： FTP、IF、VI 和起搏规则

示例： 计时赛、长距离爬坡、稳定的单人努力

可变事件

优先级指标： NP、VI、W'bal 和浪涌之间的恢复

示例： 绕圈赛、MTB、激烈的公路赛、技术攀爬

工作示例

稳定的道路爬升示例

骑手简介： FTP 300W

  • 期间： 20分钟
  • 平均功率： 282W
  • NP: 286W
  • IF： 0.95
  • 六： 1.01

解释： 这是近乎理想的稳定努力。 NP 接近平均功率，VI 确认节奏清晰，IF 显示骑手略低于阈值。对于这种类型的努力，平均功率和阈值背景说明了大部分情况。

可变山地车/标准示例

骑手简介： FTP 300W、CP 305W、W' 18 kJ

  • 期间： 60分钟
  • 平均功率： 228W
  • NP: 276瓦
  • IF： 0.92
  • 六： 1.21

解释： 如果您只读取平均功率，则该过程看起来很温和，但 NP 和 VI 显示反复高于阈值的浪涌。这正是 W'bal 和 临界功率 这很重要，因为骑手会反复消耗并部分恢复有限的无氧工作能力。

为什么这些例子很重要

同一个骑手可以以相似的平均功率和完全不同的生理消耗完成两次训练。这就是为什么 NP、IF、VI 和 W'bal 是易于引用的指标：它们解释了所完成的工作和所经历的压力之间的差异。

逐个指标摘要

归一化功效 (NP)

NP 估计权重激增后的骑行成本比稳定骑行更严重。它对于解释艰苦的团体骑行、滚动比赛和技术 MTB 文件至关重要，因为平均功率掩盖了真正的压力。

强度因子 (IF)

IF 是在阈值范围内安排行程的最简单方法。仅当您的 FTP 是可信的。如果 FTP 已过时，则 IF 会变为错误精度。

变异指数（VI）

六、 告诉你努力是稳定的还是混乱的。在计时赛中，低 VI 通常是良好的节奏。在 XC 比赛或绕圈赛中，更高的 VI 可能只是反映了赛事的要求。

沃巴尔

沃巴尔 帮助认真的骑手解释上述重复的工作 临界功率。当与比赛背景结合使用时，它是最有用的，而不是作为每次骑行的通用仪表板号码。

常见问题解答

为什么NP常常高于平均功率？

因为剧烈的浪涌会造成不成比例的生理成本。 NP 对这些浪涌的权重更大，因此通过 NP 进行可变骑行通常看起来比单独通过平均功率更困难。

什么是好的骑行VI？

没有普遍适用的良好VI。计时赛的目标可能约为 1.00-1.03，而绕圈赛或 XC 比赛自然可以设定更高的目标。评判VI针对的是该事件，而不是单一的全球基准。

如果我已经使用 FTP，我还需要临界功率吗？

并非总是如此。 FTP对于很多骑手来说已经足够了。当您关心重复的高于阈值的工作 W'bal 以及更稳健的高强度工作模型时，临界功率变得更加有用。

我应该多久更新一次这些指标？

每 6-8 周或在有意义的训练块之后重新测试或重新估计您的阈值锚定。这些指标的有用性取决于其背后的阈值。
