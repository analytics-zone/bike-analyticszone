山地自行车 (MTB) 功率分析

越野骑行的独特需求：扭矩、牵引力与可变性

MTB 功率与公路功率：主要区别

虽然瓦特就是瓦特，但产生和传递瓦特的方式在山地自行车上有着根本的不同。了解这些差异对于有效的 MTB 训练至关重要。

1. 随机性

公路： 相对稳定。可以长时间保持恒定的踏频和扭矩。

MTB： 极其多变。功率输出是"开/关"式的。微爆发之后是滑行。

指标影响： MTB 骑行通常具有更高的变异指数 (VI) (1.20 - 1.50+)，而公路骑行通常较低 (1.05 - 1.20)。

2. 踏频范围

公路： 窄范围 (80-100 RPM)。

MTB： 极宽范围 (40-120+ RPM)。

原因： 陡峭的技术性爬坡需要高扭矩/低踏频。平坦路段需要高踏频。必须训练在所有转速下产生功率。

3. 扭矩需求

公路： 较低的峰值扭矩，更平滑。

MTB： 极高的峰值扭矩。

原因： 清除障碍物（树根、岩石台阶）需要瞬间爆发力，通常是在低速/低惯性下。这需要神经肌肉力量。

4. 上半身负荷

公路： 静态，主要是支撑。

MTB： 动态，高负荷。

影响： 操控自行车（压弯、跳跃、吸收撞击）会增加显著的代谢成本，这不会显示在功率计（仅测量腿部功率）上。MTB 的心率相对于功率通常比公路高。

关键 MTB 指标

1. 标准化功率 (NP) 优于平均功率

由于大量的滑行（下坡、过弯），MTB 的平均功率通常看起来很低。NP 更好地反映了生理成本，因为它强调了那些艰难的爬坡和出弯冲刺。

  • 平均功率： 可能只有 180W。
  • NP： 可能是 260W。
  • 差异： 巨大的差异表明了骑行的“间歇性”本质。

2. FRC (功能储备容量) / W'

这是你的“火柴盒”。在 XCO（越野绕圈赛）比赛中，你可能会燃烧 50-100 根火柴。能够反复进行高于阈值的努力并快速恢复是 MTB 成功的关键。

3. Pmax (峰值功率) 和 5秒功率

对于起步、陡峭的技术路段和超车至关重要。虽然你不需要短跑运动员的 1500W，但你需要按需产生 600-800W 的能力，即使是在疲劳状态下。

针对 MTB 的训练

1. 模拟地形 (高扭矩)

在公路上或训练台上，以低踏频 (50-60 RPM) 进行高功率间歇训练，以模拟陡峭的技术性爬坡。

训练：肌肉耐力爬坡

3 x 10 分钟 @ 甜蜜点 (88-93% FTP)，55-65 RPM。

专注于平滑的踩踏和保持上半身稳定。

2. 微爆发

模拟赛道上不断的开/关特性。

训练：30/30s 或 40/20s

3 组 x (10 次 x 30秒 @ 130% FTP / 30秒 恢复)。

这建立了重复冲刺能力和乳酸清除能力。

3. 技术技能作为恢复

在间歇训练之间，练习过弯、定车或抬前轮。这教会你在心率高且疲劳时保持技术。

4. 力量训练

比公路车手更重要。核心、背部、肩膀和手臂力量对于操控自行车至关重要。深蹲和硬拉对于低踏频扭矩很有帮助。

装备对功率的影响

轮胎压力与滚动阻力

在越野中，较低的压力通常 = 较低的滚动阻力。这与公路相反。

  • 原因： 较软的轮胎会变形以适应障碍物（岩石、树根），保持向前的动量。
  • 过硬的轮胎： 会从障碍物上反弹，将向前的能量转化为向上的运动（振动损失）。
  • 最佳点： 通常在 18-25 psi 之间，取决于轮胎宽度、骑手体重和地形。

悬挂效率

现代悬挂系统非常高效，但仍会吸收一些骑手输入的能量（踏板摆动）。

  • 锁定： 在平滑的防火道爬坡上使用。
  • 全开： 在技术性地形上使用。让悬挂吸收颠簸比身体吸收更有效率，从而保持向前的动量和牵引力。
