サイクリングパワーメトリクス: NP、IF、VI、および W'bal の説明

ライド ファイルを接続するための実用的なハブ FTP, クリティカルパワー, FTP vs クリティカルパワー (CP), トレーニングゾーン、意思決定のペース調整、および TSS

簡単な回答

サイクリング電力メトリクス 生のワットを意思決定に変えます。 正規化パワー (NP) 変動乗車の真のコストを推定し、 強度係数 (IF) 乗り心地が相対的にどれだけ硬かったかを示します FTP, 変動指数 (VI) 取り組みがどれほど着実であったか、または混沌としたものであったかを示します。 ウィバル しきい値を超えた容量をどれだけ消費したかを解釈するのに役立ちます。

  • 最適な用途: トレーニングの処方、レースペース、ライド後の分析
  • 最も役立つアンカー: 最新の FTP または クリティカルパワー 価値
  • 最大の間違い: 地形、期間、イベントタイプのコンテキストなしで NP、IF、または W'bal を比較する
  • 最良の次のステップ: メトリクスを接続する トレーニングゾーン そして TSS それらを単独で読むのではなく

重要なポイント

  • NP 同じ平均パワーを持った 2 つの乗り物がまったく異なると感じられる理由を説明します
  • IF 乗り心地の強さを結び付ける FTP 数値がトレーニングの決定事項になる可能性があるため
  • VI 安定したロードでの取り組みを、負荷の高いクリテリウムや MTB レースから切り離します。
  • ウィバル 繰り返しのしきい値を超える努力とその間の回復を考慮する場合に最も役立ちます。
  • 実際のクラスター:FTP, クリティカルパワー, トレーニングゾーン, 空気力学、 そして TSS

ほとんどのサイクリストはこれ以上の指標を必要としません。ライド ファイルに既に含まれているものを解釈するためのより良い方法が必要です。このページはそのシステムの実質的な中心であり、コア電力メトリクスが何を意味するのか、いつ役立つのか、どこで失敗するのか、次にどの関連ページを使用する必要があるのか​​を示します。

役立つ質問はシンプルです。

  • この乗り心地は実際には大変でしたか、それとも単に騒音が大きかったでしょうか?
  • イベントに向けて適切なペースで取り組みましたか?
  • このワークアウトにとって、安定したパワー、変動性、またはしきい値を超える再現性のうち、どの指標がより重要ですか?
  • これらの数字によって、次に行うべきことがどのように変化するでしょうか?

これらのメトリクスが実際に測定するもの

平均電力 平均して何が起こったかを示します。コアサイクリング電力メトリクスからわかること 労力がどのように配分されたか, 閾値と比較してどの程度のストレスだったのか、 そして 制限されたしきい値を超える容量を賢く消費していたかどうか.

メトリック測定内容こんな方に最適よくある誤用
正規化パワー (NP)変動する電力出力の生理学的コストハードなグループライド、MTB、クリティカル、起伏のある地形、ペーシングエラーを解釈するイベント期間や地形コンテキストを使用せずに、それ自体をペーシング ターゲットとして扱う
強度係数 (IF)相対的な乗り心地の強さ FTP処方箋、ワークアウトのレビュー、セッションの分類時代遅れのFTPを使用しているため、すべての乗り物が実際よりも難しく見えたり、簡単に見えたりします
変動指数 (VI)努力がどれだけ着実であったか、または激しかったかペーシング分析と分野の比較イベント自体の要求が急増した場合、すべての高 VI ライドを「悪いペース」と呼ぶ
ウィバル上記の推定残作業能力 クリティカルパワーレース戦略、アタックの繰り返し、テクニカルな登り、インターバルの設計CP または W' 推定値が弱い場合にモデルを盲目的に信頼する

有用なメンタルモデル

考えてみましょう コストとしてのNP, コンテキストとしきい値としての IF, ペーシング形状としての VI、 そして 限られた急増予算に応じて調整します。これらを組み合わせると、単独の平均パワーよりもはるかに多くのことが説明されます。

各指標が役立つ場合

📋 トレーニングの処方箋

使用 FTP または クリティカルパワー 構築する トレーニングゾーン、次に IF と NP を使用して、セッションが意図した場所に到達したかどうかを確認します。

🏁 レースペーシング

VI と NP を使用して、安定したロードでうまく走れたのか、フィニッシュがフェードするような初期のサージに多大な費用を費やしすぎたかを判断します。

📊 乗車後の分析

NP、IF、 TSS すべての平均ワットが同じことを物語っているかのように装うことなく、さまざまな期間や地形にわたるセッションを比較するのに役立ちます。

⚡ 再現性

XC MTB、パンチの効いたロードレース、クリテリウムの加速など、イベントが閾値を超える繰り返しの努力に依存する場合、W'bal は最も重要です。

⚠️ 重大な依存関係: もしあなたの FTP または クリティカルパワー 推定は間違っており、IF、トレーニングゾーン、TSS、および W'bal の解釈はすべてそれに伴い変動します。

自転車利用者がこれらの指標を悪用する場所

よくある間違い

  • クリテリウムからの安定した上昇から NP までの平均パワーを、同じことを意味しているかのように比較する
  • 数か月間再テストされていない FTP とともに IF を使用する
  • 変動が避けられないレース形式で低い VI を追求する
  • 仮定に縛られたモデルではなく、文字通りの残量計のように W'bal を読み取る
  • 1 つの印象的なライド ファイルを使用して、トレンド データを使用せずにフィットネスを再定義する

より良い解釈ルール

  • イベントの種類から開始します: TT、ロードレース、グランフォンド、XC、エンデューロ、またはクリティカル
  • 数値を判断する前に、ペーシングの形状が目標と一致しているかどうかを確認してください
  • 使用 ゾーン そして TSS トレーニングにとって乗り物が何を意味するかを決定する
  • W'bal とペアリングする クリティカルパワー 繰り返しの努力の要求
  • 単一のヘッドライン指標ではなく、セッション全体の傾向を使用する

電力メトリクスを FTP、CP、ゾーンに接続する方法

このクラスターは、ページが相互にサポートしている場合にのみ役立ちます。

  • FTP トレーニングゾーンを固定し、IFを解釈可能にします
  • クリティカルパワー しきい値を超える作業と W'bal のより良いモデルを提供します
  • トレーニングゾーン 閾値ベンチマークを毎日の処方箋に変える
  • 空気力学 同じライダーが同じ速度を維持するために異なるパワーが必要になる理由を説明します
  • TSS 強度と持続時間を時間の経過とともに比較できる負荷スコアに変換します

安定したイベント

優先度の指標: FTP、IF、VI、およびペーシング規律

例: タイムトライアル、長い登り、着実なソロ努力

変数イベント

優先度の指標: NP、VI、W'bal、サージ間回復

例: クリテリウム、MTB、迫力あるロードレース、テクニカルな登り

実用例

安定した道路を登る例

ライダープロフィール： FTP 300W

  • 間隔： 20分
  • 平均パワー: 282W
  • NP: 286W
  • IF: 0.95
  • Ⅵ: 1.01

解釈： これは理想に近い地道な努力です。 NP は平均パワーに近い値を示し、VI はクリーンなペーシングを確認し、IF はライダーがしきい値をわずかに下回っていたことを示しています。このタイプの取り組みでは、平均パワーとしきい値のコンテキストがほとんどのことを物語ります。

バリアブルMTB/クリテリウム例

ライダープロフィール： FTP 300W、CP 305W、W'18kJ

  • 間隔： 60分
  • 平均パワー: 228W
  • NP: 276W
  • IF: 0.92
  • Ⅵ: 1.21

解釈： 平均パワーだけを読むと乗り心地は中程度に見えますが、NP と VI ではしきい値を超えるサージが繰り返し発生しています。ここはまさに W'bal と クリティカルパワー なぜなら、ライダーは限られた無酸素運動能力を繰り返し消費し、部分的に回復するからです。

これらの例が重要な理由

同じライダーが、同様の平均パワーと完全に異なる生理学的コストで 2 つのセッションを終了する可能性があります。 NP、IF、VI、および W'bal が引用に適した指標であるのはこのためです。これらは、実行された作業と経験したストレスの違いを説明しています。

メトリックごとの概要

正規化パワー (NP)

NP 定常走行よりも体重が急激に増加した後の乗車コストを見積もります。これは、ハードなグループライド、ローリングレース、平均的なパワーが本当の緊張を隠してしまうテクニカルなMTBファイルを解釈するのに不可欠です。

強度係数 (IF)

IF は、しきい値スケールに乗り物を配置する最も簡単な方法です。それは、次の場合にのみ役に立ちます。 FTP 信じられます。 FTP が古い場合、IF は誤った精度になります。

変動指数 (VI)

VI 努力が着実なものだったのか、混沌としたものだったかを示します。タイムトライアルでは、通常、VI が低い方が良いペースとなります。 XC レースまたはクリテリウムでは、より高い VI は単にイベントの要求を反映している可能性があります。

ウィバル

ウィバル 真剣なライダーが上記の繰り返しの作業を解釈するのに役立ちます クリティカルパワー。すべてのライドに共通のダッシュボード番号としてではなく、レースのコンテキストと組み合わせると最も役立ちます。

よくある質問

NP が平均パワーよりも高いことが多いのはなぜですか?

なぜなら、激しいサージは不釣り合いな生理的コストを生み出すからです。 NP はこれらのサージをより重く加重するため、通常、可変的な乗り心地は、平均パワーだけを使用するよりも NP を使用すると難しく見えます。

サイクリングに適した VI とは何ですか?

普遍的な良い VI はありません。タイムトライアルではおよそ 1.00 ～ 1.03 が目標となりますが、クリテリウムや XC レースでは当然それよりもさらに高い値が設定される可能性があります。単一の世界的なベンチマークではなく、イベントに対してジャッジ VI を判断します。

すでにFTPを使用している場合、クリティカルパワーは必要ですか?

いつもではありません。多くのライダーにとってFTPで十分です。クリティカル パワーは、繰り返し閾値を超える作業、W'bal、および厳しい強度の作業のより堅牢なモデルを考慮する場合にさらに役立ちます。

これらの指標はどれくらいの頻度で更新すればよいですか?

6 ～ 8 週間ごと、または有意義なトレーニング ブロックの後に、しきい値アンカーを再テストまたは再推定します。メトリックは、その背後にあるしきい値と同じくらい役に立ちます。
